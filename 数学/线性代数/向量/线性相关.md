# 线性相关

给行[[向量组/向量组|向量组]] $\vec{\alpha_{1}},\vec{\alpha_{2}},\dots,\vec{\alpha_{m}}$，如果存在一组不全为令的数 $k_{1},k_{2},\dots,k_{m}$，使得*线性组合* $k_{1}\vec{\alpha_{1}}+k_{2}\vec{\alpha_{2}}+\dots+k_{m}\vec{\alpha_{m}}=0$ 成立，则称[[向量组/向量组|向量组]] $\vec{\alpha_{1}},\vec{\alpha_{2}},\dots,\vec{\alpha_{m}}$ **线性相关**
- 反之仅当 $k$ 全为 0 时线性组合成立，其他时候不成立，则称之为**线性无关**

- 一个向量 $\alpha$ 线性相关，则 $\alpha=0$；一个向量 $\alpha$ 线性无关，则 $\alpha\neq0$
- 两个非零向量 $\alpha$ 与 $\beta$ 线性相关，则 $\alpha$ 与 $\beta$ 的分量对应成比例．
- 如果向量组中有一部分向量（称为部分组）线性相关，则该向量组线性相关
	- 若向量组线性无关，则其任意一部分组也线性无关
	- 含有向量的向量组比线性相关
- $n$ 维单位向量组必线性无关

- 设$r$维向量$\alpha_1$，$\alpha_2$，…，$\alpha_m$*线性无关*，$\beta_1$，$\beta_2$，…，$\beta_m$为$\alpha_1$，$\alpha_2$，…，$\alpha_m$的$r+l$维*接长向量*，则$\beta_1$，$\beta_2$，…，$\beta_m$*线性无关*
- 设$\alpha_1$，$\alpha_2$，…，$\alpha_m$是$r$维向量，$\beta_1$，$\beta_2$，…，$\beta_m$是其$r+l$维接长向量，若$\beta_1$，$\beta_2$，…，$\beta_m$线性相关，则$\alpha_1$，$\alpha_2$，…，$\alpha_m$也线性相关

若向量组中有两个向量成比例，则向量组必线性相关

## 定理


定理 3.9 设有两个向量组：(Ⅰ) $\alpha_1$，$\alpha_2$，…，$\alpha_r$，(Ⅱ) $\beta_1$，$\beta_2$，…，$\beta_s$，若向量组(Ⅰ)线性无关，且向量组(Ⅰ)可由向量组(Ⅱ)线性表示，则 $r\leqslant s$ ．

定理 3.10 设有两个向量组：(Ⅰ) $\alpha_1$，$\alpha_2$，…，$\alpha_r$，(Ⅱ) $\beta_1$，$\beta_2$，…，$\beta_s$，若向量组(Ⅰ)可由向量组(Ⅱ)线性表示，且 $r \geq s$，则向量组(Ⅰ)必线性相关．

推论 3.3 若（Ⅰ），（Ⅱ）均线性无关，且（Ⅰ）（Ⅱ）可相互线性表示，则 $r=s$ ．


推论 3.4 若向量组所含向量的个数大于向量的维数，则该向量组线性相关
