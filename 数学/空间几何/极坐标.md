# 极坐标

极坐标：**用角度和长度描述位置的坐标系**

- **极点**和**极轴**：以原点𝑂为起点的射线作为参考系
- **极径$\rho$**：点𝑃到原点的距离记为$\overrightarrow{OP}=𝑟(\geq0)$，称为
- **极角$\theta$**：从参考系射线出发逆时针旋转到𝑂𝑃所经过的角度，记为$\theta\in[0,2\pi)$


## 直角坐标与极坐标转换

直角坐标转极坐标$$\rho(\theta)\Rightarrow\begin{cases}x=\rho \cos\theta \\y=\rho \sin\theta \\\end{cases}$$

- 直线极坐标公式$\begin{cases}\arctan\left( \frac{y_{1}}{x_{1}} \right)\leq\theta\leq \arctan\left( \frac{y_{2}}{x_{2}} \right) \\\rho=\frac{b}{\sin\theta-k\cos\theta}\end{cases}$
- 扇形面积公式：$A=\frac{1}{2}r^{2}\theta$


极坐标转直角坐标：$y(x)\Rightarrow \begin{cases}\rho=\sqrt{ x^{2}+y^{2} } \\ \theta=\arcsin\left( \frac{y}{\sqrt{ x^{2}+y^{2} }} \right)=\arccos\left( \frac{x}{\sqrt{ x^{2}+y^{2} }} \right) \\ \sin\theta=\frac{y}{\sqrt{ x^{2}+y^{2} }} \\ \cos\theta=\frac{x}{\sqrt{ x^{2}+y^{2} }}\end{cases}$
