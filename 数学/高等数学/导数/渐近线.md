# 函数渐近线

垂直渐近线：若$\lim\limits_{x \to x_0} f(x) = \infty$，或者$\lim\limits_{x \to x_0^-} f(x) = \infty$或者$\lim\limits_{x \to x_0^+} f(x) = \infty$，则称$x = x_0$为曲线$y = f(x)$的垂直渐近线

水平渐近线：若$\lim\limits_{x \to \infty} f(x) = A$，或者$\lim\limits_{x \to +\infty} f(x) = A$或者$\lim\limits_{x \to -\infty} f(x) = A$（$A$为常数），则$y = A$为曲线$y = f(x)$的水平渐近线

斜渐近线：直线$y = ax + b$为曲线$y = f(x)$的斜渐近线，其中$a = \lim\limits_{x \to \infty} \frac{f(x)}{x}$，$(a \neq 0)$，$b = \lim\limits_{x \to \infty} [f(x) - ax]$

## 求渐近线

### 求水平渐近线

求 $y = \arctan x$ 水平渐近线
- 解 $\lim\limits_{ x \to \infty }{\arctan x}$ 不存在  
- ∴ 当 $\lim\limits_{x \to +\infty}\arctan x = \frac{\pi}{2}$
- 当 $\lim\limits_{ x \to -\infty }{\arctan x} = -\frac{\pi}{2}$ 
- 即 $y = \frac{\pi}{2}$ 及 $y = -\frac{\pi}{2}$ 为水平渐近线

### 求垂直渐近线

找 $x_{0}$ 无意义点
  
求 $f(x) = \frac{1}{(x+1)(x-2)}$ 渐近线
- $x = -1,2$ 为垂直渐近线


求 $f(x) = \ln\left(e + \frac{1}{x}\right)$ 渐近线
- 无意义点 $x = 0, -\frac{1}{e}$
- $\lim\limits_{ x \to 0^+ }{\ln\left(e + \frac{1}{x}\right)}  = +\infty \Rightarrow x = 0$ 为垂直渐近线
- $\lim\limits_{ x \to -\frac{1}{e}^- }{\ln\left(e + \frac{1}{x}\right)}  = -\infty \Rightarrow x = -\frac{1}{e}$ 为垂直渐近线


### 求斜渐近线


$a=\lim\limits_{ x \to \infty(\pm\infty) }{\frac{f(x)}{x}}$
$b=\lim\limits_{ x \to \infty }{f(x)-ax}$



求 $y = \frac{x + 4\ln x}{5x - 2\cos x}$ 水平渐近线
- $1^\circ\stackrel{\frac{\infty}{\infty}}{=} \lim\limits_{x \to \infty} \frac{1 + \frac{4}{x}}{5 + 2\sqrt{x}}$ 不行
- $2^\circ=\lim\limits_{x \to \infty} \frac{1 + \frac{4}{x}\ln x}{5 - \frac{2}{x}\cos x} = \frac{1 + 0}{5 - 0} = \frac{1}{5}$
- 所以 $y=\frac{1}{5}$ 为水平渐近线

例 8: 求 $y = \frac{1}{x} + \ln(1 + e^x)$ 渐近线
- $1^{\circ}$ $\lim\limits_{x \to 0} \left[ \frac{1}{x} + \ln(1 + e^x) \right] = \infty \Rightarrow x = 0$ 为垂直...  
- $2^{\circ}$ $\lim\limits_{x \to \infty} \left[ \frac{1}{x} + \ln(1 + e^x) \right]=0+\lim\limits_{ x \to \infty }{\ln{(1+e^{x})}}=\begin{cases}+\infty & ,x\to +\infty \\ 0 & ,x\to-\infty\end{cases}$
	- $\therefore y=0$ 为水平
- $3^{\circ}$ $a=\lim\limits_{ x \to \infty }{\frac{\frac{1}{x}+\ln{(1+e^{x})}}{x}}$$=0+\lim\limits_{ x \to \infty }{\frac{\ln{(1+e^{x})}}{x}}$$=\begin{cases}\lim\limits_{ x \to +\infty }{\frac{e^{x}}{1+e^{x}}} & =1 \\ \lim\limits_{ x \to -\infty }{\frac{\ln{(1+e^{x})}}{x}} & =0\implies水平线\end{cases}$
	- $b=\lim\limits_{ x \to +\infty }{f(x)-ax}$$0+\lim\limits_{ x \to +\infty }{\ln{\left( \frac{1+e^{x}}{e^{x}} \right)}}=0$
	- $\therefore y=x$ 为斜渐近线


例 10 $y = \frac{(1+x)^2}{\sqrt{x}}$ 斜渐近线方程  
- 解 $a = \lim\limits_{x \to +\infty} \frac{(1+x)^{\frac{3}{2}}}{\sqrt{x}\cdot x}= \lim\limits_{x \to +\infty} \frac{(1+x)^{\frac{3}{2}}}{x^{\frac{3}{2}}} =1$
- $b=\lim\limits_{ x \to +\infty }{\left[ \frac{(1+x)^{3/2}}{\sqrt{ x }} \right]}$
	- $1^{\circ}$ = $\lim\limits_{ x \to +\infty }{\frac{x^{3/2}\left[ \left( 1+\frac{1}{x} \right)^{3/2}-1 \right]}{\sqrt{ x }}}$$=\lim\limits_{ x \to +\infty }{x\cdot \frac{3}{2}\cdot \frac{1}{x}}=\frac{3}{2}$
	- $2^{\circ}$ = $\lim\limits_{ x \to +\infty }{\frac{(1+x)^{3}-x^{3}}{\sqrt{ x }(\sqrt{ (1+x)^{3} }+\sqrt{ x^{3} })}}=\frac{3}{2}$
	- $3^{\circ}$$\frac{\infty}{\infty} =\lim\limits_{ t \to +\infty }{\frac{(1+t^{2})^{3/2}-t^{3}}{t}}$
		- $=\lim\limits_{ t \to +\infty }{\frac{3}{2}(1+t^{2})^{1/2}\cdot2t-3t^{2}}$$=\begin{cases} 1^{\circ}=\lim\limits_{ t \to +\infty }{\frac{3t}{\sqrt{ 1+t^{2} }+t}} \\ 2^{\circ} 倒代换 \\ 3^{\circ}\lim\limits_{ t \to +\infty }{3t^{2}}\left[ \left( 1+\frac{1}{t^{2}} \right)^{1/2} -1 \right]\end{cases}$$=\frac{3}{2}$
