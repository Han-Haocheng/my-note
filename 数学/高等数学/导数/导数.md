# 导数

 #数学/定义 ***导数*** 
- 设函数$y=f(x)$在点$x_{0}$的某个邻域内有定义
	- 当自变量$x$在$x_{0}$处取得增量$\Delta x$(点$x_{0}+\Delta x$仍在该邻域内)时
	- 相应因变量取得增量$\Delta y=f(x+\Delta x)-f(x_{0})$；
- 如果$\Delta y$与$\Delta x$之比当$\Delta x\to0$时的极限存在
- 那么称函数$y=f(x)$在点$x_{0}$处可导
	- 并称这个极限为函数$y=f(x)$在点$x_{0}$处的**导数**
- 记作
	- $y'|_{x=x_{0}}$
	- $\frac{\mathrm{d}y}{\mathrm{d}x}|_{x=x_{0}}$
	- $\frac{\mathrm{d}f(x)}{\mathrm{d}x}|_{x=x_{0}}$
	- $f'(x_{0})=\lim_{ \Delta x\to 0 } \frac{\Delta y}{\Delta x}=\lim_{ \Delta x \to 0 } \frac{f(x_{0}+\Delta x)-f(x_{0})}{\Delta x}$
	- $\lim_{ x \to x_{0} } \frac{{f(x)-f(x_{0})}}{x-x_{0}}$


#数学/定理 ***可导一定连续，连续不一定可导***


## 1 函数单侧导数
#数学/定义 函数单侧导数

**左导数**：$f'_{-}(x_{0})=\lim_{ h \to 0^- } \frac{f(x_{0+h})-f(x_{0})}{h}=\lim_{ x \to x^-_{0} } \frac{f(x)-f(x_{0})}{x-x_{0}}$

**右导数**：$f'_{+}(x_{0})=\lim_{ h \to 0^+ } \frac{f(x_{0+h})-f(x_{0})}{h}=\lim_{ x \to x^+_{0} } \frac{f(x)-f(x_{0})}{x-x_{0}}$

### 1.1 导数的充要条件

#数学/定理 导数的充要条件 

- **函数在$x_{0}$处可导$\iff$左右导数存在且相等**
- 如果函数$f(x)$在开区间$(a,b)$内可导
	- 且$f'_{+}(a)$和$f'_{-}(b)$都存在
- 那么就说$f(x)$在闭区间$[a, b]$上可导.

## 2 导数的几何意义

- 连续的几何含义：线不分段
- 可导的几何含义：光滑，不带尖，线的切线不能垂直于x轴
- 导数的几何含义：过某点切线的斜率

**切线**：$y-y_{0}=f'(x_{0})(x-x_{0})$
**法线**：$y-y_{0}=-\frac{1}{f'(x_{0})}(x-x_{0})$


## 3 求导公式


### 3.1 导数和差积商

- $[u(x)\pm v(x)]'=u'(x)\pm v'(x)$
- $[\mathbb{C}v(x)]'=\mathbb{C}v'(x)$
- $[u(x)\times v(x)]'=u'(x)v(x)+u(x)v'(x)$
	- $(uvw)'=u'vw+uv'w+uvw'$
- $\left[ \frac{u(x)}{v(x)} \right]'= \frac{u'(x)v(x)-u(x)v'(x)}{v^2}$



### 3.2 复合函数求导

#数学/法则 复合函数的求导法则
- 如果 $u=g(x)$ 在点 $x$ 可导
	- 而 $y=f(u)$ 在点 $u=g(x)$ 可导
- 那么复合函数 $y=f[g(x)]$ 在点 $x$ 可导
	- 且其导数为 $\frac{\mathrm{d} y}{\mathrm{d} x}=f'(u)\cdot g'(x)=\frac{\mathrm{d} y}{\mathrm{d} u}\cdot\frac{\mathrm{d} u}{\mathrm{d} x}$

>[!tip]+ 提示
> - 从外层到内层一层层求导，结果相乘
> - $y=x^x$ 类型求导：需要转换为 $y=e^{x\ln x}$ 或 $\ln y=x\ln x$ 形式后再求导


### 3.3 反函数的求导法则

#数学/法则 反函数的求导法则

- 如果函数 $x=f(y)$ 区间 $I_{y}$ 内单调可导
	- 且 $f'(y)\neq 0$
- 那么它的反函数 $y=f^{-1}(x)$ 在区间 $I_{x}=\{x|x=f(y),y\in I_{y}\}$ 内也可导
	- 即 $[f^{-1}(x)]'=\frac{1}{f'(y)}$ 或 $\frac{\mathrm{d} y}{\mathrm{d} x}=\frac{1}{\frac{\mathrm{d} y}{\mathrm{d} x} }$


### 3.4 初等函数导函数

#数学/公式/导数 常数导数： $(\mathbb{C})'=0$

>[!abstract] 证明： $\lim_{ h \to 0 } \frac{{f(x+h)-f(x)}}{h}=0$


#数学/公式/导数 幂次导数
- $(x^n)'=\begin{cases}1&n=1\\nx^{n-1}&n>1\end{cases},n\in \mathbb{Z}^+$
- $(x^\mu)'=\mu x^{\mu-1},\mu \in \mathbb{R}$

>[!abstract]- 证明：$f'(x)=(x^n)'=\begin{cases}1&n=1\\nx^{n-1}&n>1\end{cases},n\in \mathbb{Z}^+$
>- 当 $n=1$ 时 $$\begin{align}&\lim_{ h \to 0 } \frac{{f(x+h)-f(x)}}{h}\\=&\lim_{ h \to 0 } \frac{{x+h-x}}{h}=1\end{align}$$
>- 当 $n>1$ 时 $$\begin{align} & \lim_{ h \to 0 } \frac{{f(x+h)-f(x)}}{h}\\=& \lim_{ h \to 0 } \frac{{(x+h)^n-x^n}}{h}\\=& \lim_{ h \to 0 } \frac{[C^{n}_{n}x^n+C^{n-1}_{n}x^{n-1}h+C^{n-2}_{n}x^{n-2}h^2+\dots+C^{0}_{n}h^n]-x^n}{h}\\=& \lim_{ h \to 0 }	\left[ nx^{n-1}+\frac{n(n_{1})}{2}x^{n-2}h+\dots+h^{n-1} \right]= nx^{n-1}\end{align}$$

>[!abstract]- 证明：$f'(x)=(x^\mu)'=\mu x^{\mu-1},\mu \in \mathbb{R}$
> $\lim_{ h \to 0 } \frac{f(x+h)-f(x)}{h}$
> $= \lim_{ h \to 0 } \frac{(x+h)^\mu-x^\mu}{h}$
> $= \lim_{ h \to 0 } x^{\mu-1}	 \frac{\left( 1+\frac{h}{x} \right)^\mu-1}{\frac{x}{h}}$
> $= \lim_{ h \to 0 } \mu x^{\mu-1}$

### 3.5 高阶导数

#数学/定义 高阶导数

- 对函数进行多次求导
- **二阶导数**记作 $y''=\frac{d}{dx}\left( \frac{dy}{dx} \right)=\frac{d^2y}{dx^2}$
- **高阶导数**记作 $y^{(n)}=\frac{d^ny}{dx^nc}$

#数学/公式/导数 高阶导数

-  $(e^{x})^{(n)}=e^{x}$
-  $(e^{ax+b})^{(n)}=a^{n}e^{ax+b}$
-  $(a^{x})^{(n)}=a^{x}\ln^{n}{a}$
- $(\sin x)^{(n)}=\sin\left( x+\frac{n}{2}\pi \right)$
-  $(\cos x)^{(n)}=\cos\left( x+\frac{n}{2}\pi \right)$
-  $\left( \frac{1}{ax+b} \right)^{(n)}=(-1)^{n} \frac{a^{n}n!}{(ax+b)^{n+1}}$
- $(u\pm v)^{(n)}=u^{(n)}\pm v^{(n)}$
- $(\mathbb{C}u)^{(n)}=\mathbb{C}\cdot u^{(n)}$
- 莱布尼茨公式：$$(u\cdot v)^{(n)}=\sum_{i=0}^{n}C_{n}^{i}u^{(n-i)}v^{(i)}$$

#数学/公式/导数 反函数求导

- 二阶反函数求导： $\frac{\mathrm{d}^{2} x}{\mathrm{d} y^{2}}=\frac{\mathrm{d} }{\mathrm{d} y}\left( \frac{\mathrm{d} x}{\mathrm{d} y} \right)=\frac{\mathrm{d} }{\mathrm{d} x}\left( \frac{\mathrm{d} x}{\mathrm{d} y} \right)\cdot\frac{\mathrm{d} x}{\mathrm{d} y}$

1. 对方程两边同时对x求导
2. 对数求导法：对方程两边同时取$\ln{}$，然后对隐函数求导
	- $y=\frac{{g(x)h(x)}}{\varphi(x)}$
	- $\ln{y}=\ln{g(x)}+\ln{h(x)}-\ln{\varphi(x)}$

### 4.5 幂指函数求导

$y(x)=u(x)^{v(x)}$
1. 通过$y(x)=e^{v(x)\ln{u(x)}}$
2. 两边同时取对数$\ln{y}=v(x)\ln{u(x)}$

### 4.6 参数方程

若函数$x=x(t)$，$y=y(t)$都可导，且$x'(t)\neq 0$，则函数$y=y(x)$可导，且$\frac{\mathrm{d} y}{\mathrm{d} x}=\frac{y'(t)}{x'(t)}$，$\frac{\mathrm{d} ^{2}y}{\mathrm{d} x^{2}}=\frac{y''(t)x'(t)-y'(t)x''(t)}{[x'(t)]^{3}}$
- 直接通过公式分别让$x,y$对$t$求导

例题：分段函数求极值

求$f(x)=\begin{cases}x^{x}-1 & ,x>0 \\xe^{x} & ,x\leq0\end{cases}$的极值

- $\lim_{ x \to 0^{-} }{f(x)=0}$
- $\lim_{ x \to 0^{+} }{f(x)}=0$
- $f(0)=0$
- 所以$f(x)$在区间$\mathbb{R}$连续

- 求$x\to0$的导数$f'_{+}(0)=\lim_{x\to0^{+}} \frac{f(x)-f(0)}{x-0}=-\infty$
- 所以$f'(0)$不存在
- 所以令$f'(x)=\begin{cases}x^{x}(\ln{x}+1) & ,x>0 \\ e^{x}(1+x) & ,x<0\end{cases}=0$
- $x=e^{-1},-1$，不可导点$x=0$

| x       | $(-\infty, -1)$ | $-1$ | $(-1,0)$ | $0$ | $(0,e^{-1})$ | $e^{-1}$ | $(e^{-1}，+\infty)$ |
| ------- | --------------- | ---- | -------- | --- | ------------ | -------- | ------------------ |
| $f'(x)$ | $-$             |      | $+$      |     | $-$          |          | $+$                |
| $f(x)$  | ⬇️              | 极小   | ⬆️       | 极大  | ⬇️           | 极小       | ⬆️                 |

- 单减区间：$(-\infty,-1],(0,e^{-1}]$
- 单增区间：$[-1,0],[e^{-1},+\infty)$
- 极小值$f(-1)=-e^{-1},f(e^{-1})=e^{-1/e}-1$
- 极大值$f(0)=0$

### 5.2 函数最值
1. 求导得$f'(x)$。
2. 令$f'(x)=0$，得到$f(x)$的驻点及一阶导数不存在的点$x=x_1,x_2,\cdots$，此为可能的极值点
3. 将可能的极值点的函数值$f(x_1),f(x_2),\cdots$与左、右端点的函数值$f(a),f(b)$作比较，最大的即为最大值，最小的即为最小值


求$f(x)=x+\sqrt{1-x}$在闭区间$[-5,1]$上的最值
- 解 由于$f'(x)=1-\frac{1}{2\sqrt{1-x}}$，$f(x)$有不可导点$x=1\in[-5,1]$。
- 令$f'(x)=0$得驻点$x=\frac{3}{4}\in[-5,1]$
- 又$f(-5)=\sqrt{6}-5$，$f(\frac{3}{4})=\frac{5}{4}$，$f(1)=1$
- 所以，比较大小可知函数$f(x)$在区间$[-5,1]$上的最大值为$f(\frac{3}{4})=\frac{5}{4}$，最小值为$f(-5)=\sqrt{6}-5$。

求数列$\{\sqrt[n]{n}\}$的最大项
- 解 将数列转化为函数，设$f(x)=\sqrt[x]{x}(x\geq1)$
- 则$f'(x)=(\sqrt[x]{x})'=\sqrt[x]{x}\frac{1-\ln x}{x^2},(x\geq1)$
- 令$f'(x)=0$得$x=e$，
- 列下表如下：

| $x$     | $[1,e)$ | $e$  | $(e,+\infty)$ |
| ------- | ------- | ---- | ------------- |
| $f'(x)$ | $+$     | $0$  | $-$           |
| $f(x)$  | ↗单增     | 极大值点 | ↘单减           |
- 可得$f(x)$在$x=e$出取得极大值，也是最大值。
- 所以，对于数列$x_n=\sqrt[n]{n}$，最大项为$x_2$或$x_3$，
- 又$x_2=\sqrt{2}$，$x_3=\sqrt[3]{3}$，$\sqrt{2}<\sqrt[3]{3}$，
- 所以数列$\{\sqrt[n]{n}\}$的最大项为$\sqrt[3]{3}$。

#### 5.4.3 不等式证明

函数不等式$f(x)>g(x)$
1. 构造$F(x)=\begin{cases}F(x)=f(x)-g(x) \\ 恒等变形后构造\end{cases}$
2. 求$F'(x)$
3. 判断单调性

- $\sin x>x>\tan x,(x<0)$
- $\sin x<x<\tan x ,(x>0)$
- $e^{x}-1>x ,(x>0)$
- $\ln{(1+x)}<x  ,(x>0)$
- $\frac{2}{\pi}x<\sin x<x  ,\left( 0<x< \frac{\pi}{2} \right)$
- $x<\tan x< \frac{4}{\pi}x,\left( 0<x< \frac{\pi}{4} \right)$



证明当$x>0$时，$\frac{x}{1+x}<\ln{(1+x)}<x$
- 令 $f(x) = \ln(1+x) - x, x>0$  
	- ∴ $f'(x) = \frac{1}{1+x} - 1 = \frac{-x}{1+x} < 0, x>0$  
	- ∴ $f(x)$ 在 $[0, +\infty)$ 上 ↓ 则 $(\forall x>0$ 有 $\lim\limits_{x \to +\infty} f(x) < f(x) < f(0) = 0)$
	- $\therefore \forall x>0$有$f(x) < f(0) = 0$
	- $\therefore\ln{(1+x)}<x$
- $g(x) = \frac{x}{1+x} - \ln(1+x)$  (麻烦)
- 或 $g(x) = x - (1+x)\ln(1+x), \ x>0$  
	- ∴ $g'(x) = 1 - \ln(1+x) - 1 = -\ln(1+x) < 0, \ x>0$  
	- ∴ $g(x)$在$[0, +\infty)$ $\downarrow$ 即$\forall x>0$有 $g(x) < g(0) = 0$  
	- ∴ $x < (1+x)\ln(1+x)$ 即 $\frac{x}{1+x} < \ln(1+x)$.


证明当 $0 < x < \frac{\pi}{2}$ 时, $\cos x < \frac{x^3}{6} - \frac{x^2}{2} + 1$  
- 令 $f(x) = \frac{1}{6}x^3 - \frac{1}{2}x^2 - \cos x + 1 \quad 0 < x < \frac{\pi}{2}$  
- $f'(x) = \frac{1}{2}x^2 - x + \sin x$
- $f''(x) = x - 1 + \cos x$
- $f''(x) = 1 - \sin x > 0$
- $\therefore f''(x)$ 在 $[0, \frac{\pi}{2}]$ 单增
- $\therefore \forall 0 < x < \frac{\pi}{2}$ 有 $f''(x) > f''(0) = 0$ 
- $\therefore f'(x)$在 $[0, \frac{\pi}{2}]$ 单增

## 6 导数的证明

费马引理

#数学/定理 费马引理

- 设函数$f(x)$在点$x_{0}$的某邻域$U(x_{0})$内有定义
	- 并且$f(x)$在$x_{0}$处可导
- 如果对任意$x\in U(x_{0})$
	- 有$f(x)\leq f(x_{0})$(或$f(x)\geq f(x_{0})$)
- 那么$f'(x_{0})=0$


### 6.1 罗尔定理

![[数学/attachments/20241226095013.svg]]
#数学/定理 罗尔定理


- 如果 $f(x)$ 满足在区间 $[a,b]$ 连续，在区间$(a,b)$可导
	- 且在区间端点处的函数值相等，即$f(a)=f(b)$
- 那么在 $(a,b)$ 内至少有一点 $\xi \in(a,b)$，使得$f'(\xi)=0$

> [!abstract]- 证明：罗尔定理
> - 由于$f(x)$在闭区间$[a,b]$上连续
> - 根据闭区间上连续函数的最大值最小值定理
> 	- $f(x)$在闭区间$[a,b]$上必定取得它的最大值$M$和最小值$m$
> - $M=m$
> 	- 这时$f(x)$在区间$[a,b]$上必然取相同的数值$M:f(x)=M$
> 	- 由此，任意$x\in(a,b)$
> 		- 有$f'(x)=0$
> 	- 因此，任取$\xi \in (a,b)$
> 		- 有$f'(\xi )=0$
> - $M>m$
> 	- 因为$f(a)=f(b)$
> 	- 所以$M$和$m$这两个数中至少有一个不等于$f(x)$在区间$[a,b]$的端点处的函数值
> 	- 设$M\neq f(a)$
> 		- 那么在区间$(a,b)$内至少有一点$\xi$
> 		- 使$f(\xi)=M$
> 	- 因此，任取$x\in[a,b]$
> 		- 有$f(x)\leq f(\xi)$
> 	- 从而由费马引理可知$f'(\xi)=0$
> - 定理证毕

### 6.2 拉格朗日中值定理

![[数学/attachments/20241226095658.svg]]

#数学/定理 拉格朗日中值定理



- 如果函数$f(x)$满足
	- 在区间$[a,b]$连续
	- 在区间$(a,b)$可导
- 那么在$(a,b)$内至少有一点$\xi \in(a,b)$
	- 使等式$f(b)-f(a)=f'(\xi)(b-a)$成立
	- 或$f'(\xi)=\frac{f(b)-f(a)}{b-a}$

> [!abstract]- 证明：拉格朗日中值定理
> - 引入辅助函数$\varphi(x)=f(x)-f(a)-\frac{{f(b)-f(a)}}{(b-a)}(x-a)$
> 	- 容易证明$\varphi(a)=\varphi(b)=0$
> 	- $\varphi(x)$在区间$[a,b]$连续
> 	- $\varphi(x)$在区间$(a,b)$可导
> 	- 且$\varphi'(x)=f'(x)-\frac{{f(b)-f(a)}}{b-a}$
> - 根据罗尔定理
> 	- 在$(a,b)$内至少有一点$\xi$
> 		- 使$\varphi'(\xi)=0$
> 	- 即$f'(\xi)-\frac{{f(b)-f(a)}}{b-a}=0$
> - 由此得$f'(\xi)=-\frac={{f(b)-f(a)}}{b-a}$
> 	- 即$f(b)-f(a)=f'(\xi)(b-a)$
> - 证毕

#数学/定理 拉格朗日中值定理的推论

- 如果函数$f(x)$
	- 在区间$I$连续
	- 在区间$I$内可导
	- $f'(x)=0$
- 那么$f(x)$在区间$I$上是一个常数

> [!abstract]- 证明：拉格朗日中值定理的推论
> - 在区间I上任取两点$x_{1},x_{2}$ ($x_{1}<x_{2}$ )
> - 根据拉格朗日中值定理
> 	- $f(x_{2})-f(x_{1}) =f'(\xi)(x_{2}-x_{1})$($x_{1}<\xi<x_{2}$)
> - 由假定，$f'(\xi)=0$
> 	- 所以$f(x_{2})-f(x_{1})=0$
> 	- 即$f(x_{2})= f(x_{1})$
> - 因为$x_{1},x_{2}$是$I$上任意两点
> 	- 所以$f(x)$在$I$上的函数值总是相等的
> - 即$f(x)$在区间$I$上是一个常数

### 6.3 柯西中值定理

#数学/定理 柯西中值定理

- 如果函数$f(x)$及$F(x)$满足
	- 在区间$[a,b]$连续
	- 在区间$(a,b)$可导
	- 对任意$x\in(a,b)$都有$F'(x)\neq 0$
- 那么在$(a,b)$中至少存在一点$\xi$
	- 使等式$\frac{{f(b)-f(a)}}{F(b)-F(a)}=\frac{f'(\xi)}{F'(\xi)}$成立
