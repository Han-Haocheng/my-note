# 偏导数


对 x 的偏导

- 设函数$z=f(x,y)$在点$(x_{0},y_{0})$的某一邻域内有定义
	- 当 ***$y$固定在$y_{0}$*** 而$x$在$x_{0}$处有增量$\Delta x$时
	- 相应的函数有增量 $f(x_{0}+\Delta_{0},y_{0})-f(x_{0},y_{0})$
- 如果极限存在
	- 即$\lim_{ \Delta x \to 0 } \frac{f(x_{0}+\Delta x,y_{0})-f(x_{0},y_{0})}{\Delta x}$
- 那么称此极限为函数$z=f(x,y)$在点$(x_{0},y_{0})$处***对x的偏导数***
	- 记作 $\left. \frac{\partial y}{\partial x} \right |_{x=x_{0},y=y_{0}}$，$\left. \frac{\partial f}{\partial x} \right |_{x=x_{0},y=y_{0}}$，$\left. z'_{x} \right |_{x=x_{0},y=y_{0}}$，$f'_{x}(x_{0},y_{0})$

对y 的偏导

- 设函数$z=f(x,y)$在点$(x_{0},y_{0})$的某一邻域内有定义
	- $x$固定在$x_{0}$
	- $y$在$y_{0}$处有$\Delta y$
- 对 $y$ 的偏导数为 $\lim\limits_{ \Delta y \to 0 } \frac{f(x_{0},y_{0}+\Delta y)-f(x_{0},y_{0})}{\Delta x}$
	- 记作 $\left. \frac{\partial y}{\partial y} \right |_{x=x_{0},y=y_{0}}$ ,$\left. \frac{\partial f}{\partial y} \right |_{x=x_{0},y=y_{0}}$ ,$\left. z'_{y} \right |_{x=x_{0},y=y_{0}}$ ,$f'_{y}(x_{0},y_{0})$


## 偏导数与连续的关系

- 对 x 的偏导数存在,在 $y=y_{0}$ 处连续
- 对 y 的偏导数存在,在 $x=x_{0}$ 处连续


> [!warning] 注意: 仅在两条线上连续, 因此***偏导数存在, 函数不一定连续***

> [!example] 例如 函数 $$z=f(x,y)=\begin{cases}\frac{xy}{x^{2}+y^{2}}, &x^{2}+y^{2} \neq 0 \\0 & x ^{2}+y^{2}=0\end{cases}$$ 在(0,0)点偏导数存在, 但是不连续

## 高阶偏导数

#数学/定义 一般高阶偏导数：$$\frac{\partial \frac{\partial z}{\partial x}}{\partial x}=\frac{\partial^{2} z}{\partial x}=f''_{xx}(xy)=z''_{x x}$$


#数学/定义 高阶混合偏导数：$\frac{\partial \frac{\partial z}{\partial x}}{\partial y}$ ，$\frac{\partial \frac{\partial z}{\partial y}}{\partial x}$

### 高阶偏导数定理 

#数学/定理 
- 条件：二阶混合偏导数的结果若***连续***
- 结果：则必相等

## 偏导数的计算

1. 求对x的偏导: ***视y为常数***,对x求导
2. 求对y的偏导: ***视x为常数***,对y求导

> [!faq] 例题1: 求$z=x^{2}+3xy+y^{2}$在点$(1,2)$处的偏导数
> 解
> - 对x的偏导数(将y视为常量)
> - $z'_{x}|_{x=1,y=2}=\frac{\partial z}{\partial x}|_{x=1,y=2}=2x+3y|_{x=1,y=2}=8$
> - 对y的偏导数(将x视为常量)
> - $z'_{y}|_{x=1,y=2}=\frac{\partial z}{\partial y}|_{x=1,y=2}=3x+2y|_{x=1,y=2}=7$

> [!faq] 例题2: 求$z=x^{2}\sin2y$的偏导数
> 解
> - 对x的偏导数
> - $\frac{\partial z}{\partial x}=2x\sin 2y$
> - 对y的偏导数
> - $\frac{\partial z}{\partial y}=2x^{2}\cos2y$

> [!faq] 例题3: 设$z=x^{y},(x>0,x\neq 1)$,求证:$$\frac{x}{y} \frac{\partial z}{\partial x}+\frac{1}{\ln x} \frac{\partial z}{\partial x}=2z$$
> 证
> $$\frac{\partial z}{\partial x} =yx^{y-1}$$
> $$\frac{\partial z}{\partial y}  =x^{y}\ln x$$
> $$\begin{align}\frac{x}{y} \frac{\partial z}{\partial x}+\frac{1}{\ln x} \frac{\partial z}{\partial x} =&\frac{x}{y}\cdot yx^{y-1}+\frac{1}{\ln x}\cdot x^{y} \\= & x^{y}+x^{y} \\= & 2z\end{align}$$

> [!faq] 例题4: 求$r=\sqrt{ x^{2}+y^{2}+z^{2} }$的偏导数
> 
> $$\frac{\partial r}{\partial x}=\frac{1}{2}\cdot \frac{x}{\sqrt{ x^{2}+y^{2}+z^{2} }}=\frac{x}{r}$$
> $$\frac{\partial r}{\partial y}=\frac{y}{r},\frac{\partial r}{\partial z}=\frac{z}{r}$$

### 隐函数多元微分

例 17 $\begin{cases} x^2 + y^2 - z = 0 \\ x^2 + 2y^2 + 3z = 20 \end{cases}$ 求 $\ \frac{dy}{dx}$ , $\frac{dz}{dx}$
- $\begin{cases} y = y(x) \\ z = z(x) \end{cases}$
- 解 两边对 x 求导：$\begin{cases}2x + 2y \frac{dy}{dx} - \frac{dz}{dx} = 0 \\ 2x + 4y\frac{dy}{dx} + 6z \frac{dz}{dx} = 0\end{cases}$


例 19 设 U=f(x,y,z)有连续偏导，z=z(x,y)由 xe^x - ye^y = z e^z 确定，求 du。
解 $z=f(x,y,z(x,y))\begin{cases} z = f(x,y,z(x,y)) \\ x e^x - y e^y = z e^z \end{cases}$
- 对 x 求偏导：$\begin{cases} \frac{\partial z}{\partial x} = f_1' \cdot 1 + f_2' \cdot 0 + f_3' \cdot \frac{\partial z}{\partial x} \\e^x + x e^x = \frac{\partial z}{\partial x} e^z + z e^z \frac{\partial z}{\partial x} \end{cases}$ 
	- $\frac{\partial u}{\partial x} = f_1' + f_3' \cdot \frac{e^x(1+x)}{e^z(1+z)}$
- 对 y 求偏导：$\begin{cases} \frac{\partial u}{\partial y} = f_1' \cdot 0 + f_2' \cdot 1 + f_3' \cdot \frac{\partial z}{\partial y} \\- e^y - y e^y = \frac{\partial z}{\partial y} e^z + z e^z \frac{\partial z}{\partial y} \end{cases}$
	- $\frac{\partial u}{\partial y} = f_2' + f_3' \cdot \frac{-e^y(1+y)}{e^z(1+z)}$


## 偏导数的应用

### 求多元函数极值

例 20 求 $f(x,y)=(6x−x^{2})(4y−y^{2})$ 极值
- $\begin{cases} f_x = (6 - 2x)(4y - y^2) = 0 \Rightarrow x = 3, \ y = 0, 4 \\f_y = (6x - x^2)(4 - 2y) = 0 \Rightarrow x = 0, 6, \ y = 2 \end{cases}$
- $\therefore$ 驻点为：$(3,2)$ , $(0,0)$ , $(0,4)$ , $(6,0)$ , $(6,4)$
- (1) 在 $(3,2)$ 处 $A = -8$ , $B = 0$ , $C = -18$ , $B^{2} - AC < 0$ , $A < 0$
	- 则在 $(3,2)$ 处取极大值 f(3,2) = 36.
-  (2) 在 $(0,0)$ 处: $A = 0$ , $B = 24$ , $C = 0$ , $B^{2} - AC > 0$ 
	- $\therefore$ 无极值 ……


例 21.求 $z=x^4+y^4-x^2-2xy-y^2$ 极值
- $\begin{cases} \frac{\partial z}{\partial x} = 4x^3 - 2x - 2y = 0 \\\frac{\partial z}{\partial y} = 4y^3 - 2x - 2y = 0 \end{cases}$
- 驻点为 $(0,0)$ , $(1,1)$ , $(-1,-1)$
- ∴ $A=f_{xx}=12x^2-2$，$f_{xy}=-2$，$f_{yy}=12y^2-2$
- (1)在 $(0,0)$ 处：$A=-2$，$B=-2$，$C=-2$，$B^2-AC=0$
	- 不能通过化简转换，带入 $f(0,0)=0$，取 $\varepsilon>0$，则 $f(\varepsilon,\varepsilon)=2\varepsilon^4-4\varepsilon^2<0$，$f(\varepsilon,-\varepsilon)=2\varepsilon^4>0$
	- ∴ $(0,0)$ 不是极值点

### 求多元函数条件极值


例 35 求 $f(x,y)=x^2 + y^2 - 3$ 在 $x - y + 1 = 0$ 下极值
- $1^\circ$ 转化为无条件极值：
- $y = x + 1$
	- $f = x^2 + (x + 1)^2 - 3 = 2x^2 + 2x - 2, \ x \in \mathbb{R}$
- $\therefore f' = 4x + 2 = 0$ 驻点为 $\ x = -\frac{1}{2}, \ y = \frac{1}{2}$
- $\therefore f'' = 4 > 0$ ,则在 $\left(-\frac{1}{2}, \frac{1}{2}\right)$ 处取极小值 $\ f\left(-\frac{1}{2}, \frac{1}{2}\right) = -\frac{5}{2}$
- $2^\circ$ 拉格朗日乘数法：
- 构造 $L(x,y,\lambda) = x^2 + y^2 - 3 + \lambda(x - y + 1)$
- $\therefore \begin{cases} \frac{\partial L}{\partial x} = 2x + \lambda = 0 \\\frac{\partial L}{\partial y} = 2y - \lambda = 0 \\\frac{\partial L}{\partial \lambda} = x - y + 1 = 0 \end{cases}$
- 解得 $x + x + 1 = 0 \Rightarrow \text{驻点} \ x = -\frac{1}{2}, \ y = \frac{1}{2}$（为极小值点）
- $x - y + 1 = 0$ 确定 $y = y(x)$，则 $f = x^2 + y^2 - 3$，由一元函数求极值知 $\left(-\frac{1}{2}, \frac{1}{2}\right)$ 为极小值点，且极小值为 $f\left(-\frac{1}{2}, \frac{1}{2}\right) = -\frac{5}{2}$


例 24. 求 $f(x,y,z)=xyz$ 在 $\frac{1}{x} + \frac{1}{y} + \frac{1}{z} = \frac{1}{a}$（$x,y,z,a>0$）下的极值。
- $2^\circ$ 构造拉格朗日函数：$L = xyz + \lambda\left(\frac{1}{x} + \frac{1}{y} + \frac{1}{z} - \frac{1}{a}\right)$
- $\therefore \begin{cases} L_x = yz - \frac{\lambda}{x^2} = 0 \\L_y = xz - \frac{\lambda}{y^2} = 0 \\L_z = xy - \frac{\lambda}{z^2} = 0 \\L_\lambda = \frac{1}{x} + \frac{1}{y} + \frac{1}{z} - \frac{1}{a} = 0 \end{cases}$
- 消去 $\lambda$：$\frac{yz}{x^2} = \frac{xz}{y^2} \Rightarrow \frac{y}{x} = \left(\frac{y}{x}\right)^2 \Rightarrow y = x$
- 同理 $y = z$，故 $x = y = z$。
- $\therefore \frac{3}{x} - \frac{1}{a} = 0 \Rightarrow \text{驻点为} \ (3a, 3a, 3a)$
- 由 $\frac{1}{x} + \frac{1}{y} + \frac{1}{z} = \frac{1}{a}$ 确定 $z = z(x,y)$，则 $f = xy \cdot z(x,y)$，利用一元函数求极值方法可得 $(3a, 3a, 3a)$ 为极小值点，且极小值为 $f(3a, 3a, 3a) = 27a^3$。

例 25 某工厂要做体积为 $8m^{3}$ 的长方体有盖水箱，问长宽各是多少时，用料最省？
- $L = 2xy + 2yz + 2zx + \lambda(xyz - 8)$
- $\begin{cases}L_x = 2y + 2z + \lambda yz = 0 \\L_y = 2x + 2z + \lambda xz = 0 \\L_z = 2y + 2x + \lambda xy = 0 \\L_\lambda = xyz - 8 = 0\end{cases}$
- $\frac{y + z}{x + z} = \frac{y}{x} \Rightarrow x = y$
- $\frac{x + z}{x + y} = \frac{z}{y} \Rightarrow y = z$
- ∴驻点为 $(2,2,2)$
- 根据实际，最小值必存在，且在内部 $x>0,y>0,z>0$ 取得
- ∴唯一驻点即为最小值点，则当长宽高均为 $2m$ 时，用料最省。
