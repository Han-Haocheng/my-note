# 基础解系

设 $\xi_{1},\xi_{2},\dots,\xi_{s}$ 是齐次[[线性方程组]] $Ax=0$ 的解向量，若 $\xi_{1},\xi_{2},\dots,\xi_{s}$ [[线性相关|线性无关]]，且方程组 $Ax=0$ 的任意一个解 $\xi$ 均可由 $\xi_1, \xi_2, \dots, \xi_t$ [[线性表示]]，则称向量组 $\xi_1, \xi_2, \dots, \xi_t$ 为该方程组的一个**基础解系**


## 性质

设 $\xi_1, \xi_2, \dots, \xi_{n-r}$ 为 $Ax = \mathbf{0}$ 的一个基础解系，则具有以下性质：

**向量个数确定性**：基础解系中所含向量的个数恒等于 $n - r(A)$，该数称为解空间 $S$ 的维数，记作 $\dim(S) = n - r(A)$。

**不唯一性**：若 $n - r(A) > 0$，基础解系不唯一。解空间 $S$ 的任意一组基均可作为基础解系，且任意两个不同的基础解系是等价的。

**解的结构性（齐次通解）**：方程组 $Ax = \mathbf{0}$ 的任意解（通解）均可表示为基础解系的线性组合，即解集为：
$$ S = \{ k_1\xi_1 + k_2\xi_2 + \dots + k_{n-r}\xi_{n-r} \mid k_i \in F, i = 1, 2, \dots, n-r \} $$


**延拓性质（与非齐次方程组的关系）**：设 $\eta^*$ 是非齐次线性方程组 $Ax = b$ 的一个特解（即 $A\eta^* = b$），则非齐次方程组的解集可表示为：
$$ \{ x \mid Ax = b \} = \{ \eta^* + k_1\xi_1 + k_2\xi_2 + \dots + k_{n-r}\xi_{n-r} \mid k_i \in F \} $$
即：**非齐次通解 = 非齐次特解 + 齐次通解**。


## 定理

当齐次方程组 $Ax=0$ 的系数矩阵 $r(A)<n$ 时，即有非零解时，基础解系必存在
