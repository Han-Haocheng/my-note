# 标准型

对于二次型，如果存在可逆的线性变换 $x = Cy$，使二次型只含平方项$f = k_1y_1^2 + k_2y_2^2 + \cdots + k_ny_n^2$，其中 $k_i(i=1,2,\cdots,n)$ 为实数这种只含平方项而不含混合项的二次型，称为**二次型的标准形**.  


## 定理

(三)惯性定理：在二次型化为标准形和规范形的过程中，正、负惯性指数不变；且 $p + q = r$.  

定理 任给二次型 $f = x^TAx$，总有正交变换 $x = Py$，使二次型 $f$ 化为标准形 $f = \lambda_1y_1^2 + \lambda_2y_2^2 + \cdots + \lambda_ny_n^2$，  其中 $\lambda_1, \lambda_2, \cdots, \lambda_n$ 是二次型 $f$ 的矩阵 $A$ 的特征值.  

推论 任给二次型 $f = x^TAx$，总有可逆变换 $x = Cy$，使 $f(Cy)$ 为规范形.  

## 计算
### 用配方法化二次型为标准形的步骤  

1、用配方法化二次型为标准形的步骤：  
(1)若二次型中至少有一个平方项，不妨设 $a_{11} \neq 0$，则对所有含 $x_1$ 的项配方(经配方后所余各项不再含 $x_1$)，如此继续配方，直至每一项都包含在各完全平方项中，引入新变量 $y_1, y_2, \cdots, y_n$，由 $y = C^{-1}x$，得 $f = \lambda_1y_1^2 + \lambda_2y_2^2 + \cdots + \lambda_ny_n^2$；  
(2)若二次型中不含平方项，只有混合项，不妨设 $a_{12} \neq 0$，则可令 $x_1 = y_1 + y_2, x_2 = y_1 - y_2, x_3 = y_3, \cdots, x_n = y_n$，经此坐标变换，二次型中出现 $a_{12}y_1^2 - a_{12}y_2^2$ 后，再按 1)进行配方.

### 通过正交变换法转换为二次型  


用正交变换化二次型为标准形的步骤如下：  
1. 将二次型表示为矩阵形式 $f = x^TAx$，写出其矩阵 $A$；  
2. 求出 $A$ 的全部特征值 $\lambda_i$，设 $\lambda_i$ 是 $k_i$ 重根；  
3. 对每个特征值 $\lambda_i$，解方程组 $(A - \lambda_iE)x = 0$，求得基础解系，即属于特征值 $\lambda_i$ 的线性无关的特征向量；  
4. 将 $A$ 的属于同一特征值的特征向量正交化；  
5. 将全部特征向量单位化；  
6. 以正交单位化后向量为列，且按 $\lambda_i$ 在对角线的主对角线上的位置构成正交矩阵 $Q$；  
7. 令 $x = Qy$，则 $f = \lambda_1y_1^2 + \lambda_2y_2^2 + \cdots + \lambda_ny_n^2$.

注意：
(1)正交变换中的变换矩阵 $Q$ 是一个正交矩阵，通过正交变换得到的标准形中平方项的系数是矩阵的特征值，而配方法中的坐标变换矩阵 $C$ 是一个可逆矩阵，其平方项的系数不一定是矩阵的特征值；
(2)虽然两种方法得到的标准形不一样，但是正、负惯性指数是唯一确定的.
