# 规范形


如果标准形的系数 $k_1, k_2, \cdots, k_n$ 只在 $1, -1, 0$ 三个数中取值，也就是 $f = y_1^2 + y_2^2 + \cdots + y_p^2 - y_{p+1}^2 - \cdots - y_{p+q}^2$ 则该二次型称为规范形. 则 $f$ 的**规范形**便可确定为 $f = y_1^2 + y_2^2 + \cdots + y_p^2 - y_{p+1}^2 - \cdots - y_r^2$  
- 其中正系数项数 $p$ 称为**正惯性指数**
- 负系数项数 $q$ 称为**负惯性指数**.  
- 正惯性指数与负惯性指数的差称为**符号差**

二次型化为规范形是唯一的


## 性质

二次型的秩等于正惯性指数与负惯性指数的和即 $r=p+q$
