# 合同

若存在可逆矩阵 $C$，使 $C^{T}AC=B$，则称 $A$ 与 $B$ **合同**
- 记为 $A \simeq B$
- 并称由 A 到 B 的变换为**合同变换**，C 称为**合同变换的矩阵**．


本质：若矩阵 $A$ 与 $B$ 合同，即 $B = C^T A C$，则  
二次型 $f = x^T A x \xlongequal{x = Cy} (Cy)^T A (Cy) = y^T (C^T A C) y = y^T B y$  

合同矩阵是指同一个二次型的两个不同的阶段。

## 性质

$A$ 与 $B$ 是同型矩阵，则 $A$ 与 $B$ 等价 $\Leftrightarrow r(A)=r(B)$；  

若 $A$ 与 $B$ 是同阶矩阵，则 $A$ 与 $B$ 相似 $\Leftrightarrow$ 存在可逆矩阵 $P$，使得 $P^{-1}AP=B$；  

若 $A$ 与 $B$ 是对称同阶矩阵，则 $A$ 与 $B$ 合同 $\Leftrightarrow A$ 与 $B$ 具有相同的正负惯性指数。  

若实对称矩阵 $A$ 与 $B$ 相似，则 $A$ 与 $B$ 合同.
