# 内积
$n$ 维*列向量* $\alpha = \begin{pmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{pmatrix}$，$\beta = \begin{pmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{pmatrix}$ 的**内积**：$(\alpha, \beta) = \alpha^T \beta = x_1 y_1 + x_2 y_2 + \cdots + x_n y_n$；*行向量* $\alpha = (x_1, x_2, \ldots, x_n)$，$\beta = (y_1, y_2, \ldots, y_n)$ 的**内积**：$(\alpha, \beta) = \alpha \beta^T = x_1 y_1 + x_2 y_2 + \cdots + x_n y_n$.


注 内积是一个数，是两个向量对应分量的乘积之和。


## 性质

性质 1 非负性：$(\alpha, \alpha) \geq 0$，且 $(\alpha, \alpha) = 0 \Leftrightarrow \alpha = 0$；
> [!cite]- 证明
> 分析 设 $\alpha = (x_1, x_2, \ldots, x_n)$，则
> $(\alpha, \alpha) = x_1^2 + x_2^2 + \cdots + x_n^2 \geq 0$
> 且 $(\alpha, \alpha) = 0 \Leftrightarrow x_1 = x_2 = \cdots = x_n = 0 \Leftrightarrow \alpha = 0$.

性质 2 对称性：$(\alpha, \beta) = (\beta, \alpha)$；

性质 3 齐次性：$(k\alpha, \beta) = k(\alpha, \beta)$；

性质 4
- $(\alpha_1 + \alpha_2, \beta) = (\alpha_1, \beta) + (\alpha_2, \beta)$
- $(\alpha, \beta_1 + \beta_2) = (\alpha, \beta_1) + (\alpha, \beta_2)$
- $(\alpha_1 + \alpha_2, \beta_1 + \beta_2) = (\alpha_1, \beta_1) + (\alpha_2, \beta_1) + (\alpha_1, \beta_2) + (\alpha_2, \beta_2)$


# 模

$\|\alpha\| = \sqrt{(\alpha, \alpha)} = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}$ 为 $\alpha$ 的**长度（模、范数）**
- 特别地，长度为 1 的向量称为*单位向量*

## 运算法则


性质 5 存在向量 $\alpha$，自身的内积 $(\alpha,\alpha)=\left\|\alpha\right\|^{2}$

### 单位化

对 $R^n$ 中任一非零向量 $\alpha$，令 $\beta = \frac{1}{\|\alpha\|}\alpha$，则 $\beta$ 是一*单位向量*，将这一过程称为把向量 $\alpha$ 单位 **单位化**
- 单位化：将非单位向量化为单位向量



# 正交

若 $(\alpha, \beta) = 0$，则称向量 $\alpha$ 与 $\beta$ **正交**，
- 记作 $\alpha \perp \beta$ 

两两正交的非零[[向量组]] 称为**正交向量组**
- 特别地，由一个非零向量组成的向量组是正交向量组．

注 (1) 零向量与任意向量都正交；
(2) 与本身正交的向量仅为零向量．




定义 5.8 每个向量均为*单位向量*的*正交向量组*称为**标准正交向量组（单位正交向量组）**．


注 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 为标准正交向量组，当且仅当$(\alpha_i, \alpha_j) = \begin{cases} 0, & i \neq j \\ 1, & i = j \end{cases} (i, j = 1, 2, \ldots, s).$


## 定理

定理 5.4 正交向量组必线性无关

- 已知 $\alpha_{1},\alpha_{2},\dots,\alpha_{s}$ 为正交向量组
- 设$k_{1}\alpha_{1}+k_{2}\alpha_{2}+\dots+k\alpha_{s}=0$
- 与向量 $\alpha_{1}$ 做内积 $(\alpha_{1},k_{1}\alpha_{1}+k_{2}\alpha_{2}+\dots+k_{s}\alpha_{s})=(\alpha_{1},0)=0$
- 有 $k_{1}(\alpha_{1},\alpha_{1})+k_{2}(\alpha_{1},\alpha_{2})+\dots+k_{s}(\alpha_{1},\alpha_{s})=0$
