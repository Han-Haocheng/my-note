# 相似矩阵

设$A,B$都是$n$阶矩阵，若存在可逆矩阵$P$，使$P^{-1}AP=B$，则称$B$是$A$的**相似矩阵**，$A$和$B$相似，$P^{-1}AP$是对$A$做相似变换


## 相似对角化

若对角 $A$ 能与对角矩阵 $\Lambda$ 相似，则称矩阵 $A$ 可**相似对角化**，即 $P^{-1}AP=\Lambda$，即 $A$ 可以与对角矩阵相似


## 性质

相似矩阵的迹、特征值、行列式、秩相同


若 A 与 B 相似，则$A^T \sim B^T$，$A^{-1} \sim B^{-1}$，$(A^n \sim B^n)$，$A^* \sim B^*$相似.
- 证：$P^{-1}AP = B$
- $B^n = (P^{-1}AP)^n = (P^{-1}AP)(P^{-1}AP)\cdots(P^{-1}AP) = P^{-1}A^nP$
- $B^* = (P^{-1}AP)^* = P^*A^*(P^{-1})^* = \left[(P^*)^{-1}\right]^{-1}A^*(P^*)^{-1} = \alpha^{-1}A^*\alpha$
