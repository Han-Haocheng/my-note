# 线性代数


## 考研

- 选择 x3：有特点

- 基础：行列式、矩阵
- 主题：向量组、方程组
- 应用：相似理论

| 判断条件              | A可逆（可逆矩阵）                                  | A不可逆（奇异矩阵）                                 |
| ----------------- | ------------------------------------------ | ------------------------------------------ |
| 伴随矩阵法             | $\begin{vmatrix}A\end{vmatrix}\neq 0$      | $\begin{vmatrix}A\end{vmatrix}= 0$         |
| 初等行变换             | $B$ 中没有零行                                  | $B$ 中有零行                                   |
| 秩（矩阵的秩）           | $r(A) = n$                                 | $r(A) < n$                                 |
| 列向量组线性相关性         | $\alpha_1, \alpha_2, \dots, \alpha_n$ 线性无关 | $\alpha_1, \alpha_2, \dots, \alpha_n$ 线性相关 |
| 齐次线性方程组 $Ax = 0$  | 只有零解                                       | 有非零解                                       |
| 非齐次线性方程组 $Ax = b$ | 有唯一解                                       | 无解或有无穷多解                                   |
| 特征值               | $0$ 不是 $A$ 的特征值                            | $0$ 是 $A$ 的特征值                             |
|                   |                                            |                                            |
