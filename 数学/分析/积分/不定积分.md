# 不定积分


#数学/定义 **原函数**

- 如果在区间I上可导函数$F(x)$的导函数$f(x)$
	- 即对任意$x \in I$都有$F'(x)=f(x)$(或$dF(x)=f(x)dx$)
- 那么函数 $F(x)$ 称为 $f(x)$ (或 $f(x)\mathrm{d}x$)在区间 $I$ 上的一个**原函数**
- 记作
	- $\frac{d}{dx}\left[ \int f(x) \, dx \right]=f(x)$
	- $d\left[ \int f(x) \, dx \right]f(x)dx$
	- $\int F'(x) \, dx=F(x)+\mathbb{C}$
	- $\int  \, dF(x)=F(x)+\mathbb{C}$



#数学/定义 **不定积分**：在区间$I$上函数$f(x)$的带有任意常数项的*原函数*，称为$f(x)$或$f(x)\mathrm{d}x$在区间$I$上的不定积分
- 记做$$\int f(x) \, dx=F(x)+\mathbb{C}$$
- 积分号$\int$
- 被积函数$f(x)$
- 被积表达式$f(x)\mathrm{d}x$
- 表明积分变量$dx$

>[!warning]+ 注意：不定积分结果一定有常数C

*不定积分的几何含义*：积分的结果为一组平行的曲线簇


## 定理

### 原函数存在定理

#数学/定理 原函数存在定理：*连续函数一定有原函数*
- 如果函数$f(x)$在区间$I$上连续
- 那么在区间$I$上存在可到函数$F(x)$，使对任意一$x \in I$，都有$[F(x)+C]'=f(x)$
- 如果 $f(x)$ 在区间 $I$ 上有第一类间断点，则 $f(x)$ 在 区间 $I$ 上没有原函数

### 幂函数积分

### 指数函数积分

#数学/公式/积分 指数函数积分

-  自然**指数**积分：$$\int e^{x} \, dx=e^{x}+\mathbb{C}$$
-  自然**对数**积分：$$\int \ln{x} \, \mathrm{d}x=x\ln{x}-x+\mathbb{C}$$
- 积分->自然对数函数：$$\int \frac{1}{x} \, dx=\ln \mid x\mid+\mathbb{C}$$
-  指数积分：$$\int a^{x} \, dx=\frac{a^{x}}{\ln a}+\mathbb{C}$$           


### 三角函数积分 

#数学/公式/积分 三角函数积分
- 正弦积分$$\int \sin x \, dx=-\cos x+\mathbb{C}$$
- 余弦积分$$\int \cos x \, dx=\sin x+\mathbb{C}$$                                
- 正切积分$$\int \tan x \, \mathrm{d}x=-\ln{\lvert{\cos x}\rvert}+\mathbb{C}$$
- 余切积分$$\int \cot x \, \mathrm{d}x=\ln{\lvert{\sin x}\rvert}+\mathbb{C}$$
- 正割积分$$\int \sec x \, \mathrm{d}x=\ln{\lvert{ \tan x+\sec x }\rvert}+\mathbb{C}$$
- 余割积分$$\int \csc x \, \mathrm{d}x=-\ln{\lvert{\cot x-\csc x}\rvert}+\mathbb{C}$$
- 积分->正切函数$$\int \sec ^{2}x \, dx=\tan x+\mathbb{C}$$ 
- 积分->余切函数$$\int \csc ^{2}x \, dx=-\cot x+\mathbb{C}$$
- 积分->正割函数$$\int \sec x\tan x \, dx=\sec x+\mathbb{C}$$
- 积分->余割函数$$\int \csc x\cot x \, dx=-\csc x+\mathbb{C}$$

### 第二类换元积分法

#数学/法则

- 设$x=\psi(t)$是单调的可导函数，且$\psi'(t)\neq 0$
	- 又设$f[\psi(t)]\psi'(t)$具有原函数
- 则有换元公式$$\int f(x) \, dx =\left[ \int f[\psi(t)]\psi'(t) \, dt  \right]_{t=\psi^{-1}(x)}$$
	- $x=\psi (t)$的反函数$\psi^{-1}(x)$

#### 三角代换

   - 被积函数含有 $f(\sqrt{a^2 - x^2})$，常用换元公式 $x = a \sin t, t \in (-\frac{\pi}{2}, \frac{\pi}{2})$
   - 被积函数含有 $f(\sqrt{a^2 + x^2})$，常用换元公式 $x = a \tan t, t \in (-\frac{\pi}{2}, \frac{\pi}{2})$
   - 被积函数含有 $f(\sqrt{x^2 - a^2})$，常用换元公式 $x = a \sec t, t \in (0, \frac{\pi}{2})$；

例18 $\int x^3 \sqrt{4 - x^2} dx$

- $\therefore$ 原式 $= \int 8 \sin^3 t \cdot 2 \cdot \cos t \cdot 2 \cos t dt = 32 \int \sin^2 t \cos^2 t \sin t dt$
- $= -32 \int (1 - \cos^2 t) \cos^2 t d(\cos t)$
- $= -32 \left( \frac{1}{3} \cos^3 t - \frac{1}{5} \cos^5 t \right) + C$

- $=-\frac{4}{3}(4-x^{2})^{3/2}+\frac{1}{5}(4-x^{2})^{5/2}+\mathbb{C}$

例2 $\int \frac{1}{x^2 \sqrt{1 + x^2}} dx$
- 令 $x = \tan t, t \in (-\frac{\pi}{2}, \frac{\pi}{2})$，则 $dx = d(\tan t)$

$$\begin{align} & (uv)'=u'v+uv' \\ & uv'=(uv)'-u'v \\ & \int uv' \, dx=\int (uv)' \, dx-\int u'v \, dx \\ & \int u \, dv=uv-\int v \, du & \end{align}$$

### 有理分式函数积分

#### 待定系数法

当分母是多个因式相乘形式的时候，可拆成多个分母是因式的和
1. 设分式等于(多个分子是待定项，分母是因式的分式)和
2. 分式通分，分子是一个方程
3. 提取公因子 x，变成 $G(x)=F_{1}(A,B,\dots)x^{n}+F_{2}(A,B,\dots)x^{n-1}+\dots+F_{n}(A,B,\dots)x+F_{n+1}(A,B,\dots)$ 形式
4. $G(x)$ 与原分子相等，根据 x 的次数进行对齐
5. 可得到方程组
6. 解方程组

#### 分子是常数

1. $Q(x)$ 的最高次项是一次：直接换元运算
2. $Q(x)$ 的最高次项是二次
	- $Q(x)$ 尝试因式分解，拆成两个分式相减，换元计算
	- $Q(x)$ 尝试提取成 $\frac{c}{(x\pm a)^{2}+ b}$ 形式，将 $b$ 转换成 1，三角函数，换元积分
	- $Q(x)$ 尝试提取成 $\frac{c}{(x\pm a)^{2}- b}$，用 t 替代 $x\pm a$，重复尝试
3. $Q(x)$ 的最高次项是高次
	- $Q(x)$ 尝试因式分解，待定系数法，拆分成一个分子为常数的两个分式和

#### 分子是多项式

- $Q(x)$ 的最高次项是高次：因式分解，通过待定系数法






例13 $\int \frac{2}{(1+x)(1+x^2)} dx$
- 令 $\frac{2}{(1+x)(1+x^2)} = \frac{A}{1+x} + \frac{Bx + C}{1 + x^2}$．
- ∴ $2 = A(1 + x^2) + (Bx + c)(1 + x)$．（$x$对$\forall x$成立）
- 令$x = -1 \Rightarrow 2 = 2A + 0 \Rightarrow A = 1$，$x = 0 \Rightarrow 2 = 1 + c \Rightarrow c = 1$
- 令$x = 1 \Rightarrow 2 = 2 + (B + 1) \cdot 2 = 4 + 2B \Rightarrow B = -1$
- ∴ 原式$= \int \frac{1}{1 + x} dx + \int \frac{-x + 1}{1 + x^2} dx$
- $= \ln|1+x| - \int \frac{x}{1+x^2} dx + \int \frac{1}{1+x^2} dx$
- $\frac{1}{2} d(1+x^2)$
- $= \ln|1+x| - \frac{1}{2} \ln(1+x^2) + \arctan x + C$．


例14 $\int \frac{x-3}{(x-1)(x^2-1)} dx$
- 令 $\frac{x-3}{(x-1)(x^2-1)} = \frac{x-3}{(x-1)^2(x+1)} = \frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{C}{x+1}$

### 特殊函数积分

例18 $f(\ln x) = \frac{\ln(1+x)}{x}$ 求 $\int f(x) dx$
- 令 $\ln x = t$，$x = e^t$
- 则 $f(t) = \frac{\ln(1+e^t)}{e^t}$ 即 $f(x) = \frac{\ln(1+e^x)}{e^x}$
- ∴ $\int f(x) dx = \int \frac{\ln(1+e^t)}{e^t} dt = \int -\ln(1+e^t) d(e^{-t})$
- $= -\left( e^{-t} \cdot \ln(1+e^t) - \int e^{-t} \cdot \frac{e^t}{1+e^t} dt \right)$
- $= -e^{-t} \ln(1+e^t) + \int \frac{e^t + 1 - e^t}{1+e^t} dt$
- $= -e^{-t} \ln(1+e^t) + t - \ln(e^t + 1) + C$
- $\int f(x) dx = -e^{-x} \ln(1+e^x) + x - \ln(e^x + 1) + C$


例19 $f(x) = \begin{cases} x^2 + 2x, & x \geq 0 \\ e^x - 1, & x < 0 \end{cases}$ 求 $\int f(x) dx$
- 令 $F(x) = \int f(x) dx = \begin{cases} \int (x^2 + 2x) dx = \frac{1}{3}x^3 + x^2 + C_1, & x \geq 0 \\ \int (e^x - 1) dx = e^x - x + C_2, & x < 0 \end{cases}$
- $\lim\limits_{x \to 0^+} F(x) = \lim\limits_{x \to 0^+} \left( \frac{1}{3}x^3 + x^2 + C_1 \right) = C_1$ 又 $F(x)$ 连续
- $\lim\limits_{x \to 0^-} F(x) = \lim\limits_{x \to 0^-} \left( e^x - x + C_2 \right) = 1 + C_2$ ∴ $C_1 = 1 + C_2$
- ∴ 令 $C_2 = C$，则 $C_1 = C + 1$
- ∴ $\int f(x) dx = \begin{cases} \frac{1}{3}x^3 + x^2 + C, & x \geq 0 \\ e^x - x + C - 1, & x < 0 \end{cases}$
