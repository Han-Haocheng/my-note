## 接口概述

接口：用于模块之间的链接，屏蔽模块内部细节，转换模块信号的交界

### 操作系统接口类型

操作系统接口(系统调用，System Call)：具体表现为操作系统实现，并提供给应用程序调用的函数

- 命令行
- 图形按钮
- 应用程序形式

#### 命令行

命令：已经包装的可执行的程序

shell程序

```C
void shell(){
	char cmd[20];
	while(1){
		scanf("%s",cmd)
		if(!fock())
			exec(cmd)
		else
			what();
	}
}
```

#### 图形按钮

图形按钮：基于消息机制实现的包含图形的界面
- 消息机制：内核中存在一个消息队列，应用程序通过获取不同的消息，从而执行对应的用户代码


### 系统接口

POSIX：IEEE指定的unix的控制系统的接口标准

| 分类   | POSIX定义          | 描述      |
| ---- | ---------------- | ------- |
| 任务管理 | fock()           | 创建一个进程  |
|      | exec()           | 执行一个程序  |
|      | pthread_create() | 创建一个线程  |
| 文件系统 | open()           | 打开一个文件  |
|      | EACCES           | 返回值，无权限 |
|      | mode_t st_mode   | 文件头结构   |
