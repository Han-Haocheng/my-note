# 线性回归

假设根据房子的大小来预测价格，通过房子大小和价格的数据集，建立线性回归方程，直线将拟合数据，输入为 1250

| 大小   | 价格  |
| ---- | --- |
| 2104 | 400 |
| 1416 | 232 |
| 1534 | 315 |
| 852  | 178 |
| ...  | ... |
| 3210 | 870 |

**training set/训练集**：Data used to train the model/用于训练模型的数据
- Univariate linear regression/单变量回归模型: linear regression with one variable/只有一个变量的线性回归模型
- 模型函数：$f_{w,b}(x)=wx+b$、$f(x)=wx+b$

![[工程科学/人工智能/attachments/20250222173300.svg]]


| 符号                  | 含义                                   |
| ------------------- | ------------------------------------ |
| $x$                 | "input" variable, feature/输入变量、特征    |
| $y$                 | "output" , "target" variable/输出、目标变量 |
| $m$                 | number of training examples/训练样例的数量  |
| $(x, y)$            | single training example/单个训练样例       |
| $(x^{(i)},y^{(i)})$ | $i^{th}$ training example/第$i$个训练样例  |
| $w,b$               | parameters/参数                        |
