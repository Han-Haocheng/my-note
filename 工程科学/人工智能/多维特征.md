# 多维特征

multiple features(variables)/多维特征：输入参数为多个变量
- feature/特征：输入参数，$x_{j}$
- number of features/特征数：输入参数的数量，$n$
- features of the training example/训练样本的单次特征向量：一次输入的特征集合，$\vec{x_{i}}$
- value of featureintraining example/训练样本的单次特征向量的具体特征：一次输入参数的一个参数，$x_{ij}$

# 多元线性回归

multiple linear regression/多元线性回归模型：$f_{\vec{w},b}(\vec{x})=\vec{w}\cdot\vec{x}+b$
- 损失函数：$J(\vec{w},b)$
- 梯度下降：
	- $w_{j}=w_{j}-a\frac{ \partial  }{ \partial w_{j} }J(\vec{w},b)$
	- $b=b-a\frac{ \partial  }{ \partial b }J(\vec{w},b)$

# 特征缩放

Feature scaling/特征缩放：用于在多元线性回归中，通过约束调整多维特征的范围，解决多维特征的步长因差距过大而导致的方向导数过大，所导致的步长过长，可以加快梯度下降速度的技术

- 参数除以最大值：$x_{i}=\frac{x_{i}}{\max}$
- mean normalized/均值归一化：$x_{i}=\frac{x_{i}-averge(x_{i})}{\max-\min}$
- Z-score normalized/Z分数归一化：$x_{i}=\frac{x_{i}-average(x_{i})}{standard\_deviation(x_{i})}$
