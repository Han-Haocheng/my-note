# 格式化表格

## 设置对齐方式

#计算机/软件应用/MSOffice/Word操作 设置对齐方式

- 通过`表格工具/布局`选项卡
	1. 选中表格
	2. 单击`表格工具/布局`选项卡->`对齐方式`组->对应对齐命令


- 对应对齐命令
	- 靠上两端对齐
	- 靠上居中对齐
	- 靠上右对齐
	- 靠中两端对齐
	- 靠中居中对齐
	- 靠中右对齐
	- 靠下两端对齐
	- 靠下居中对齐
	- 靠下右对齐

## 重复标题行

#计算机/软件应用/MSOffice/Word操作 重复标题行

- 通过`表格工具/布局`选项卡
	1. 选择要重复的行
	2. 单击`表格工具/布局`选项卡->`数据`组->`重复标题行`命令

>[!FAQ]+ 在Word文档中有一个占用3页篇幅的表格，如需将这个表格的标题行都出现在各页面首行，最优的操作方法是（B）（单选）</br>A.将表格的标题行复制到另外2页中</br>B.利用`重复标题行`功能</br>C.打开`表格属性`对话框，在列属性中进行设置</br>D.打开`表格属性`对话框，在行属性中进行设置

## 设置边框底纹

#计算机/软件应用/MSOffice/Word操作 设置边框和底纹

- 通过`表格工具/设计`选项卡
	1. 选中表格
	2. 单击`表格工具/设计`选项卡->`表格样式`组->对应样式
- 通过`边框和底纹`对话框
	1. 选中表格
	2. 单击`表格工具/设计`选项卡->`边框`->`边框和底纹`
	3. 选择`边框和底纹`对话框->`应用于`下拉列表->`单元格`选项
	4. 设置对应样式
- 通过`边框和底纹`对话框
	1. 右键选中的表格
	2. 单击`表格属性`选项
	3. 单击`表格`选项卡->`边框和底纹`按钮
	4. 选择`边框和底纹`对话框->`应用于`下拉列表->`单元格`选项
	5. 设置对应样式
- 通过`边框和底纹`对话框
	1. 选中表格
	2. 单击`开始`选项卡->`段落`组->`边框`命令->`边框和底纹`按钮
	3. 选择`边框和底纹`对话框->`应用于`下拉列表->`单元格`选项
	4. 设置对应样式
