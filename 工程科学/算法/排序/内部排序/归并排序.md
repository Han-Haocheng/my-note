# 归并排序
归并：将两个或两个以上的有序序列归并成一个有序序列的过程。

- 二路归并排序：将两个有序序列归并成一个有序序列的过程。

二路归并排序基本思想
1. 将待排序的$n$个元素看成$n$个有序的子序列，每个子序列的长度为1
2. 然后两两归并，得到$\left\lceil \frac{n}{2} \right\rceil$个长度为2或1（最后一个有序序列的长度可能为1）的有序子序列；
3. 再两两归并，得到$\left\lceil \frac{n}{4} \right\rceil$个长度为4或小于4（最后一个有序序列的长度可能小于4）的有序子序列；
4. 再两两归并，……
5. 直至得到一个长度为$n$的有序序列。

该图像展示了一个名为 `merge` 的函数，用于将两个有序序列归并成一个有序序列。函数的参数包括：两个数据类型数组 `A[]` 和 `B[]`，以及三个整数 `pa`、`pb`、`pc`（用于标记序列的边界）。代码及注释如下：  
```c
void merge (datatype A[], datatype B[], int pa, int pb, int pc) {
    int j=pb+1;          // A第二部分下标（即第二个有序子序列的起始位置）
    int k=pa;            // 指向B（归并结果存储在B中，k为B的当前写入位置）
    while (pa<=pb && j<=pc) {  // 当A的第一部分（pa到pb）和第二部分（j到pc）均未遍历完时
        if (A[pa]<A[j])       // 比较A的两个部分当前元素
            B[k]=A[pa]; pa++; // 若A[pa]更小，将其存入B，并移动pa指针
        else
            B[k]=A[j]; j++;   // 若A[j]更小，将其存入B，并移动j指针
        k++;                 // 移动B的写入位置
    }
    if (pa<=pb)              // 若A的第一部分（pa到pb）有剩余元素
        B[k..pc]=A[pa..pb];  // 将剩余元素直接复制到B中
    if (j<=pc)               // 若A的第二部分（j到pc）有剩余元素
        B[k..pc]=A[j..pc];   // 将剩余元素直接复制到B中
}
```  

算法分析  
- **时间复杂度**：$T(n)=O(n\log_2 n)$  
  归并排序的时间复杂度由“分治”过程的递归深度（$\log_2 n$）和每次归并的线性扫描（$n$）共同决定，因此整体为 $O(n\log_2 n)$。  
- **空间复杂度**：$S(n)=O(n)$  
  归并过程中需要额外的数组（如 `B[]`）存储中间结果，空间消耗与待排序元素数量 $n$ 成正比。  
- **稳定性**：是稳定的排序方法  
  归并排序在合并过程中，若遇到相等元素，会优先保留前一个序列的元素，因此不会改变相等元素的相对顺序
