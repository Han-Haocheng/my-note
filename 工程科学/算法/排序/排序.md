# 排序

排序：设有记录序列$\{R_1, R_2, \ldots, R_n\}$，其相应的关键字序列为$\{K_1, K_2, \ldots, K_n\}$，若存在某种确定的关系$K_x \leq K_y \leq \ldots \leq K_z$，其中$1 \leq x, y, z \leq n$且$x$、$y$、$z$各不相同，则将记录序列$\{R_1, R_2, \ldots, R_n\}$排成按关键字有序的序列$\{R_x, R_y, \ldots, R_z\}$的操作称为排序。
排序依据的关系是任意的，通常使用小于（递增）、大于（递减）等关系。

- 稳定排序：若有一个待排序列为$\{R_1, R_2, \ldots, R_i, \ldots, R_n\}$（$i \neq 2$），其对应的关键字序列为$\{K_1, K_2, \ldots, K_i, \ldots, K_n\}$，若$R_2$与$R_i$的关键字$K_2 = K_i$，如果排序前后，他们的相对位置保持不变，则这种排序方法是稳定的，否则是不稳定的。


- 正序：若待排序列中的记录已按关键字排好序
- 逆序：若待排序列中记录的排列顺序与排序后的排列顺序正好相反

- 单关键字排序：根据一个关键字进行的排序。
- 多关键字排序：根据多个关键字进行的排序，主要针对关键字有重复的情况。
