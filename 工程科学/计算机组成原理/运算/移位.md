# 移位

移位：数据相对于小数点进行移动
- 左移：绝对值扩大，左移1位=数值部分乘以2
- 右移：绝对值缩小，右移1位=数值部分除以2

在计算机中，移位与加减法配合，能够进行乘除运算

## 逻辑移位

逻辑移位：逻辑移位将操作数 ( 无论是有符号数还是无符号数 ) 视为无符号数

- 逻辑左移：高位移丢 , 低位添 0
- 逻辑右移：低位移丢 , 高位添 0 

## 算数移位


算术移位的规则：
- 左移 ( 规则同逻辑移位 ) : 高位移出 , 低位补 0, 若左移前后的符号位不同 , 则发生溢出
- 右移 ; 低位移出 , 高位补符号位 , 若低位的 1 移出 , 则损失精度


算数移位：符号位不变，数值按以下规则添补代码


| 符号  | 类型  | 方向  | 补位  | 丢 1  | 丢 0 |
| --- | --- | --- | --- | ---- | --- |
| 正数  |     | 左移  | 0   | 溢出   | 正确  |
| 正数  |     | 右移  | 0   | 损失精度 | 正确  |
| 负数  | 原码  | 左移  | 0   |      |     |
| 负数  | 原码  | 右移  | 0   |      |     |
| 负数  | 反码  | 左移  | 1   |      |     |
| 负数  | 反码  | 右移  | 1   |      |     |
| 负数  | 补码  | 左移  | 0   |      |     |
| 负数  | 补码  | 右移  | 1   |      |     |

正数
- 左移补 0：丢 1 溢出，丢 0 正确
- 右移补 0：丢 1 损失精度，丢 0 正确

负数原码
- 左移补 0：

| 操作   | 原码  | 反码  | 补码  |
| ---- | --- | --- | --- |
| 正数移位 | 0   | 0   | 0   |
| 负数左移 | 0   | 1   | 0   |
| 负数右移 | 0   | 1   | 1   |

## 算数移位和逻辑移位的区别
- 算数移位：有符号数的移位
- 逻辑移位：无符号数的移位

例如

| 左移  | 01010011 | 右移  | 10110010 |
| --- | -------- | --- | -------- |
| 逻辑  | 10100110 | 逻辑  | 01011001 |
| 算数  | 00100110 | 算数  | 11011001 |
