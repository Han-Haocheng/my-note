
 
 CF(Carry Flag) 进位标志 ; 反应无符号数运算是否发生溢出
 - 滥出 CF=1, 否则 CF=0

OF(Overflow Flag) 溢出标志 ; 反应有符号数运算是否发生溢出 
- 滥出OF=1, 否则 OF=0 

ZF(Zero Flag) 零标志 ; 反映运算结果是否为 0
- 结果为零ZF=1， 否则 ,ZF=0

SF(Sign Flag) 符号标志 ; 反映运算结果符号是否为 1( 负数 ) 
- 负数 SF=1, 否则 SF=0 


无符号数加减运算溢出判断 ; CF 的判断 ( 对于 n 位无符号数 )
- 如果两无符号数相加 , 结果大于 20-1 (n 为位数 ) , 则 CF=1, 否则 CF=0; 如果两无符号数相减 , 减数大于被减数 ( 也即结果不在 0 ~ 20-1 内 ) , 则 CF=1, 否则 CF=0;


有符号数加减运算溢出判断 : OF 的判断 ( 对于 n+1 位有符号数 ) 
- 如果两有符号数相加减 , 结果不在 -2n ~ 2n-1 内 , 则 OF=1, 否则 OF=0;
