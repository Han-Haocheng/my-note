# 加减运算

## 无符号数的加减运算

二进制无符号数加法
- 转换成十进制真值计算 ( 要注意溢出情况 )
- 列坚式加 ( 跟十进制竖式一样 )



二进制无符号数减法 
- 转换成十进制真值计算 ( 要注意溢出情况 )
- 列基式减 ( 根十进制坚式一样 )
- 减法转换成加法再列坚式加 ( 计算机思路 )


## 有符号数的加减运算

连同符号位一起相加，符号位产生的进位自然丢掉


有符号数加法
- 整数：$[A]_补+[B]_补=[A+B]_补\pmod 2^{n+1}$
- 小数：$[A]_补+[B]_补=[A+B]_补\pmod 2$


有符号数减法
- 转换$A-B=A+(-B)$
- 整数：$[A+(-B)]_补=[A]_补+[-B]_补\pmod 2^{n+1}$
- 小数：$[A+(-B)]_补=[A]_补+[-B]_补\pmod 2$

## 加减运算溢出判断

### 无符号数溢出

无符号数溢出：CF 标志的判断，对于 n 位无符号数


范围判断
- 相加：结果大于 $0\sim2^{n-1}$  , 则 CF=1, 否则 CF=0;
- 相减：结果不在 $-2^{n} \sim 2^{n-1}$ 内，即减数大于被减数，则 CF=1, 否则 CF=0


计算判断： $CF=Cin|Sub\oplus Cout$
- 最高位进位 $Cout$
- 减法控制位 $Cin_{sub}$ ：加法为0，减法为1

### 有符号数溢出

有符号数溢出 : OF 标志的判断，对于 n+1 位有符号数
- 正数 + 正数 , 负数 + 负数才可能溢出

范围判断
- 如果两有符号数相加减 , 结果不在 $-2^{n} \sim 2^{n-1}$ 内 , 则 OF=1, 否则 OF=0;


双符号位判断
- 溢出：双符号位不全为 0 或者全为 1
- 正溢出：双符号位为01
- 负溢出：双符号位为10  


计算判断：$OF =  C_{n}\oplus C_{n-1}$
- 最高位的进位 $C_{n}$
- 次高位的进位 $C_{n-1}$
