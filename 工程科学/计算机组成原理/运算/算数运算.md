## 加减运算溢出判断

### 无符号数溢出

无符号数加减运算溢出判断 CF 的判断 ( 对于 n 位无符号数 )
- 如果两无符号数相加 , 结果大于 $0\sim2^{n-1}$ ($n$ 为位数 ) , 则 CF=1, 否则 CF=0;
- 如果两无符号数相减 , 减数大于被减数 ( 也即结果不在 $-2^{n} \sim 2^{n-1}$ 内 ) , 则 CF=1, 否则 CF=0;



CF ( 进位 / 借位标志 ) : 用来判断最高位是否发生进位 / 借位 ( 加法是否发生进位 , 减法是否发生借位 ) , 可用来判断无符号数加减运算是否溢出
`CF=Cin|Sub` $\oplus$ `Cout`
- 最高位进位 $Cout$
- 减法控制位 $Cin_{sub}$ ：（加法 0,减法 1）

### 有符号数溢出

有符号数加减运算溢出判断 : OF 的判断 ( 对于 n+1 位有符号数 ) 
- 如果两有符号数相加减 , 结果不在 -2n ~ 2n-1 内 , 则 OF=1, 否则 OF=0;


- 双符号位判断 : 双符号位不全为 0 或者全为 1 就溢出 ,01 表示正溢出 ,10 表示负溢出 

- OF ( 溢出标志 ) : 可用来判断有符号数加减运算是否滥出 , 溢出为 1, 不溢出为 0 ( 正数 + 正数 , 负数 + 负数才可能溢出)

$OF =  C_{n}\oplus C_{n-1}$
- 最高位的进位 $C_{n}$
- 次高位的进位 $C_{n-1}$
