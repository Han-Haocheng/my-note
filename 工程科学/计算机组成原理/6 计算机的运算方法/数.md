# 数


定点数：小数点的位置是固定不变的
- 定点小数：小数点位于符号位（如果有符号）之后，所有数值之前
- 定点整数：小数点位于数值位之后

浮点数：小数点的位置通过

## BCD 码


BCD 码：用于表示 10 进制的数，每一位十进制数对应 4 位二进制数

## 无符号数

无符号数：只有数值位，没有符号位，只能表示从 0 开始的数

## 有符号数

有符号数：包含数值和一位符号，通常以 0 为中心

- 真值：带符号的数，是真正的数值
- 机器数：符号数字化的数，小数点位置是约定的结果

| 真值      | 机器数     |
| ------- | ------- |
| +0.1011 | 0 .1011 |
| -0.1011 | 1 .1011 |
| +1100   | 0 1100. |
| -1100   | 1 1100. |

### 原码表示法

原码表示法：通过在二进制数的最高位（符号位）来表示数的正负，0 表示正数，1 表示负数，其余位表示数值的绝对值


源码表示定点整数： $[x]_原=\begin{cases}0,x &  2^n>x \ge 0\\2^n-x &0\ge x >-2^n \end{cases}$
- 真值x
- 整数的位数n
- 表示范围：$[-2^n-1, 2^n-1]$


- 记作：用逗号将符号位和数值部分隔开

| 真值    | 原码                    |
| ----- | --------------------- |
| +1110 | $0, 1110$             |
| -1110 | $2^4-(-1110)=1, 1110$ |

源码表示定点小数：$[x]_原=[\begin{cases}x&1>x≥0\\1-x&0≥x>-1\end{cases}]$
- 记作：用**小数点**将符号位和数值部分分开

| 真值         | 原码                   |
| ---------- | -------------------- |
| +0.1101    | $0. 1101$            |
| -0.1101    | $1-(-0.1101)=1.1101$ |
| +0.1000000 | $0.1000000$          |
| -0.1000000 | $1.1000000$          |

| 真值            | 原码        |
| ------------- | --------- |
| $-0.0011$     | $1. 0011$ |
| $-1100$       | $1, 1100$ |
| $+0.1101$     | $0.1101$  |
| $[+0.0000]_原$ | $0.0000$  |
| $[-0.0000]_原$ | $1.0000$  |
| $[+0]_原$      | $0, 0000$ |
| $[-0]_原$      | $1, 0000$ |

特点：简单直观、不易运算

|要求|数1|数2|实际操作|结果符号|
|-|-|-|-|-|
|加法|正|正|加|正|
|加法|正|负|减|可正可负|
|加法|负|正|减|可正可负|
|加法|负|负|加|负|
