## 通用寄存器

| 32位   | 16位  | 8上位  | 8下位  |
| ----- | ---- | ---- | ---- |
| `EAX` | `AX` | `AH` | `AL` |
| `EBX` | `BX` | `BH` | `BL` |
| `ECX` | `CX` | `CH` | `CL` |
| `EDX` | `DX` | `DH` | `DL` |

**字(word)**：由两个字节组成，分别成为高位字节和地位字节


### 几条汇编指令

```z80
mov ax,18    ; ax==0012H                            
mov ah,78    ; ax==4E12H

```

```z80

```

## 物理地址

物理地址：每一个内存单元在这个空间中都有唯一的地址


### N位CPU结构

- 运算器一次处理的最大位数
- 寄存器的最大宽度
- 片内数据总线的最大宽度

### CPU访问物理地址

访问物理地址：通过地址加法器，将*段地址*和*偏移地址*进行组合，可以获取超过CPU位数的物理地址
- 8086CPU访问物理地址：`物理地址=段地址<<4+偏移地址`


### 段

段：程序员按照CPU结构将内存分割成许多大小一致的块，从而通过段地址和偏移地址获取真实地址
- 段地址是16的倍数
- $偏移地址<=2^{16} = 64KB$


检测点2.2

- 给定段地址为 `0001H`，仅通过变化偏移地址，CPU的寻址范围为 `00010H`到`1000FH`
- 有一数据存放在内存`20000H`单元中，现给定段地址为`SA`，若想用偏移地址寻到此单元，则`SA`应满足的条件是：最小为 `1001H` 最大为 `2000H`

## 段寄存器

段寄存器：提供段地址

| 含义  | 段寄存器 | 偏移地址 |
| --- | ---- | ---- |
| 代码  | `CS` | `IP` |
| 数据  | `DS` |      |
| 栈顶  | `SS` | `SP` |
| 扩展  | `ES` |      |


- 指令地址：`CS:IP`，由代码寄存器`CS`和指令寄存器`IP`保存
	- [[计算机/计算机组成原理/8 CPU的结构和功能/8.2 指令周期|具体运行方式]]
- 栈顶地址：SS:SP

### 修改CS:IP指令

- 同时修改CS:IP：jmp 段地址:偏移地址
- 用寄存器修改IP：jmp 寄存器


```z80
; 初始CS:IP=2000H:0000H
 
mov ax,0123H ; 10000H DB2301   ; AX=0123H
mov ax,0000H ; 10003H B80000   ; AX=0000H
mov bx,ax    ; 10006H 8BD8     ; BX=AX==0000H
jmp bx       ; 10008H FFE3     ; jmp 1000H:0H==10000H

mov ax,6622H ;20000H B82266    ; AX=6622H
jmp 1000:3   ;20003H EA03000010; jmp 1000H:3H==10003H
mov cx:ax    ;20008H 89C1


```

### 代码段

代码段：用于存放代码的段


检测点 2.3

- 以下代码CPU修改了*3*次IP，分别在 `mov`->`sub`,`sub`->`jmp`,`jmp`，最后IP为0

```
mov ax,bx
sub ax,ax

```
