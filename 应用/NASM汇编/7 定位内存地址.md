# 定位内存地址

## `and`和`or`指令

`and`：逻辑与，可将操作对象的相应位设置为0

`or`：逻辑或，可将操作对象的相应位设置为1

## ASCII码

ASCII码：一种特殊的英文字符编码

### 以字符形式给出数据

使用`''`将汇编中的内容标识为字符数据

```asm
assume cs:code,ds:data
data segment
	db 'unIX'
	db 'foRK'
data ends

code segment
start: mov al,'a'
	mov bl,'b'
	mov ax,4c00h
	int 21h
code ends

end start
```

### 大小写转换

大写字母+`20H`=小写字母


```
assume cs:codesg,ds:datasg
datasg segment
	db 'BaSic'
	db 'iNfOrMaTiOn'
datasg ends

codesg segment
start:
	mov ax,datasg
	mov ds,ax
	mov bx,0
	mov cx,5
s:
	mov al,[bx]
	and al,11011111b
	mov [bx],al
	inc bx
	loop s
	
	*
codesg ends
end start


```

## `[bx+idata]`

`[bx+idata]`：表示内存单元$(bx)+idata$，
- 其他写法：`[idata+bx]`、`idata[bx]`、`[bx].idata`

```ASM
; 2000:1000 BE 00 06 00 00

mov	ax,2000H
mov	DS,AX

MOV	BX,1000H
MOV	AX,[BX]
MOV	CX,[BX+1]
ADD CX,[BX+2]
```

### 用`[bx+idata]`处理数组
将datasg 中定义的第一个字符串转化为大写，第二个字符串转化为小写。

## `SI`和`DI`

`SI`/`DI`：功能类似于`BX`

```ASM
MOV BX,0
MOV AX,[BX+123]

MOV SI,0
MOV AX,[SI+123]

MOV DI,0
MOV AX,[DI+123]
```


用si和di实现将字符串'welcome to masm!'复制到它后面的数据区中。

```ASM
assume cs:codesg,ds:datasg

datasg segment
	db	'welcome to masm!'
	db	'................'
datasg ends

CODESG SEGMENT
START:
	MOV AX,DATASG
	MOV DS,AX
	MOV SI,00H
	MOV DI,10H

	MOV	CX,8H
S:	
	MOV	AX,DS:[SI]
	MOV DS:[DI],AX
	ADD SI,2
	ADD DI,2
	LOOP	S

	MOV AX,4C00H
	INT 21H
CODESG ENDS

END START
```

### `[bx+si]`和`[bx+di]`

 `[bx+si]`和`[bx+di]`：可以更加灵活的获取地址
 - 数学形式：`((DS)<<4+(BX)+(SI))`
 - 其他格式：`[bx][si]`、`[bx][di]`

```ASM
;2000:1000 BE 00 06 00 6A 22
MOV AX,2000H
MOV DS,AX

MOV BX,1000H
MOV SI,0
MOV AX,[BX+SI]    ;(AX)=00BEH
INC SI
MOV CX,[BX+SI]    ;(CX)=0600H
INC SI
MOV DI,SI         ;(DI)=0002H
ADD CX,[BX+DI]    ;(AX)=0606H
```

### `[bx+si+idata]`和`[bx+di+idata]`

`[bx+si+idata]`和`[bx+di+idata]`：可以更加灵活的获取地址
- 其他格式：`[bx+200+si]`、`[200+bx+si]`、`200[bx][si]`、`[bx].200[si]`、`[bx][si].200`

```ASM
;2000:1000 BE 00 06 00 6A 22
MOV AX,2000H
MOV DS,AX

MOV BX,1000H
MOV SI,0
MOV AX,[BX+2+SI]    ;(AX)=0006H
INC SI
MOV CX,[BX+2+SI]    ;(CX)=6A00H
INC SI
MOV DI,SI           ;(DI)=0002H
MOV BX,[BX+2+DI]    ;(BX)=226AH
```

## 不同寻址方式的灵活运用

- `[idata]`：直接寻址
- `[bx]`：寄存器间接寻址
- `[bx+idata]`：基址寻址
- `[si+idata]`：变址寻址
- `[bx+si]`：基址变址寻址
- `[bx+si+idata]`：带位移的基址变址寻址
