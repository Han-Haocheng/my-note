# 结构化程序的算法设计

结构化程序设计方法的基本思路：把一个复杂问题的求解过程分阶段进行，每个阶段处理的问题都控制在人们容易理解和处理的范围内
- 将一个大的程序按功能分解成为一些功能单一、结构清晰、接口简单、容易理解的功能模块
- 采用自顶向下、逐步分解、分而治之


结构化程序：问题可以分解成相互独立的几个部分，每个独立部分可以通过简单的语句或结构来实现
- 分解问题的过程就是算法设计的过程

## 算法

算法：解决问题的方法和要遵循的步骤，描述了程序要执行的操作及操作的步骤顺序，描述程序的功能


### 结构化算法性质

- 算法名称：给算法命名，是为了方便算法的描述，在C语言中，算法的名字通常就是函数名
- 输入：算法应有输入的数据或初始条件
- 输出：算法通常会有一个或多个输出，是对输入数据加工后的结果
- 有效性：算法的每一步都是可执行的，可通过人工计算的
- 正确性：算法的结果必须是正确的，可验证的
- 有限性：任何算法必须在执行有限条指令后结束


### 结构化算法的结构

- 顺序结构：程序在执行过程中是按语句的先后顺序来执行的，每一条语句都代表着一个功能
- 选择结构：程序在执行过程中，会根据条件的不同有选择的执行不同的功能
- 循环结构：程序在执行过程中，在一定的时间段内或一定的条件下，重复地执行某个功能，直到时间已到或条件不再满足

### 结构化算法的描述方法

常用的描述方法有自然语言、流程图、伪代码等

类自然语言：用类自然语言表示算法，比如汉语、英语或其他语言。
- 特点：通俗易懂，简单明了

要求从键盘输入3个数，找出其中最小的那个数，将其输出到屏幕。请给出解决这个问题的算法。

分析：程序对于从键盘输入的3个数必须用3个变量来保存，分别为a，b，c代表输入的3个数，另外，还需要一个变量min来保存最小的那个数。先比较a和b的值，把数值小的放入min中；再将min与c比较，又把数值小的放入min中。经过两次比较，min中已存放的是a，b，c 3个数中最小的数。把min的值输出就是所需结果。

算法步骤：将C语言代码翻译成中文
1. 输入三个数，其值分别赋给三个变量a，b，c；
2. 比较a与b的值，如果a < b 则min = a；否则min = b；
3. 比较c与min的值，如果c < min，则min = c；
4. 输出最后结果min的值


计算3个数的和的平均值

分析：程序对于3个数必须用3个变量保存，分别为n1,n2,n3,输入3个数给n1,n2,n3。需要1个变量sum保存三个数的和，让sum等于n1加n2再加n3的值。需要1个变量res保存平均值作为结果，让res存放sum除以3的值，res就是最终所需的结果

算法步骤
1. 输入3个数，分别赋值给三个变量n1,n2,n3
2. 将n1,n2,n3加起来，赋值给变量sum
3. 将sum除以3,赋值给变量res
4. 输出最后结果res的值
