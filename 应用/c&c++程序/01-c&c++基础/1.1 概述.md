## C语言特点
- 高效性：比汇编低10%-20%，并支持位操作以及物理地址访问
- 可移植性：能在多个平台进行编译
- 结构化编程
- 内存管理
- 简洁性：包含36个关键字

## C语言结构

```c
#include<stdio.h>
main()
{
	print("Hello,World!\n");
}
```

1. `include`是文件包含命令，扩展名为`.h`的文件称为头文件，表示在程序中要用到这个文件中的函数
2. `main`是主函数的函数名，表示这是一个主函数，一个C语言源程序只允许有一个main函数
3. printf是函数调用语句
4. main()函数中的内容必须放在{}中
