1. 文件和目录管理命令：  
	- ls：列出目录内容![[应用/Linux系统/Pasted image 20260309214533.png]]
	-  cd：改变当前工作目录![[应用/Linux系统/Pasted image 20260309214625.png]]
	-  pwd：显示当前工作目录![[应用/Linux系统/Pasted image 20260309214647.png]]
	-  mkdir：创建新目录  
	-  rmdir：删除空目录  
	-  cp：复制文件或目录  
	-  rm：删除文件或目录  
	-  mv：移动文件或目录  
	-  ln：创建文件链接
2. 文件操作命令：  
	-  cat：查看文件内容
	-  more：分页显示文件内容  
	-  less：分页显示文件内容，并支持前后翻页  
	-  head：显示文件的前几行  
	-  tail：显示文件的后几行  
	-  touch：修改文件时间戳或创建新文件  
	-  grep：在文件中查找指定模式  
	-  sed：使用指定规则编辑文件内容
3. 系统管理命令：  
	-  ps：显示当前进程的状态  
	-  top：实时显示系统资源使用情况  
	-  kill：终止指定进程  
	-  shutdown：关机或重启系统  
	-  reboot：重启系统  
	-  su：切换用户身份  
	-  sudo：以其他用户身份执行命令
4. 网络命令：  
	-  ifconfig：显示或配置网络接口信息  
	-  ping：测试与指定主机的连通性  
	-  netstat：显示网络连接、路由表和网络接口信息  
	-  nslookup：查询域名的 IP 地址  
	-  wget：从网络上下载文件  
	-  scp：使用安全加密的方式在本地和远程主机之间传输文件
5. 压缩和解压命令：  
	-  tar：打包或解压文件  
	-  gzip：压缩文件  
	-  gunzip：解压缩文件  
	-  zip：压缩文件或目录  
	-  unzip：解压缩文件
