# 交换排序

交换排序：==两两比较==，如果发生逆序则交换，直到所有记录都排好序为止。

## 冒泡排序

起泡排序

基本思想：重复地遍历要排序的数列，一次比较两个相邻元素的关键字，如果顺序错误就将它们进行交换，直到没有任何一对关键字需要交换为止，这个过程会使关键字最大的元素如气泡一样逐渐“上浮”。

在进行第i遍过程时，只需对前$n+1-i$个数据元素进行比较，比较结束后，第$n+1-i$位置即为第i遍比较过程中产生的最大数据元素，同时还理顺了部分其他数据元素。
```c
void BubbleSort (SqList &L) {
    // 对顺序表L做冒泡排序
    m=L.length-1; flag=1;
    while ((m>0)&&(flag=1)) {
        flag=0;          // flag置为0，如果本趟排序没有发生交换，则不会执行下一趟
        for (j=1; j<=m; j++) {
            if (L.r[j].key<=L.r[j+1].key)
				continue;
			flag=1;      // flag置为1，表示本趟排序发生了交换
			t=L.r[j];    // 交换前后两个记录
			L.r[j]=L.r[j+1];
			L.r[j+1]=t;
        }
        --m;
    }
}
```

算法分析：
- 最坏/平均时间复杂度：$O(n^2)$
- 空间复杂度：$O(1)$
- 稳定性：是稳定的排序方法

## 快速排序
快速排序：1962年由Hore提出的一种排序算法

基本思想
- 通过对关键字的比较和交换，以待排序序列中的某个数据为基准元素（或称枢轴元素，可以是第一个、最后一个或中间位置的元素）
- 将待排序序列分成两个部分，其中左半部分数据小于等于基准元素，右半部分数据大于基准元素。然后，对左右两部分分别进行快速排序的递归处理，直到整个序列按关键字有序为止

```c
void quicksort (datatype data[], int low, int high) {
    if(low<high) {
        int pivotkey=partition (data,low,high);  // 确定枢轴
        quicksort (data, low, pivotkey-1);      // 递归排序
        quicksort (data, pivotkey+1, high);     // 递归排序
    }
}
int partition (datatype data[], int low, int high) {

```
算法分析：
- 时间复杂度：
	- 最好情况（当每次都将待排序序列划分成两个长度相等的子列）$T(n)=O(n\log_2 n)$
	- 最坏情况（如每次划分都只得到一个子列时）$T(n)=O(n^2)$
- 平均情况$T(n)=O(n\log_2 n)$
- 空间复杂度（需栈空间实现递归）
	- 最坏情况，要进行$n-1$次递归调用：$S(n)=O(n)$
	- 一般情况，其容量与递归调用的最大层数一样：$S(n)=O(\log_2 n)$
- 稳定性：是不稳定的排序方法
