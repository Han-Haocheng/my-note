# 非监督学习

**Unsupervised learning/非监督学习**：Find structure in unlabele data./从无标签的数据中寻找结构

## 聚类算法

**Clustering/聚类算法**：group similar data points together/将未标记的数据放入不同的集群中

举例：从文章中寻找相似词

## 异常检测算法

**Anomaly detection/异常检测**：Find unusual data points/检测异常数据点

## 压缩算法

**Dimensionality reduction/压缩算法**：Compress data using fewer numbers./从大数据中提取小数据集，丢失尽可能少的信息。
