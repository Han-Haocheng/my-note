## 启动流程

操作系统启动流程(X 86)

1. CPU 处于实模式
	- 实模式：寻址方式为 `CS<<4:IP`
2. `CS=0xFFFF;IP=0x0000`
	- 段寄存器(CS)：存储当前执行指令所在代码段的基地址
	- 偏移寄存器(IP)：指定内存地址的在代码端基地址的偏移量
3. 寻址 ROM 中的 BIOS 映射区起始地址
	- ROM BIOS 映射区：`0xFFFF0`
4. 检查 RAM、键盘、显示器、软硬磁盘等硬件设备
5. 将磁盘 0 磁道 0 扇区写入 `0x7C00`
	- 系统引导扇区(0 磁道 0 扇区)：启动设备的第一个扇区
6. 设置 `CS=0x07C0;IP=0x0000`
