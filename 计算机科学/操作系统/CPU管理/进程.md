# 进程

**进程**：程序的实体，引入进程的目的是使多个程序能并发执行，提高系统的资源利用率和吞吐量。
- 是一段程序的一次执行过程(*动态性，与程序的主要区别)*。
- 是多个程序段在同一时间间隔内正确运行过程(*体现异步性*)。
- 是具有独立功能的程序在一个具有标准数据结构上运行的过程，是系统资源分配和调度的基本单位(*体现结构性*)


## 特征

- 结构性：进程包括程序代码、堆栈段、数据段、堆等内容。每个进程都必须包括进程控制块，进程控制块是进程存在唯一标志。因此从结构上看，一个进程实体是由程序段、数据段和进程控制块三部分组成的。
- 动态性：主要体现在进程是程序的执行过程，每段程序在运行时都要经历进程的创建、状态转换、终止等过程，是在一个生命周期内动态的变化
	- 动态性是进程区别程序的最基本特征。
- 并发性：指在同一段时间间隔内多个进程实体同时存在于内存中，并且在同一段时间内正确地运行。并发性也是操作系统最基本的特征，体现了进程引入的目的，是实现在一个处理机上同时完成多个程序的正常运行，提高系统资源的利用率。


- 独立性：进程是资源分配与调度的基本单位，一个进程实体拥有独立的资源，能够独立地运行。
- 异步性：指进程按异步方式运行的，即按各自独立的、不可预知的速度向前推进


进程是进程实体的运行过程，是系统进行资源分配和调度的一个独立单位。

### 多进程交替

多进程交替：队列操作+调度+切换

#### 切换流程

1. 启动磁盘读写
2. `pCur.state = 'W';`
3. 将`pCur`放到`DiskWaitQueue;`
4. `schedule();`

```c
void schedule(){
	pNew = getNext(ReadyQueue);
	switch_to(pCur, pNew);
}
```

#### 调度方式

调度
- FIFO：公平策略，没有任务区别
- Priority：优先级策略，会产生饥饿进程

#### 切换方式

```c
void switch_to(pCur,pNew){
	pCur.ax = CPU.ax;
	pCur.bx = CPU.bx;
	//...
	pCur.cs = CPU.cs;

	CPU.ax = pNew.ax;
	CPU.bx = pNew.bx;
	//...
	CPU.cs = pNew.cs;
	CPU.retpc = pNew.pc;
}

```

### 多进程影响

问题：多进程同时存在于内存中，会导致进程跨越访问另一个进程数据

解决方式：在每个进程中建立一个映射表，映射代码的数据访问


### 多进程合作

打印过程
1. 应用进程提交打印任务
2. 打印任务放进打印队列
3. 打印进程从队列中取出任务
4. 打印进程控制打印机打印

问题：进程会抢夺打印资源，类比于消费者-生产者问题
解决方式：进程同步，在写入共享资源时，阻断其他进程访问该资源


## 进程同步


### 生产者-消费者问题


```c
// P0
while(true){
	if(counter==BUFFER_SIZE)
		sleep();
	buffer[in]=item;
	in=(in+1)%BUFFER_SIZE;
	counter++;
	if(counter==1)wakeup(P1)
}

//P1
while(true){
	if(counter==0)
		sleep();
	item = buffer[out];
	out=(out+1)%BUFFER_SIZE;
	counter--;
	if(counter==BUFFER_SIZE-1)wakeup(P0)
}
```

问题：无法处理多个生产者或多个消费者被唤醒的问题
解决：引入信号量，每一次睡眠都记录一次，每一次唤醒都减少一次记录

### 信号量实现
```c
struct semaphore{
	int value;
	PCB *queue;
}

void Proberen(semaphore s){
	s.value--;
	if(s.value<0){
		sleep(s,queue);
	}
}

void Verhogen(semaphore s){
	s.value++;
	if(s.value<=0){
		wakeup(s.queue);
	}
}
```

### 解决生产者-消费者问题


```c
int fd = open("buffer.text");

write(fd,0,sizeof(int)); //写in
write(fd,0,sizeof(int)); //写out

semaphore full = 0;
semaphore empty = BUFFER_SIZE;
semaphore mutex = 1;

void Producer(item){
	P(empty);
	
	loc = setloc(fd);
	P(mutex);
	write(item,loc);
	V(mutex);
	
	V(full);
}

void Consumer(item){
	P(empty)
	
	loc = getloc(fd);
	P(mutex)
	item = read(loc);
	V(mutex);
	printf(item);
	
	V(full);
}
```
