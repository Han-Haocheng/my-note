# 信号量临界保护

信号量竞争：当同时修改信号量时，会发生不可避免的竞争
- 竞争条件（Race Condition）：和调度有关的共享数据语义错误

目标：解决竞争条件

## 解决竞争条件逻辑

```c++
P1.register = empty
--P1.register;
P2.register = empty
--P2.register 
empty=P1.register;
empty=P2.register
```

在写共享变量empty时，阻止其他进程访问empty，一段代码只允许一个进程进入
1. 生产者<font color="#c0504d">P1检查</font>并给empty<font color="#c0504d">上锁</font>
2. P1.register = empty
3. P1.register = P1.register -1;
4. 生产者<font color="#c0504d">P2检查</font>empty的锁
5. empty=P1.register;
6. 生产者<font color="#c0504d">P1</font>给empty<font color="#c0504d">开锁</font>
7. 生产者<font color="#c0504d">P2检查</font>并给empty<font color="#c0504d">上锁</font>
8. P2.register = empty
9. P2.register = P2.register -1;
10. empty=P2.register;
11. 生产者<font color="#c0504d">P2</font>给empty<font color="#c0504d">开锁</font>

临界区（Critical Section）：一次只允许一个进程进入的该进程的代码段
- 读写信号量的代码一定是临界区

## 解决竞争条件实现
原则

互斥进入：如果一个进程在临界区中执行，则其他进程不允许进入
- 互斥（mutual exclusion）：执行临界区时的约束关系

有空让进：若干进程要求进入空闲临界区时，应尽快使一进程进入临界区

有限等待：从进程发出进入请求到允许进入，不能无限等待


### 轮换法

```c
int turn=0;

void P1(){
	// 剩余区
	while(true!=0);
	// 临界区
	turn = 1;
	// 剩余区
}

void P2(){
	// 剩余区
	while(true!=1);
	// 临界区
	turn = 0;
	// 剩余区
}
```

问题：P1完成后不能再次进入，尽管P2不再临界区

### 标记法

```
bool flag[2];

void P1(){
	// 剩余区
	flag[0]=true;
	while(flag[1]);
	// 临界区
	flag[0] = false;
	// 剩余区
}

void P2(){
	// 剩余区
	flag[1]=true
	while(flag[0]);
	// 临界区
	flag[1] = false;
	// 剩余区
}

```

问题：P1和P0的进入请求会无限等待

### 非对称标记法

非对称标记法（Peterson）：结合率标记和 轮转两种思想

```c
bool flag[2];
int turn=0;

void P1(){
	// 剩余区
	flag[0]=true;
	turn = 1;
	while(flag[1] && turn == 1);
	// 临界区
	flag[0] = false;
	// 剩余区
}

void P2(){
	// 剩余区
	flag[1]=true
	turn = 0;
	while(flag[0] && turn == 0);
	// 临界区
	flag[1] = false;
	// 剩余区
}
```

### 面包店算法

```c
bool choosing[n];
int num[n];

void Pi(){
	//剩余区
	choosing[i]=true;
	num[i]=max(num[0],...num[n-1])+1;
	choosing[i]=false;
	for(j=0;j<n;j++){
		while(choosing[j]);
		while((num[j]!=0)&&(num[j],j)<(num[i],i));
	}
	// 临界区
	num[i]=0;
	//剩余区
}
```

### 禁止调度

只允许一个进入进入，进入另一个进程代表发生调度

```c
void Pn(){
	cli();
	// 临界区
	sti();
}
```

不能在多核CPU中处理
