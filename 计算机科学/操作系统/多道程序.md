# 多道程序

多道程序设计技术：在主存中同时存放多个程序，通过共享系统资源并发执行。目的是提高 CPU 的工作效率

## 程序的顺序执行

*程序的顺序执行*：在早期无操作系统及单道批处理系统时，程序的顺序执行是指一个应用程序的若干程序段在必须按照某种先后次序执行，仅当前一段程序执行完后才能执行后一段程序

- 基本特征有：顺序性、封闭性、可再现性

## 程序的并发执行

程序的并发执行：程序的并发执行是指在同一时间间隔内可以运行多个程序，即一个程序在运行结束之前，可以与其他的程序交互运行。从宏观上看，多个程序在同时向前推进，从微观上看，任意时刻 CPU 上都 只有一个程序在运行。

特点：间断性、失去封闭性和不可再现性
