
# 存储器

## 存储器分类

按存储介质

- 半导体存储器：介质为TTL、MOS，信息容易丢失
- 磁表面存储器：介质为磁头、载磁体，信息不易丢失
- 磁芯存储器CoreMemory：介质为硬磁材料、换装原件，信息不易丢失
- 光盘存储器：介质为激光、磁光材料，信息不易丢失

按存取方式
- 顺序存取：数据按照顺序存放，读取按顺序读取
- 随机访问：存取时间与物理地址无关
	- 随机存储器 ROM
	- 只读存储器 RAM
- 串行访问（直接存取）：存取时间与物理地址有关，
	- 顺序存取存储器 磁带
	- 直接存取存储器 磁盘

按计算机中的作用

- 主存储器
	- 随机存取存储器**RAM**：包括静态 RAM 和动态 RAM
	- 只读存储器**ROM**：只能读不能写
	    - 掩膜只读存储器MROM
	    - 可编程只读存储器PROM
	    - 可编程可擦除只读存储器EPROM
	    - 电可擦除可编程只读存储器EEPROM
- 辅助存储器：磁盘、磁带、光盘
- **FlashMemory**：比主存速度慢，比辅助存储器快，可以当作主辅存储器间的缓存
- 高速缓冲存储器Cache：最快，由静态 RAM 组成


按易失性区分
- 易失性存储器：通常是随机存取存储器
	- 动态随机存储器
	- 静态随机存储器：Cache、TLB
- 非易失性存储器：断电后不易失
	- 半导体电路
		- 只读存储器：ROM、PROM、EPROM、EEPROM
		- 闪存：U 盘、固态硬盘 SSD
	- 非半导体电路：光盘、只读光盘、磁带、机械硬盘 HDD、软盘
