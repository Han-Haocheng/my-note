
# 存储器


## 存储方式


- 顺序存取：最基础的存取方式，数据是按顺序存放的，读取时也必须按顺序逐个读取，无法跳过。例如磁带存储就是这种方式，你要访问的数据必须依次扫描到目标点才能读取。这种方式的优点是实现简单，适合存储大量数据，但访问速度较慢。
- 随机存取：也称为直接访问存储器（Direct Access Memory, RAM），可以在任意时间直接访问任何存储单元，不必按顺序来。常见的 RAM、和闪存（U 盘、固态硬盘 SSD）都支持这种方式。其特点是存取速度快，适合需要频繁访问和修改数据的场景。
- 直接存取：数据被分成了几个块，你可以直接跳转到某个大块的位置，但在大块内的精确定位仍需要顺序访问【选取信息所在区域（随机存取特性），然后顺序存取（顺序存取特性）】。例如机械硬盘（HHD）就是这种方式，允许快速找到某个大块的数据，但要找到具体的字节位置，还需要一些调整。
- 相联存取：这种方式通过数据的内容来进行存取，而不是通过数据的地址。你提供某些标识符（例如关键字），系统会自动帮你找到相匹配的数据。相联存储器（CAM）是一种典型的相联存取方式，常用于高速缓存或路由器中，以快速查找数据，常见的例子是 Cache 和 TLB。


## 存储器分类

按存储介质

- 半导体存储器：介质为TTL、MOS，信息容易丢失
- 磁表面存储器：介质为磁头、载磁体，信息不易丢失
- 磁芯存储器CoreMemory：介质为硬磁材料、换装原件，信息不易丢失
- 光盘存储器：介质为激光、磁光材料，信息不易丢失

按存取方式
- 随机访问：存取时间与物理地址无关
	- 随机存储器 ROM
	- 只读存储器 RAM
- 串行访问
	- 顺序存取：磁带
	- 直接存取：磁盘
- 相联存取：Cache、TLB

按计算机中的作用

- 主存储器
	- 随机存取存储器**RAM**：包括静态 RAM 和动态 RAM
	- 只读存储器**ROM**：只能读不能写
	    - 掩膜只读存储器MROM
	    - 可编程只读存储器PROM
	    - 可编程可擦除只读存储器EPROM
	    - 电可擦除可编程只读存储器EEPROM
- 辅助存储器：磁盘、磁带、光盘
- **FlashMemory**：比主存速度慢，比辅助存储器快，可以当作主辅存储器间的缓存
- 高速缓冲存储器Cache：最快，由静态 RAM 组成


按易失性区分
- 易失性存储器：通常是随机存取存储器
	- 动态随机存储器
	- 静态随机存储器：Cache、TLB
- 非易失性存储器：断电后不易失
	- 半导体电路
		- 只读存储器：ROM、PROM、EPROM、EEPROM
		- 闪存：U 盘、固态硬盘 SSD
	- 非半导体电路：光盘、只读光盘、磁带、机械硬盘 HDD、软盘

## 性能指标
### 存储周期

存储周期（存取周期）：存取时间+回复时间，两次独立访问内存之间的时间间隔

### 传输速率

传输速率：数据输入或从存储器输出的速率



### 存储容量

存储容量：存放二进制信息的总位数
- 主存容量（字节数） = 存储单元个数 * 存储字长

| MAR | MDR | 容量     |
| --- | --- | ------ |
| 10  | 8   |        |
| 16  | 32  | 64 k*32 |

- 辅存容量（字节数）
