# 定点数

定点数：小数点的位置是固定不变的


![[计算机科学/计算机组成原理/数字/2026-03-04_214916.svg]]

定点小数：小数点位于符号位（如果有符号）之后，所有数值之前
- 数符$S_f$
- $S_1\sim S_n$：数值部分
- 小数点位于数符和数值中间


![[计算机科学/计算机组成原理/数字/2026-03-04_215144.svg]]


顶点整数：小数点位于数值位之后
- $S_f$：数符
- $S_1\sim S_n$：数值部分
- 小数点：位于数值后


| 定点机 | 小数定点机                        | 整数定点机                  |
| --- | ---------------------------- | ---------------------- |
| 原码  | $-(1-2^{-n})\sim+(1-2^{-n})$ | $-(2^n-1)\sim+(2^n-1)$ |
| 补码  | $-1\sim+(1-2^{-n})$          | $-2 \sim+(2^n)-1$      |
| 反码  | $-(1-2^{-n})\sim+(1-2^{-n})$ | $-(2^n-1)\sim+(2^n-1)$ |
