# IO与显示器

## 外设的工作方式

1. 通过总线控制器向设备中的控制寄存器写命令
2. 设备向CPU发出中断
3. CPU将数据放置在设备的寄存器/设备向寄存器放入数据等待CPU读取

文件视图：用于查寄存器地址、内容的格式和语义，操作系统给用户提供简单视图

## 操作外设

```c
int fd=open("/dev/xxx");
for(int i =0;i<10;i++){
	write(fd,i,sizeof(int));
}
close(fd);
```

- 操作系统提供统一接口，所有设备都是open->read->write->close
- 根据文件找到控制器地址、内容格式等，不同的设备对应不同的设备文件

kernel/tty_io
```c
int tty_write(unsigned channel,char*buf,int nr){
	struct tty_struct*tty;
	tty=channel+tty_table;
	shlle_if_fall(&tty->write_q); // 判断写的缓冲区队列是否已满
	char c,*b=buf;
	while(nr>0&&!FULL(tty->write_q)){
		c=get_fs_byte(b);
		if(c=='\r'){
			PUTCH(13,tty->write_q);
			continue;
		}
		if(O_LCUC(tty))
			c = toupper(c);
		b++;
		nr--;
		PUTCH(c,tty->write_q);
	}// 输出完成或写队列已满
	tty->write(tty);
}
```

void con_write(struct tty_struct *tty){
	GETCH(tty->write_q,c);
	if(c>31&&c<127){
		__asm__(
			"movb _attr,%%ah\n\t"
			"movw %%ax,%%1\n\t" // 统一编址用mov，独立编址用out
			::"a"(c),"m"(*(short*)pos):"ax"
		);
		pos+=2;
	}
}
