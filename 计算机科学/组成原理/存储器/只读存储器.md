# 只读存储器


## 早期的只读存储器 MROM


早期的只读存储器MROM（掩膜 ROM）

- 行列选择线加查出有 MOS 管为 1，无 MOS 管为 0
- 在厂家就写好了内容

## 改进 1：PROM（一次性 ROM）

![image.png](计算机科学/组成原理/attachments/4.2.4image.png)

- 熔丝断为 0，熔丝未断为 1

- 用户可以自己写，一次性

## 改进 2：EPROM（可擦写 ROM）

![image.png](计算机科学/组成原理/attachments/4.2.4image1.png)

- D 端加正电压，形成浮动栅，S 与 D 不导通为 0

- D 端不加正电压，不形成浮动栅，S 与 D 导通为 1

- 紫外线全部擦洗

- 要能对信息进行擦除，可以多次写，便宜

## 改进 3-EEPROM（电可擦写 ROM）

- 电可擦写，特定设备

- 局部擦写

- 全部擦写

## 改进 4-Flash Memory（闪速型存储器）

- 电可擦写（直接连接到计算机）

- 例如：U 盘

- 比 EEPROM 快，部分具有 RAM 功能
