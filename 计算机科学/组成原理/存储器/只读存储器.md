# 只读存储器


只读存储器：非易失性存储器，保存要长期存储的数据

## MROM


常规只读储存器MROM（掩膜 ROM）：在厂家出厂就写好了内容
- 行列选择线加查出有 MOS 管为 1，无 MOS 管为 0

## PROM


一次可编程只读存储器PROM：用户可以自己写，一次性
- 熔丝断为 0，熔丝未断为 1

## EPROM（可擦写 ROM）

![[计算机科学/组成原理/attachments/4.2.4image1.png|image.png]]

可擦写只读存储器EPROM：紫外线全部擦洗，要能对信息进行擦除，可以多次写，便宜
- D 端加正电压，形成浮动栅，S 与 D 不导通为 0
- D 端不加正电压，不形成浮动栅，S 与 D 导通为 1




## EEPROM


电可擦写只读存储器 EEPROM：电可擦写，特定设备

- 局部擦写
- 全部擦写

## 闪速型存储器


闪速型存储器 Flash Memory：直接连接到计算机电可擦写，比 EEPROM 快，部分具有 RAM 功能
- 例如：U 盘
