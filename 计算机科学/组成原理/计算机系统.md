# 计算机系统

计算机系统：由计算机软件系统和计算机硬件硬件系统组成
- 软硬件在逻辑上是等效的

|      | 硬件              | 软件                 |
| ---- | --------------- | ------------------ |
| 特性   | 硬件乘法器           | 软件模拟乘法             |
| 速度   | 快（1 个时钟周期完成）     | 慢（需要多次循环和加法）       |
| 资源占用 | 需要专用电路（面积大、功耗高） | 无需额外硬件（节省芯片面积）     |
| 灵活性  | 固定功能（只能做乘法）     | 可通过编程修改算法（如优化速度）   |
|      | 电路实现            | 编程实现               |
|      | 性能好、成本高、不易拓展    | 性能低（相对于硬件）、成本低、易拓展 |



**硬件**：计算机的实体，如主机外设等

**软件**：由具有各类特殊功能的信息（程序）组成
- **系统软件**：用来管理整个计算机系统
	- 举例：语言处理程序、操作系统、服务性程序、数据库管理系统、网络软件
	- 语言处理程序：高级语言、汇编语言、机器语言
- **应用软件**：按任务需要编制成的各种程序


现代计算机系统的多态性

> [!example] **举例**：传感器、手机屏幕、电脑、服务器、超算、物联网（未明确规定）
> - **超级计算机(HPC)**：高耗电、高费用、高算力的计算机

### 运算器

![[计算机科学/组成原理/attachments/20241128190406.svg]]


**运算器**：通过算逻运算单元完成运算

累加器类型运算器结构
- 累加器 ACC(Accumulator)
- 乘商寄存器 MQ(Multiplier-Quotient Register)
- 通用寄存器 X
- 算逻运算单元 ALU(ArithmeticLogicUnit)

#### 加法

**加法**：`ACC` 存放*被加数*与*和，* `X` 存放**加数**
1. 被加数-> `ACC` ;
2. $[M]$-> `X` ;
3. $[ACC]+[X]$-> `ACC` ;

#### 减法

**减法**：`ACC` 保存*被减数*与*差*，`X` 保存*减数*
1. 被减数-> `ACC`
2. $[M]$-> `X`
3. $[ACC]-[X]$-> `ACC`

#### 乘法

1,01111010;0.0100101

**乘法**：`ACC` 存放*乘积高位*与*乘数*，`MQ` 存放*乘积低位*，`X` 存放*被乘数*
1. 被乘数-> `ACC`
2. $[M]$-> `MQ`
3. $[ACC]$-> `X`
4. $0$-> `ACC`
5. $[X]×[MQ]$-> `ACC|MQ`：`MQ` 是运算的高位

#### [[0203-计算机组成原理/06-计算机的运算方法/6.3 定点运算.md.md#除法运算|除法]]

**除法**：`ACC` 存放*被除数*和*余数*，`MQ` 存放*商*，`X` 存放*除数*
1. 被除数-> `ACC` ;
2. $[M]$-> `X` ;
3. $[ACC]÷[X]$-> `MQ|ACC` ;//余数在 `ACC` 中

### 控制器

**控制器**：解释指令，保证指令按序执行
- 指令解析器(CU)：用于解析指令，并发出相应控制信号
- 指令地址寄存器 PC：存放当前域执行指令的地址，具有计数功能 $[pc]+1$-> `pc`
- 指令寄存器(IR)：存发当前欲执行的指令
- 控制流程：取指令->分析指令->执行指令

## 技术指标

### 机器字长

CPU 一次能处理数据的位数，与 CPU 中的寄存器位数有关

机器字长：计算机进行一次整数运算可处理的二进制位数。
- CPU 一次可处理位数 $=$ CPU 总线的宽度 $=$ 运算器 ALU 的位数 $=$ 通用寄存器的位数

### 时钟周期和主频

**时钟周期**：通常为节拍脉冲或 T 周期，主频的倒数，是 CPU 中最小的时间单位，执行指令的每个动作至少需要 1 个时钟周期
- CPU 时钟周期= $\frac{1}{主频}$

**CPU 主频**：1 秒执行多少个时钟周期，主频= $\frac{1}{CPU时钟周期}$，主频通常以 Hz(赫兹)为单位，1 Hz 表示每秒 1 次

### CPI

CPI：执行一条指令所需要时钟周期数（加权平均计算）
- 程序时钟周期：$程序指令数\times CPI$
- 程序执行时间：$总时钟周期\times时钟周期时间=\frac{总时钟周期}{时钟频率}$
- 每条指令运行时间：$该指令时钟周期个数(CPI) \times CPU时钟周期$
- 程序的 CPI $= \frac{程序运行需要的CPU时钟周期个数}{ 程序的指令个数（指令CPI平均值）}$

### 浮点数运算性能


**浮点数运算性能的评价指标**：MFLOPS、GFLOPS、TFLOPS、PFLOPS、EFLOPS、ZFLOPS

- **FLOPS(Floating-point operations per second、flops、flop/s、每秒峰值速度)**：每秒所运行的浮点运算次数

### MIPS

MIPS：CPU 每秒执行多少个百万条指令。
- $MIPS= \frac{指令条数}{执行时间\times10^{6}}=\frac{主频}{CPI\times10^{6}}$



### 其他运算速度

- 核数 / 每个核支持的线程数
- 吉普森法：$T_{M}= \sum_{i=1}^{n}f_it_i$
	- 指令出现的频率 $f_i$
	- 指令出现的次数 $t_i$
- 指令的静态使用频率：按照程序清单直接计算
- 指令的动态使用频率：计算程序运行时每条指令的频率
- IPC：一个时钟周期能完成的指令数
