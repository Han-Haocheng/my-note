# 计算机系统

计算机系统：由计算机软件系统和计算机硬件硬件系统组成
- 软硬件在逻辑上是等效的

|      | 硬件              | 软件                 |
| ---- | --------------- | ------------------ |
| 特性   | 硬件乘法器           | 软件模拟乘法             |
| 速度   | 快（1 个时钟周期完成）    | 慢（需要多次循环和加法）       |
| 资源占用 | 需要专用电路（面积大、功耗高） | 无需额外硬件（节省芯片面积）     |
| 灵活性  | 固定功能（只能做乘法）     | 可通过编程修改算法（如优化速度）   |
|      | 电路实现            | 编程实现               |
|      | 性能好、成本高、不易拓展    | 性能低（相对于硬件）、成本低、易拓展 |

现代计算机系统的多态性

> [!example] **举例**：传感器、手机屏幕、电脑、服务器、超算、物联网（未明确规定）
> - **超级计算机(HPC)**：高耗电、高费用、高算力的计算机


计算机体系结构(抽象层)：程序员(机器语言程序员)所见到的计算机系统的属性(指令系统、数据类型、寻址技术、io 机理)，概念性的结构与功能特性

## 计算机硬件

**硬件**：计算机的实体，如主机外设等

![[计算机科学/组成原理/attachments/20241128174552.svg]]

冯诺依曼计算机
- 计算器有五大部分组成
- 指令和数据以同等地位存于存储器，可按地址寻访
- 指令和数据用二进制表示
- 指令由操作码和地址码组成
- 具有存储程序
- 以运算器为中心

冯诺依曼计算机硬件结构
- 运算器 ALU：算数运算逻辑运算
- 控制器 CU：指挥程序的运行
- 存储器：存放数据和程序
    - 主存
    - 辅存
- 输入设备：将信息转换成机器能识别的形式
- 输出设备：将结果转换成人们熟悉的形式

以运算器为中心的计算机硬件框图
- 运算器为中心，导致运算器成为系统瓶颈
- 不具有层次化特征，系统部件耦合度过高

## 计算机软件

**软件**：由具有各类特殊功能的信息（程序）组成
- 包含：应用软件、系统软件
### 应用软件

**应用软件**：按任务需要编制成的各种程序

### 系统软件


**系统软件**：直接与硬件进行交换的软件，用来管理整个计算机系统
- 开发周期长
- 制作成本高昂
- 检测软件产品质量的特殊性
- 软件是程序以及开发、使用和维护程序所需要的所有文档

相关系统软件
- **语言处理程序**：如汇编程序、编译程序 、解释程序等
- **操作系统**：管理计算机软硬件资源的软件
- **服务性程序**：装配软件，调试软件，诊断排错软件
- **数据库管理系统**：数据库和数据库管理软件
- **网络软件**

## 技术指标

### 机器字长

CPU 一次能处理数据的位数，与 CPU 中的寄存器位数有关

机器字长：计算机进行一次整数运算可处理的二进制位数。
- CPU 一次可处理位数 $=$ CPU 总线的宽度 $=$ 运算器 ALU 的位数 $=$ 通用寄存器的位数

### 时钟周期和主频

**时钟周期**：通常为节拍脉冲或 T 周期，主频的倒数，是 CPU 中最小的时间单位，执行指令的每个动作至少需要 1 个时钟周期
- CPU 时钟周期= $\frac{1}{主频}$

**CPU 主频**：1 秒执行多少个时钟周期，主频= $\frac{1}{CPU时钟周期}$，主频通常以 Hz(赫兹)为单位，1 Hz 表示每秒 1 次

### CPI

CPI：执行一条指令所需要时钟周期数（加权平均计算）
- 程序执行时间：$总时钟周期\times时钟周期时间=\frac{总时钟周期}{时钟频率}$
	- 程序时钟周期：$程序指令数\times CPI$
- 每条指令运行时间：$该指令时钟周期个数(CPI) \times CPU时钟周期$
- 程序的 CPI $= \frac{程序运行需要的CPU时钟周期个数}{ 程序的指令个数（指令CPI平均值）}$



### 浮点数运算性能


**浮点数运算性能的评价指标**：MFLOPS、GFLOPS、TFLOPS、PFLOPS、EFLOPS、ZFLOPS

- **FLOPS(Floating-point operations per second、flops、flop/s、每秒峰值速度)**：每秒所运行的浮点运算次数

### MIPS

MIPS：CPU 每秒执行多少个百万条指令。
- $MIPS= \frac{指令条数}{执行时间\times10^{6}}=\frac{主频}{CPI\times10^{6}}$



### 其他运算速度

- 核数 / 每个核支持的线程数
- 吉普森法：$T_{M}= \sum_{i=1}^{n}f_it_i$
	- 指令出现的频率 $f_i$
	- 指令出现的次数 $t_i$
- 指令的静态使用频率：按照程序清单直接计算
- 指令的动态使用频率：计算程序运行时每条指令的频率
- IPC：一个时钟周期能完成的指令数

### 微型计算机的出现和发展

- 微处理器：用一片大规模集成电路构成的，集成了运算器和控制器，
- 微型计算机：使用微处理器芯片，高集成度的，面向用户，体积小、重量轻、价格低的计算机

前置技术
- 第一代微处理器：4040，生产于 1971 年
- 第一代存储芯片：生产于 1970 年


ENIAC
- 生产时间：1946-1955
- 生产地点：美国
- 运算元件：电子管/继电器
- 特点
    - 电子管：18000
    - 继电器：1500
    - 功耗：150 千瓦
    - 重量：30 吨
    - 占地：1500 平方英尺
    - 速度：5000 次加法/秒
- 缺点
    - 没有存储器
    - 用布线接板进行控制
- 意义
    - 世界上第一台计算机
    - 为提出冯诺依曼架构提供了技术基础

IAS
- 生产时间：1942 设计 1952 运行
- 生产地点：美国普林斯顿大学
- 特点：以运算器为中心的冯诺依曼结构


IBM System/360

- 时间：1961 研制 1964 成功
- 地点：美国 IBM 公司
- 特点：大主机结构，系列产品，同指令系统，可互联
