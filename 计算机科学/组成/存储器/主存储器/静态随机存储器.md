## 静态随机存储器SRAM

静态存储器 SRAM（易失性存储器）：存集成度低，读写快，价格高，通常用作 Cache、TLB


![[计算机科学/组成/attachments/Pasted image 20260312211437.png]]

- $T_1\sim T_4$ 触发器，比如双稳态触发器
- $T_5、T_6$ 行开关
- $T_7、T_8$ 列开关

注意：虚线部分为存储单元，因此可以进行堆叠

读操作
1. $行选择\to T_5、T_6 开$
2. $列选择\to T_7、T_8 开$
3. 读选择有效
4. $V_A\to T_6\to T_8\to读放\to D_{out}$

写操作
1. $行选择\to T_5、T_6 开$
2. $列选择\to T_7、T_8 开$
3. 写选择有效
4. $（左）D_{IN}\to反向\to T_7\to T_5\to A`$
5. $（右）D_{IN}\to T_8\to T_6\to A$


举例

![[4.2.3image2.png]]
Intel 2114：芯片容量为 $1K×4位$，因此可以将芯片划为 64×64 矩阵，共 64 行 16 列 4 组，每次 4 位输出
- $\overline{WE}$ 是读写信号线
- $\overline{CS}$ 是片选信号线
- $A_0\sim A_9$ 是地址线，代表芯片有 $2^10=1024=1K$ 存储单元
- $I/O_1 \sim I/O_4$ 是数据线，代表 1 个存储单元能保存 4 bit 数据

Intel 2114 RAM 矩阵（64×64）矩阵图

![image.png](4.2.3image3.png)

Intel 2114 RAM 矩阵（64×64）读取 `b000000 0000` 地址
1. $\overline{WE}$ 信号高电平有效、$\overline{CS}$ 信号低电平有效
2. 行选择器选择第 0 行位置
3. 列选择器选择每组第 0 列位置，也就是第 0、16、32、48 列
4. 数据会从 4 个读写电路输出对应数据

Intel 2114 RAM 矩阵（64×64）写`b000000 0000` 地址

1. $\overline{WE}$ 信号低电平有效、$\overline{CS}$ 信号低电平有效
2. 行选择器选择第 0 行位置
3. 列选择器选择每组第 0 列位置，也就是第 0、16、32、48 列
4. 数据会从 4 个读写电路输入到对应位置
