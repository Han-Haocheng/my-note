## 光盘存储器

**光盘存储器**：采取光存储技术，利用激光写入读出

### 光盘存储分类 
- 第一代光存储技术：采用非磁性介质，不可擦写
- 第二代光存储技术：采用磁性介质，可擦写


### 光盘存储原理

- 只读型和只写一次型：热作用（物理或化学）
- 可擦写型：热磁效应

## 磁盘阵列


独立磁盘冗余阵列RAID（Redundant Array of Independent Disks）：将多个物理磁盘以不同的方式组合成一个逻辑硬盘的技术，用于提高硬盘的读写性能或数据安全可靠性

### 磁盘阵列种类
- RAID 0：无冗余的存储空间条带化，只提高了速度
- RAID 1（磁盘镜像）：采用磁盘镜像技术。数据被写入磁盘时，同时写入另外的镜像磁盘
- RAID 2（海明码校验）：数据存放在若干磁盘上，使用单独的磁盘存储数据对应的纠错信息。纠错方法采用海明校验
- RAID 3（位交叉奇偶校验）：数据按照位交叉的方式存放在若干磁盘上，使用单独的磁盘存储数据对应的奇偶校验信息
- RAID 4（块交叉奇偶校验）：数据按照块交叉的方式存放在若干磁盘上，使用单独的磁盘存储数据块对应的奇偶校验信息
- RAID 5（分布式奇偶校验）：把数据块、数据块的奇偶校验块交叉存放在若干磁盘上（无专用校验盘）
- RAID 6（双重奇偶校验）：采用两个独立的奇偶校验方案

### 关键差异总结表

| RAID 级别    | 数据存放方式  | 校验/冗余方式      | 校验盘位置      |                              |
| :--------- | :------ | :----------- | :--------- | ---------------------------- |
| **RAID 1** | 镜像写入    | 完全复制         | 无校验盘，使用镜像盘 | 提供冗余，适合存储关键数据。               |
| **RAID 2** | 数据分条    | 海明码校验        | 独立磁盘       | 当数据产生错误时，利用纠错信息恢复原数据。        |
| **RAID 3** | **位**交叉 | 奇偶校验         | 独立磁盘       | 当数据产生错误时，利用奇偶校验信息恢复原数据。      |
| **RAID 4** | **块**交叉 | 奇偶校验         | 独立磁盘       | 当数据块产生错误时，利用奇偶校验信息恢复原数据。<br> |
| **RAID 5** | **块**交叉 | 奇偶校验         | 分布式（交叉存放）  |                              |
| **RAID 6** | 块交叉     | **双重**独立奇偶校验 | 分布式        | 允许两块磁盘同时故障而不丢失数据，可靠性更高。      |


## 固态硬盘

固态硬盘：固态硬盘又称固态驱动器，是用固态电子存储芯片阵列制成的硬盘。

### 固态硬盘的组成

***构造**：固态硬盘内部主体是一块 PCB 板，其最基本的配件包括：
*   **控制芯片**：负责数据管理和控制。
*   **缓存芯片**：辅助数据传输（注：部分固态硬盘无独立缓存芯片）。
*   **闪存芯片**：用于存储数据的核心部件。
### 固态硬盘的特点

优点：与传统机械硬盘相比，固态硬盘具备以下显著优势：
*   **高速读写**：数据传输速度远超机械硬盘
*   **物理特性**：质量轻、能耗低、体积小

缺点：尽管 SSD 已进入存储市场主流行列，但其劣势也较为明显
*   **价格与容量**：价格相对昂贵，且容量通常低于同价位机械硬盘
*   **数据恢复**：一旦硬件损坏，数据较难恢复
*   **耐用性**：部分观点认为固态硬盘的耐用性（寿命）相对较短
