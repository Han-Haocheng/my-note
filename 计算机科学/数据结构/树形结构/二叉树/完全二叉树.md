# 完全二叉树

完全二叉树：深度为$k$且具有$n$个结点的二叉树，每一个结点都与深度为$k$的[[计算机科学/数据结构/树形结构/二叉树/满二叉树|满二叉树]]中==顺序编号==为$1\sim n$的结点一一对应（或前$m$个结点）
- 至多只有最下面两层上的结点的度可以小于2，并且最下层上的结点都集中在该层最左边的若干位置上
- 完全二叉树除开最后一层是一个[[计算机科学/数据结构/树形结构/二叉树/满二叉树|满二叉树]]

## 完全二叉树的性质

#计算机/数据结构/性质  具有$n$（$n>0$）个结点的[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]的深度为$\lceil \log_2(n+1) \rceil$

> [!cite]- 证明：
> - 设[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]深度为$k$
> - 由[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]的定义可知，深度为$k$的[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]前$k-1$层是[[计算机科学/数据结构/树形结构/二叉树/满二叉树|满二叉树]]，共有$2^{k-1}-1$个结点，第$k$层还有若干个结点
> 	- 因此有：$n > 2^{k-1}-1$ 
> - 又根据性质2
> 	- 深度为$k$的二叉树至多有$2^k-1$个结点，即：$n \leq 2^k-1$  ②
> - 由①和②可推出：$2^{k-1}-1 < n \leq 2^k-1$，即$2^{k-1} < n+1 \leq 2^k$
> 	- 取对数后有：$k-1 < \log_2(n + 1) \leq k$
> - 又因$k-1$和$k$是相邻的两个整数，因此有：$k = [\log_2(n + 1)]$
> - 故性质4成立

#计算机/数据结构/性质  如果将一棵有n个结点的[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]按照自顶向下，同一层自左向右的顺序连续给结点编号1,2,3,…,n，则对任一结点i有以下结论：
- 若i=1，则结点i是[[计算机科学/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]的根，无父结点；若i>1，则结点i的父结点为结点$\left\lfloor  \frac{i}{2}  \right\rfloor$。
- 若$2i>n$，则结点$i$无左孩子（结点$i$为叶子结点）；否则其左孩子是结点$2i$。
- 若$2i+1 > n$，则结点$i$无右孩子，否则其右孩子为结点$2i+1$。
- 结点i所在层次（深度）为$\lfloor \log_{2}{i} \rfloor+ 1$

## 完全二叉树的顺序存储
![[计算机科学/数据结构/attachments/2026-01-23_155801.svg]]
完全二叉树的顺序存储：按照顺序存储的定义，用一组地址连续的存储单元依次自上而下、自左至右存储完全二叉树上的结点结构，存储时只保存各结点的值
- 结点之间的层次关系由性质5决定
