生成树：具有$n$个顶点的连通图$G$的生成树是包含$G$中全部顶点的一个==极小连通子图==，包含原图中的所有顶点，而且有尽可能少的边
- 若删除一条边，会使生成树变成非连通图；若增加一条边，会形成图中的一个回路
- 不同遍历算法，得到的生成树不同；不同顶点出发，得到的生成树不同


- 一棵具有$n$个顶点的生成树有且仅有$n-1$条边
- 生成树中减少任意一条边必然会成为非连通图
- 在生成树中添加任意一条属于原图中的边必定会产生回路或环

> [!tip] 无向图连通至少$n-1$条边
> 

## 1 生成森林

![[计算机科学/数据结构/attachments/Pasted image 20260128200558.png]]

生成森林：非连通图的每个连通分量都可以得到一棵生成树（极小连通子图）。这些连通分量的生成树构成的森林

- 有向图的生成森林：由若干棵有向树组成，含有图中全部顶点。其中有向树是只有一个顶点的入度为0、其余顶点的入度均为1的有向图

## 2 最小生成树

最小生成树MST（最小代价生成树、Minimum Spanning Tree）：$G(V,E)$是一个==无向连通带权图==，在$G$的所有生成树中，==代价最小的生成树==
- 边的代价Cost：生成树上任一条边的权值
- 生成树的代价：树上各边代价之和
- 时间复杂度：$T(n)=O(n^{2})$

最小代价生成树的准则：若连通带权图由$n$个顶点组成，则其生成树必含$n$个顶点$n-1$条边
- 尽可能选取权值最小的边
- 能且只能使用$n-1$条边来连接网络中的$n$个顶点
- 选用的$n-1$条边不能产生回路

构造最小生成树的算法：普里姆算法、克鲁斯卡尔算法


### 2.1 普里姆算法

![[计算机科学/数据结构/attachments/2026-02-05_194302.svg]]
![[计算机科学/数据结构/attachments/Pasted image 20260130192700.png]]

 普里姆算法Prim：归并顶点，适用于稠密网
1. 从某顶点$v_0$出发，选择与它关联的具有最小权值的边$(v_0, v)$，将其顶点加入到生成树的顶点集合$U$中  
2. 每一步从一个顶点在$U$中，而另一个顶点不在$U$中的各条边中选择权值最小的边$(u, v)$，把它的顶点加入到$U$中  
3. 直到所有顶点都加入到生成树顶点集合$U$中为止

```c
//辅助数组的定义,用来记录从顶点集U到V-U的权值最小的边
struct {
    VerTexType adjvex;    // 最小边在U中的那个顶点
    ArcType lowcost;      // 最小边上的权值
} closedge[MVNum];

/*
* @breaf 普里姆算法 无向网G以邻接矩阵形式存储，从顶点u出发构造G的最小生成树T，输出T的各条边
* @prams G MGraph类型 邻接矩阵形式的图
* @prams u VerTexType 起始顶点
*/
void MiniSpanTree_Prim(const MGraph& G,const VerTexType& u) {
    int k, j, i;
    VerTexType u0, v0;
    k = LocateVex(G, u);  // k为顶点u的下标
    
    // 对V-U的每一个顶点v_j，初始化closedge[j]
    for (j = 0; j < G.vexnum; ++j) {
	    // 跳过起始节点
        if (j == k) 
	        continue;
	    // 初始化closedge为顶点的邻接关系
		closedge[j].adjvex = u;
		closedge[j].lowcost = G.vdegs[k][j];// {adjvex, lowcost}
    }
    closedge[k].lowcost = 0;          // 初始,U = {u}
    
    // 选择其余n-1个顶点,生成n-1条边(n=G.vexnum)
    for (i = 1; i < G.vexnum; ++i) {  
        k = Min(G); // 获取
        u0 = closedge[k].adjvex;// u0为最小边的一个顶点,u0∈U
        v0 = G.vexs[k];         // v0为最小边的另一个顶点,v0∈V-U
        printf("边%c--->%c\n", u0, v0);// 输出当前的最小边(u0, v0)
        closedge[k].lowcost = 0;       // 第k个顶点并入U集
        
        // 新顶点并入U后重新选择最小边
        for (j = 0; j < G.vexnum; ++j) {  
            if (G.vdegs[k][j] >= closedge[j].lowcost) 
	            continue;
			closedge[j].adjvex  = G.vexs[k];
			closedge[j].lowcost = G.vexs[k][j];
        }
    }
}
```


### 2.2 克鲁斯卡尔算法

克鲁斯卡尔算法Kruskal：归并边，适用于稀疏网
- $T(n)=n+e\log_{2}{e}+e=T(e\log_{2}{e}+n)$
	- 边规模$e$
	- 结点规模$n$

1. 构造一个只有$n$个顶点，没有边的非连通图$T$，每个顶点自成一个连通分量  
2. 在$E$中选最小权值的边（若有多条，任取其一），若该边的两个顶点落在不同的连通分量上（保证不形成回路），则加入$T$中；否则舍去，重新选择  
3. 重复下去，直到所有顶点在同一连通分量上为止

```c
typedef struct{
	int beg;
	int end;
	int weight;
}Edge;

void MiniSpanTree_Kruskal(MGraph G){
	int i,n,m;
	Edge edges[MAXEDGE];
	int parent[MAXVEX];
	// 对列表按权排序
	
	for(i=0;i<G.numEdges;i++){
		n=Find(parent, edges[i].beg);
		m=Find(parent, edges[i].end);
		if(n==m)
			continue;
		parent[n]=m;
		printf("(%d,%d)%d",edges[i].beg,edges[i].end,edges[i].weight)
	}
}
```
