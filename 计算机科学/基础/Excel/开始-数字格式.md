## 数字格式

### 开始选项卡->数字组

![[计算机科学/基础/attachments/Pasted image 20250311090456.png]]

- `数字格式`下拉菜单：选择单元格的格式
- `会计格式`命令：格式设置为当前地区的货币样式
- `会计格式`下拉菜单：格式设置为其他货币样式
- `百分比样式`命令：格式设置为百分比
- `千位分隔符样式`命令：使用千位分隔符设置样式
- `增加小数位数`命令：显示更多的小数位数
- `减少小数位数`命令：显示较少的小数位数

### 设置单元格格式对话框->数字选项卡


![[计算机科学/基础/attachments/Pasted image 20250311085837.png]]


- 示例：所选单元格内容的结果样式
- 分类：数字类型


| 类型   | 含义                     | 可设置的内容                    |
| ---- | ---------------------- | ------------------------- |
| 常规   | 无特殊格式                  |                           |
| 数值   | 用于一般数字的表示              | 小数位数<br>使用前为分隔符<br>负数     |
| 货币   | 用于表示一般货币数值             | 小数位数<br>货币符号(国家/地区)<br>负数 |
| 会计专用 | 可对一列数值进行货币符号和小数点对齐     | 小数位数<br>货币符号              |
| 日期   | 将日期和时间系列数值显示为日期值<br>   | (日期时间)类型<br>区域设置(国家/地区)   |
| 时间   | 将日期和时间系列数值显示为时间值       | (日期时间)类型<br>区域设置(国家/地区)   |
| 百分比  | 将单元格中数值乘以100，并以百分数形式显示 | 小数位数                      |
| 分数   |                        | (分数)类型                    |
| 科学计数 |                        | 小数位数                      |
| 文本   | 数字作为文本处理               |                           |
| 特殊   | 可用于跟踪数据列表和数据库的值        | (特殊)类型<br>区域设置(国家/地区)     |
| 自定义  | 以现有格式为基础，生成自定义的数字格式    | (自定义)类型                   |

- 货币符号(国家/地区)：选择货币的符号
- 负数：选择负数的样式
- 小数位数：设置小数位数
- (日期时间)类型：以\*开头的日期格式，受到操作系统特定的区域日期和时间设置的影响
- (分数)类型
- (特殊)类型
- (自定义)类型
