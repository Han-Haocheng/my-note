# 计算学科
计算学科：是一门学科，与计算有关的学科的统称
- 如：计算物理、计算化学

# 计算机学科

计算机学科：研究计算机的一门学科
- 如：计算机系统结构、计算机应用

# 计算机科学

计算机科学：研究计算机及其周围各种现象和规律的科学
- 如：理论计算机科学、实验计算机科学
