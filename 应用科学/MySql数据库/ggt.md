
常用查询条件

| 查询条件   | 操作符                                                      |
| ------ | -------------------------------------------------------- |
| 比    较 | = ,<> ,!= ,< ,<= ,> ,>= ,!< ,!> ，<br><br> NOT+含比较运算符的表达式 |
| 确定范围   | BETWEEN AND，NOT BETWEEN AND                              |
| 确定集合   | IN，NOT IN                                                |
| 字符匹配   | LIKE，NOT LIKE                                            |
| 空    值 | IS NULL，IS NOT NULL                                      |
| 多重条件   | AND，OR                                                   |

比较大小
 比较运算符用于比较目标列表达式的值，
 当目标列表达式的值与比较的值比较成立时，返回逻辑值TRUE(真)，否则返回FALSE(假)。
 
带BETWEEN…AND关键字的范围查询
 当查询的过滤条件被限定在某个取值范围时，可以使用BETWEEN…AND操作符。
 其语法格式是：
  <表达式> [NOT] BETWEEN <表达式1> AND <表达式2> 
 其中,表达式1的值不能大于表达式2的值
 
带IN关键字的集合查询
 使用关键字IN可以用来查询字段值属于指定集合范围内的记录，
 当要判断的值匹配集合范围内的任意一个值时，会返回结果TRUE，否则返回FALSE。
