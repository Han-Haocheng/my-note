# 数据查询语言

```mysql
SELECT [ALL|DISTINCT|DISTINCTROW] <目标列表达式1> [,<目标列表达式2>] …
	FROM <表名1或视图名1> [,<表名2或视图名2> ]…  
	[WHERE <条件表达式> ]
	[GROUP BY <列名1> [HAVING <条件表达式>] ]
	[ORDERBY <列名2> [ASC|DESC]]
	[LIMIT [m,] n];
```

`ALL|DISTINCT|DISTINCTROW`：为可选项，用于指定是否返回结果集中的重复行。
- 默认为 ALL，即返回 SELECT 操作中所有匹配的行，包括存在的重复行；
- 若指定选项 DISTINCT 或 DISTINCTROW(任选)，则取消结果集中的重复行，


`FROM` 子句：用于指定数据来源于哪些表或视图；

`WHERE` 子句：为可选项，用于指定对记录的筛选过滤条件；

` GROUP BY` 子句：为可选项，用于将查询结果集按指定的字段值分组；
- 其 HAVING 子句为可选项，包含在 GROUP BY 子句后，用于指定分组时的筛选过滤条件；

`ORDER BY` 子句：为可选项，用于将查询结果集按指定字段值的升序或降序排序，
- `ASC` 为升序，`DESC` 为降序，默认为升序；

`LIMIT` 子句：为可选项，用于指定查询结果集包含的记录数。

| 子句       | 说  明        | 是否必须使用      |
| -------- | ----------- | ----------- |
| SELECT   | 指定返回的列或表达式  | 是           |
| FROM     | 指定检索数据的表    | 仅在从表选择数据时使用 |
| WHERE    | 指定行级过滤条件    | 否           |
| GKOUP BY | 指定分组字段      | 仅在按组计算聚合时使用 |
| HAVlNG   | 指定分组过滤条件    | 否           |
| ORDER BY | 指定排序字段及排序方式 | 否           |
| LIMIT    | 指定返回的记录数    | 否           |


## 单表查询


```mysql
SELECT [目标列表达式1 [,目标列表达式2[, …]]]  FROM <`表名`> ; 
```

`<目标列表达式序列>`：用于指定需要查询显示的内容
- 包括：字段名、算术表达式、字符串常量、函数和列别名

### 选择字段
#### 查询指定字段

查询指定字段：查询时若只需要显示数据表中的部分字段，这就是关系运算中的投影
- 可以在SELECT子句的“<目标列表达式序列>”中指定要查询的字段
- 若查询的字段有多个，则各个字段名之间需要用逗号分隔
- 查询返回显示结果时，结果集中各字段是依照SELECT子句中指定字段的次序给出
- 从查询结果可以看出，“<目标列表达式序列>”中各个字段的先后顺序可以与表中字段定义的顺序不一致，
- 用户可以根据应用的需要改变字段的显示顺序
- 如果想要去掉重复记录，必须在SELECT关键字后面指定关键字 `DISTINCT``或DISTINCTROW`，缺省为ALL

#### 查询所有字段




查询一个表中的所有字段有两种方法：
 一种方法是在SELECT关键字后面列出所有字段名；
 另一种更简单方法是在SELECT关键字后面直接使用星号(`*`)通配符，其语法格式如下：

```mysql
SELECT * FROM 表名 ;
```



#### 查询包括计算值

SELECT 子句的“<目标列表达式>”不仅可以是表中的字段名，也可以是表达式，还可以是字符串常量、函数等
- 可以使用`CURDATE()`函数返回当前日期，`YEAR()`函数返回指定日期对应的年份

#### 定义字段别名


当 SELECT 子句的目标列为英文字段名或表达式时，可以用 AS 子句为目标列定义别名，
- 结果集将显示别名
- 注意：当自定义的别名中含有空格时，必须使用单引号将别名括起来
- 字段的别名不允许出现在WHERE子句中

```mysql
字段名 [AS] 字段别名
```


#### 选择指定记录


在SELECT语句中，可以通过`WHERE`子句设置选择的条件(也称搜索条件或查询条件)。
- 查询时根据需要可以选择数据表中指定的记录，对数据进行过滤，这就是关系运算中的选择

```mysql
SELECT <目标列表达式序列> FROM <表名> WHERE <查询条件>
```

### 选择指定记录

常用查询条件

| ------ | -------------------------------------------------------- |
| 比    较 | = ,<> ,!= ,< ,<= ,> ,>= ,!< ,!> ，<br><br> NOT+含比较运算符的表达式 |

| 空    值 | IS NULL，IS NOT NULL                                      |
| 多重条件   | AND，OR                                                   |
#### 比较大小


比较大小：比较运算符用于比较目标列表达式的值， 当目标列表达式的值与比较的值比较成立时，返回逻辑值 TRUE(真)，否则返回 FALSE(假)。
- “大于等于”也可以表达为“不小于”,所以该查询也可以使用关键字NOT来改写
