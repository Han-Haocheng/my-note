## 三、问答题
1.  什么是函数重载？C++中实现函数重载的依据是什么？
    答：函数重载：在同一个作用域内，可以定义多个同名函数，但它们的参数列表必须不同。重载依据：C++通过函数的参数列表（参数的类型、个数、顺序）来区分重载函数。
2.  什么是运算符重载？它有哪些限制？
    答：运算符重载：赋予 C++ 运算符新的含义，使其能够操作用户自定义类型。
    主要限制：
    1) 不能创建新的运算符
    2) 不能改变运算符的优先级和结合性
    3) 至少有一个操作数是自定义类型
    4) 那些运算符不能重载（如：:、. *、. 、? :、sizeof等）
3.  C++的 string 类和C风格字符串主要区别是什么？
    答：
    1) 类型不同，C++字符串是 string 类型，C风格是字符数组。
    2) 头文件不同，C++字符串必须使用头文件 string，C风格字符串不需要。
    3) 赋值操作不同，C++字符串使用=赋值，C风格使用strcpy函数。
    4) 连接操作不同，C++字符串使用+实现连接，C风格使用strcat函数。
    5) 计算长度操作不同，C++字符串使用 length 函数，C风格使用 strlen 函数。
4.  什么是类？什么是对象？它们之间是什么关系？
    答：类：是一种用户自定义的数据类型，它是对象的蓝图或模板。类定义了一组属性（数据成员）和方法（成员函数）。对象：是类的具体实例，是类所描述的属性和方法的实际体现。
5.  C++中的访问控制符有哪些？它们各自的作用是什么？
    答：共有三种，分别为 public、protected 和 private。
    作用：
    1) public（公有）：在任何地方都可以访问。
    2) private（私有）：只能在类的成员函数中访问，类外不可访问。
    3) protected（保护）：可以在类的成员函数和派生类的成员函数中访问。
6. 什么是构造函数？它有哪些特点？
    答：构造函数是一种特殊的成员函数，用于初始化对象的数据成员。
    特点：
    1) 函数名与类名完全相同
    2) 没有返回类型（连void都没有）
    3) 在创建对象时自动调用
7.  问：在构造派生类对象时，构造函数和析构函数的调用顺序是怎样的？
    答：
    1) 构造函数调用顺序：基类的构造函数 -> 派生类自己的构造函数体。
    2) 析构函数调用顺序与构造函数调用顺序完全相反：派生类自己的析构函数体 -> 基类的析构函数。
