# 数据类型

## 二进制表示的数


整型数的二进制表示

- 整型分为有符号、无符号
- 有符号最高位为符号位，0正1负
- 整型占用字节数1、2、4、8

| 字节数 | 有符号范围                          | 无符号范围             |
| --- | ------------------------------ | ----------------- |
| 1   | -128 \| +127                   | 0 \| 255          |
| 2   | -3,2768 \| +3,2767             | 0 \| 6,5535       |
| 4   | -21,4748,3648 \| +21,4748,3648 | 0 \| 42,9496,7295 |
| 8   |                                |                   |

浮点数的二进制表示
- 浮点数分为单精度浮点数、双精度浮点数
- 单精度浮点数占4字节、双精度浮点数占8字节
- 浮点型数据的一般表达式：$(-1)^{S}\times 2^{E}\times M$
	- 实际的阶码值$e$：表示浮点的取值范围，
	- 尾数$m$：表示浮点的精度

- 移码$E$：占8位，$E=127+e$

### 常量

- 在C和C++中通常使用`const`修饰类型使类型变成常量
- 字面量：一种特殊的常量，包括整型、实型、字符、字符串、枚举等常量

### 变量


变量Variable：计算机程序运行中其值可以发生改变的量
- 要素：数据类型、变量名（名字、标识符）
- 原则：先定义后使用
- 一般形式：`<数据类型> <变量列表>`
	- 变量列表：可以由一个或多个由逗号分割的多个标识符名构成
- 变量初始化格式：`<数据类型> <变量名1>=<初值1>[, <变量名2>=<初值2>[,...]];`


标识符：对变量名、函数名、标号和其他各种用户定义的对象名
- 建议取有实际意义的标识符
- 第一个字符只能是字母或下划线
- 不能出现特殊符号，
- 最长不能超过32字符
- 严格区分大小写
- 不能和C语言关键字相同
- 不能出现同名


```c
int a;
double x,y,z; 
char c;       
long total;   
```
## 标准IO函数

### 格式化输入函数
格式化输入函数：按格式控制字符串指定的格式，从键盘输入数据，并将输入的值赋给输入项列表相对应的变量。空白字符表示当前变量输入结束
```c
void scanf(char* str,...);
```
- 参数
	- `str`：规定了数据的类型，其内容可以由格式转换说明符和输入分隔符组成
	- `...`：由一个或多个变量地址组成，各变量地址之间用逗号`,`分割

### 格式化输出函数

格式化输出函数：按照控制字符串指定格式，向标准输出设备输出指定的输出项列表的值
```c
void printf(char* str,...);
```

`...`：可以是 常量、变量、表达式

### 格式控制字符串
格式控制字符串(修饰符)：
- 一般形式：`%[<修饰符>]<格式字符>`
- 修饰符：用于确定输入数据的宽度、精度等。省略则系统默认
	- 字段宽度修饰符：用数字修饰输入输出字符占用的宽度（字符数），输出时仅指定最小输出宽度
	- 对齐方式修饰符：默认右对齐，在对应字段宽度的区间中进行对齐，通过在百分号后添加`-`可进行左对齐
	- 数据类型长短：l表示长整型或双精度浮点型，h表示短整型
		- 输入时如果大于指定宽度，则会读取指定宽度，剩余保存在缓冲区
	- 字符`*`：跳过相应的数据，输入的数据不赋给变量

| 格式转换说明符 | 含义                                          | 输出                        | 输入         |
| ------- | ------------------------------------------- | ------------------------- | ---------- |
| %d或%i   | 输入输出十进制整数                                   |                           |            |
| %o      | 输入输出八进制整数                                   | 不输出前导符0                   |            |
| %x      | 输入输出十六进制整数                                  | 不输出前导符0x，输出小写形式           |            |
| %c      | 输入输出一个字符                                    |                           | 允许空白字符     |
| %s      | 输入输出一个字符串                                   |                           | 截止到第一个空白字符 |
| %f      | 输入输出一个单精度浮点型                                |                           | 可以是小数或指数形式 |
| %e      | 输入输出一个单精度浮点型                                |                           | 可以是小数或指数形式 |
| %lf     | 输入输出一个双精度浮点型                                |                           | 可以是小数或指数形式 |
| \%%     | 输入输出一个百分号%                                  |                           |            |
| l       | 位于d、i、o、x、u之间输入long类型数据<br>位于f、e之前输入双精度浮点数据 |                           |            |
| L       | 位于f、e之前输入三精度浮点类型数据                          |                           |            |
| h       | 位于d、i、o、x、u之间输入short类型数据                    |                           |            |
| m       | 域宽，指定输入数据的宽度(列数)，自动截取数据                     |                           |            |
| .n      | 显示精度                                        | 输出小数位数<br>输出从左侧开始截取的字符串字串 |            |
| `*`     | 跳过相应的数据，输入的数据不赋给变量                          |                           |            |
| `-`     | 左对齐显示                                       |                           |            |

## 数组类型

数组：在内存中分配一片固定连续的空间，存放多个相同类型的数据  

```C
// 数据类型 数组名[元素数量];
// 声明了10个连续的int类型
int arr_1[10];
```

## 指针类型

指针：可以保存数据的内存地址  
 
```C++
// 数据类型* 指针名;
int* ptr = &value;
```
