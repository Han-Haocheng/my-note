# 程序流程控制

语句：组成程序的基本元素，通常以`;`结尾。
- 表达式语句：执行运算的语句
- 函数调用语句：执行函数调用的语句
- 控制语句：控制语句执行顺序的语句
- 复合语句：由0条或多条语句组合起来的复合结构，通常使用`{}`作为边界

程序流程：代码中各语句的执行次序
- 包含顺序结构、选择结构，循环结构
- 
- 循环结构：重复执行某些语句
## 顺序结构

顺序结构：顺序执行每一条语句，从前往后依次执行每一条语句，每一条语句只执行一次

## 选择结构

选择结构：有选择的执行部分语句，程序中部分代码的执行收到==预设条件==的控制，当预设条件符合时才执行这些语句
- 适用于实现选择结构的语句：`if`、`if-else`、`switch`结构


### if语句

if语句：是常用的选择结构，是最简单的一种单分治结构，该结构构成一个二叉结构

一般形式
![[应用科学/c&c++程序/attachments/2026-01-17_191709.svg]]
```c
if(<表达式>){
	语句A;
}
```
- 先判断`<表达式>`的值，若为真，则执行语句A，否则跳过
- `<表达式>`：通常为关系表达式、条件表达式或逻辑表达式
- 语句A：通常是复合语句

举例：从键盘任意输入两个实数a和b，要求a的值总是小于或等于b的值，然后输出这两个数a和b的值。


```c
#include<stdio.h>

int main(){
	double a=0,b=0;
	scanf("a=%f,b=%f\n",a,b);
	
	if(a>b){
		double tmp = a;
		a=b;
		b=tmp;
	}
	printf("a=%f,b=%f\n",a,b);
}
```
### if-else语句

一般形式
![[应用科学/c&c++程序/attachments/2026-01-17_193301.svg]]

```c
if(<表达式>){
	语句A;
}else{
	语句B;
}
```
- 语句功能：先判断<表达式>的值，若该值为“真”，则：执行<语句A>；否则：什么也不执行
- <表达式>：一般为关系表达式、条件表达式或逻辑表达式
- if语句中的<语句A>一般情况下都是以复合语句的形式出现，即用一对花括号将语句括起来


### 嵌套if-else

if嵌套：可判定两种以上的情形

一般格式
![[应用科学/c&c++程序/attachments/2026-01-17_194029.svg]]
```c
if(<表达式1>){
	语句1;
}else if(<表达式2>){
	语句2;
}else if(<表达式3>){
	语句3;
}
//...
else if(<表达式n>){
	语句n;
}else{
	语句n+1;
}
```


举例：从键盘输入`+`、`-`、`*`、`/`中的任一个，输出对应的英文单词：plus、minus、mutiply、divide，若输入的不是这4个字符中的任一个，则输出“Sorry you are wrong”。算法的流程图如图所示：

```c
#include<stdio.h>

int main(){
	char ch=0;
	ch = getc();
	if(ch=='+'){
		printf("plus");
	}else if(ch=='-'){
		printf("minus");
	}else if(ch=='*'){
		printf("mutiply");
	}else if(ch=='/'){
		printf("divide");
	}else{
		printf("Sorry you are wrong");
	}

}
```


- 在if嵌套中会出现多个if和else。如果没有使用大括号明确，则存在else究竟与哪个if配对的问题。
- 在这种情况下，C语言规定：else总是与它前面最近的、且没有配对的if相匹配。
- 为避免出现误解，推荐每个语句块都使用大括号，哪怕只有一条语句。
- if_else结构的嵌套层次不提倡太多，否则会影响程序的执行效率，并且容易出现判断上的漏洞，导致程序出现不正确的结果

### switch


一般形式
```c
switch(<表达式>)
{
	case 常量表达式1:语句1;
	case 常量表达式2:语句2;
	case 常量表达式3:语句3;
	//...
	case 常量表达式n:语句n;
	default: 语句n+1;
}
```
- <表达式>可以为整型、字符型或者枚举型，但不允许是实型。
- <常量表达式i>的值必须为常量，且类型应与switch中表达式的类型相同。
- 语句序列i允许有多条语句，不需要将语句序列用一对花括号括起；
- default分支主要用于处理switch表达式与所有case常量都不匹配的情况。它在语法上可以省略，但推荐使用。


条件分支独立
```c
switch(<表达式>)
{
	case 常量表达式1:语句1;break;
	case 常量表达式2:语句2;break;
	case 常量表达式3:语句3;break;
	//...
	case 常量表达式n:语句n;break;
	default: 语句n+1;
}
```

## 循环结构


循环：在“预设条件”的控制下可以执行多次的语句和代码
- 程序中的==循环次数是有限==的，由循环条件决定可以确定循环次数

- 循环结构语句：for语句、while语句、do_while语句
- 循环中断语句：break与continue




不同循环结构的特点比较:
- do_while与while和for循环的区别:</br>do_while循环中的<循环体语句>至少会执行一次;</br>而while和for循环中的<循环体语句>有可能一次也不被执行。
-  for循环和while循环的算法流程图描述是一致的。
- for循环适合于循环次数确定的情况。
- 对大多数问题，do_while、while和for循环是可以互换的。


### for语句


for语句是一种计数循环。循环次数由循环变量来控制。

![[应用科学/c&c++程序/attachments/2026-01-18_201311.svg]]

for语句的一般形式为：
```c
for (<初始表达式>;<条件表达式>;<循环表达式>){
    <循环体语句>
}
```

- 初始表达式：初始化循环控制变量
- 条件表达式：控制循环的条件
- 循环表达式：更新循环控制变量的值

for语句语法功能
1. 计算<初始表达式>的值。该表达式是对循环控制变量进行初始化。
2. 判断<条件表达式>的值。该表达式就是循环条件</br>若该表达式的值为“真”，则执行<循环体语句>；</br>若该表达式的值为“假”，则退出循环，执行循环结构外的语句。
3. 计算<循环表达式>的值。该表达式更新循环控制变量的值。
4. 转第2步。


for语句特性
- <初始表达式>可以省略，但须保留分号（;），同时在for之前必须给循环控制变量赋值，形式为：
	```c
	<初始表达式>;
	for (;<条件表达式>;<循环表达式>)
	    <循环体语句>
	```
- <条件表达式>一般不可省略，否则为无限循环，相当于条件总为真，程序会一直不停地执行直到“数据溢出”。 
	```c
	for (i=1;;i++) {
	    sum = sum+i;
	```
- <循环表达式>亦可省略，但在循环语句体中必须有语句来修改循环变量，以使条件表达式的值在某一时刻为假，使程序能正常结束循环
	```c
	for(sum=0, i=1; i<=100;){
	    sum = sum +i;
	    i++;
	}
	```
- 3个表达式均省略，即`for(;;)`，为无限循环，程序中要==避免==这种情况的发生。
- 条件表达式可以是关系表达式、数值表达式。只要表达式的值不等于零就执行循环体语句。
	```c
	for(i=0; (c=getchar())!='\n';i+=c);
	```
- 初始表达式、循环表达式可以是逗号表达式，用来完成逗号表达式中各表达式的功能。
	```c
	for (sum=0, i=1; i<=100; i++, i++)
	```
	相当于：
	```c
	sum=0;
	for(i=1; i<=100; i=i+2)
	```
- 在循环语句的循环体中，也可以再次出现循环语句，这种情况称为循环嵌套。


从1加到100;

```c
int i,sum=0;
for(i=1;i<=100;i++){
	sum+=i;
}
```

打印长方形`m*n`

```c
int m,n,i,j;  
printf("请输入长方形的长和宽");  
scanf("%d*%d",&m,&n);  
if(m<0||n<0)return;  
for(i=0;i<m;i++){  
    for(j=0;j<n;j++)  
        printf("* ");;  
    printf("\n");  
}
```

### while语句
while语句的一般形式为：
```c
while (<循环表达式>) {
    循环语句;
    循环变量表达式;
}
```
- <循环语句>和<循环变量表达式>一起构成循环体语句。
1. 计算<循环表达式>的值</br>若该值为“假”，则跳出循环，执行循环体后面的语句；</br>若该值为“真”，则执行循环体语句。
2. 重复步骤1的操作。


说明
- <循环变量表达式>不可缺少，其作用是更新计算循环变量的值，使循环能正常结束。若没有<循环变量表达式>，则有可能会使程序出现无限循环而发生错误。
- 由于while循环是先判断<循环表达式>的值，后决定是否执行<循环体语句>，因此，<循环体语句>有可能一次也没有执行
### do_while语句
do_while语句（直到型循环）的一般形式为:
```c
do {
    <循环体语句>
}while(<条件表达式>);
```
do_while语句的语法功能:
- 执行<循环体语句>，
- 计算<条件表达式>；</br>若该表达式的值为“真”，则执行步骤1；</br>若该表达式的值为“假”，则退出循环语句结构。
注意：要避免出现无限循环而发生错误。


### break语句

break语句：可用于==选择结构==和==循环结构==，结束==本层==循环体的运行，==退出循环==
- 常用于循环语句体内某一个if条件分支的语句中，用来表示在循环过程中满足某一条件时，结束循环


### continue语句

continue语句：只是结束==本次==循环体的运行，并未退出循环体。

一般形式为：continue，

- continue语句不会跳出循环结构，而是提前进行下一个循环。
- while语句和do_while语句遇到continue时，程序会立刻转到条件表达式，开始下一轮循环；
- 而在for语句中遇到continue时，程序会立刻转到〈循环表达式〉，更新循环变量，开始下一轮循环。
