# 转移指令的原理

转移指令：可以修改IP,或同时修改CS和IP的指令统称

转移指令的种类
- 无条件转移
- 条件转移
- 循环
- 过程
- 中断

## 操作符offset

操作符offset：取得标号的偏移地址

```ASM
ASSUME CS:CODESG

CODESG SEGMENT
START:
	MOV AX,OFFSET START; == MOV AX,0
S:
	MOV AX,OFFSET S    ; == MOV AX,3
CODESG ENDS

END START
```

问题9.1：有如下程序段，添写两条指令，使该程序在运行中将s处的一条指令复制到s0处。

```
ASSUME CS:CODESG
CODESG SEGMENT
S: 
	MOV AX,BX ;MOV AX,BX的机器码占两个字节
	MOV SI,OFFSET S
	MOV DI,OFFSET S0
	MOV	AX,CS:[SI]
	MOV	CS:[DI],AX

S0: 
	NOP;NOP的机器码占一个字节
	NOP
CODESG ENDS
END S
```



## `jmp`指令

`jmp`指令：无条件转移，可以只修改`IP`，也可同时修改 `CS:IP`
- 需要：转移目的地 或 转移距离

### 依据位移进行转移

`jmp 段内范围 标号`：段内转移，转移到标号处执行指令
- `jmp short 标号`：`-80H<IP<7fH`
- `jmp near ptr 标号`：`-8000H<IP<7fffH`
- 编译器会自动将标号所在的位置转译成 偏移 形式
- CPU不需要目的地址就可以实现对IP的修改
- 机器指令：从jmp指令的下一条指令到目标地址的距离
- 表示方法：补码

```
assume cs:codesg
codesg segment
start:
	mov ax,0
	jmp short start
	add ax,1
s:
	inc ax
codesg ends
end start
```


### 转移目的地址在指令中

`jmp far ptr 标号`：段间转移，远转移
- 编译器会自动将标号所在的位置转译成`CS:IP`形式
```
assume cs:codesg
codesg segment
start:
	mov ax,0
	mov bx,0
	jmp far ptr s
	db 256 dup(0)
s: 
	add ax,1
	inc ax
codesg ends
end start
```

### 转移地址在寄存器中

`jmp 16位寄存器`：段内转移

### 转移地址在内存中


- `jmp word ptr [内存单元]`：段内转移，转移到内存单元中的值所对应的地址

```
mov ax,0123H
mov ds:[0],ax
jmp word ptr ds:[0]

mov ax,0123H
mov [bx],ax
jmp word ptr [bx]
```

`jmp dword ptr [内存单元地址]`：段间转移

```
mov ax,0123H
mov ds:[0],ax
mov word ptr ds:[2],0
jmp dword ptr ds:[0]

mov ax,0123H
mov [bx],ax
mov word ptr [bx+2],0
jmp dword ptr [bx]
```

## `jcxz`指令

`jcxz 标号`指令：如果`(cx)=0`，则`(IP)=(IP)+偏移`。有条件跳转指令，段内短转移($-128\sim127$)，在对应机器码中包含位移

```c
if((cx)==0) jmp short 标号;
```

检测点9.2：补全编程，利用jcxz 指令，实现在内存2000H段中查找第一个值为0的字节，找到后，将它的偏移地址存储在dx中。

```
assume cs:code
code segment
start:
	mov ax,2000H
	mov ds,ax
	mov bx,0
s:
	mov cl,ds:[bx]
	mov ch,0
	jcxz ok
	inc bx
	jmp short s
ok:
	mov dx,bx
	mov ax,4c00h
	int 21h
code ends
end start
```

## `loop`指令

`loop 标号`指令：`(cx)=(cx)-1`，如果`(cx)!=0`，则跳转到标号。有条件跳转指令，段内短转移($-128\sim127$)，在对应机器码中包含位移


```c
(cx)--;
if((cx)!=0) jmp short 标号;
```

检测点9.3：补全编程，利用loop指令，实现在内存2000H段中查找第一个值为0的字节，找到后，将它的偏移地址存储在dx中。

```
assume cs:code
code segment
start:
	mov ax,2000H
	mov ds,ax
	mov bx,0
s: 
	mov cl,[bx]
	mov ch,0
	inc cx
	inc bx
	loop s
ok:
	dec bx ;dec指令的功能和inc相反，dec bx进行的操作为：(bx)=(bx)-1
	mov dx,bx
	mov ax,4c00h
	int 21h
code ends
end start
```

## 根据位移进行转移的意义

- `jmp short 标号`
- `jmp near ptr 标号`
- `jcxz 标号`
- `loop 标号`

根据位移进行转移，方便程序段在内存中的浮动装配，在内存中任何位置都可以达到相同的结果

### 编译器对转移位移超界的检测

当转移位移超界，编译器会自动报错

```
assume cs:code
code segment
start:
	jmp short s
	db 128 dup(0)
s:mov ax,0ffffh
code ends
end start
```
