# 数据处理的两个基本问题

- 定义描述性符号：reg和sreg
- reg：ax-dx，ah-dh，al-dl，sp，bp，si，di
- sreg：ds、ss、cs、es

## `bx`,`si`,`di`,`bp`

`bx`,`si`,`di`,`bp`：可以通过`[...]`方式寻址
- 可使用的寻址方式
	- `[bx]`、`[bp]`、`[si]`、`[di]`
	- `[bx+si]`、`[bx+di]`
	- `[bp+si]`、`[bp+di]`
- bp隐性对应段：`ss`
- bx隐性对应段：`ds`


## 机器指令处理数据的位置

机器指令
- 大致分为三类：读写算
- 数据位置：CPU内部、内存、端口

### 汇编语言数据位置的表达

数据位置
- 立即数(idata)
- 寄存器
- 段地址(SA)、偏移地址(EA)：可强制显性使用段

## 寻址方式

| 寻址类型     | 寻址方式                                                                     | 含义                                                                                                                   |
| -------- | ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| 直接寻址     | `[idata]`                                                                | EA=idata;SA=(ds)                                                                                                     |
| 寄存器间接寻址  | `[bx]`<br>`[bp]`<br>`[si]`<br>`[di]`                                     | EA=(bx);SA=(ds)<br>EA=(bp);SA=(ss)<br>EA=(di);SA=(ds)<br>EA=(si);SA=(ds)                                             |
| 寄存器相对寻址  | `[bx+idata]`<br>`[bp+idata]`<br>`[si+idata]`<br>`[di+idata]`             | EA=(bx)+idata;SA=(ds)<br>EA=(di)+idata;SA=(ds)<br>EA=(si)+idata;SA=(ds)<br>EA=(bp)+idata;SA=(ss)                     |
| 基址变址寻址   | `[bx+si]`<br>`[bx+di]`<br>`[bp+si]`<br>`[bp+di]`                         | EA=(bx)+(si);SA=(ds)<br>EA=(bx)+(di);SA=(ds)<br>EA=(bp)+(si);SA=(ss)<br>EA=(bp)+(di);SA=(ss)                         |
| 相对基址变质寻址 | `[bx+si+idata]`<br>`[bx+di+idata]`<br>`[bp+si+idata]`<br>`[bp+di+idata]` | EA=(bx)+(si)+idata;SA=(ds)<br>EA=(bx)+(di)+idata;SA=(ds)<br>EA=(bp)+(si)+idata;SA=(ss)<br>EA=(bp)+(di)+idata;SA=(ss) |

### 指令处理数据的长度

指令长度：`byte`、`word`
- 通过寄存器：寄存器指明了指令的数据长度
- 通过操作符：`X ptr`方式指定数据长度
	- `mov word ptr ds:[0],1`
	- `mov byte ptr ds:[0],1`
- 通过其他方式：指令本身就有固定长度

### 寻址方式运用



```
ASSUME CS:CODESG,DS:DATASG

DATASG SEGMENT		; SEG:60  
	DB	'DEC'		    ; +00     公司名称
	DB	'Ken Oslen'	; +03     总裁姓名
	DW	137			    ; +0C     排名
	DW	40			    ; +0E     收入
	DB	'PDP'		    ; +10     著名产品
DATASG ENDS

CODESG	SEGMENT

START:
	MOV	AX,DATASG
	MOV	DS,AX
	MOV	BX,0;60H

	MOV WORD PTR DS:[BX].0CH,38
	ADD	WORD PTR DS:[BX].0EH,70

	MOV	BYTE PTR DS:[BX].10H[SI],'V'
	INC	SI
	MOV	BYTE PTR DS:[BX].10H[SI],'A'
	INC	SI
	MOV	BYTE PTR DS:[BX].10H[SI],'X'

	MOV	AX,4C00H
	INT	21H

CODESG	ENDS

END START
```

- bx定位整个结构体变量
- idata定位结构体中的数据项
- si定位数据项中的每个元素

## 指令
### `div`指令

`div`指令：division除法指令
- `div reg`、`div 内存单元`

| 位数    | 被除数       | 除数    | 商    | 余数   |
| ----- | --------- | ----- | ---- | ---- |
| 16bit | `AX`      | 8bit  | `AL` | `AH` |
| 32bit | `DX`+`AX` | 16bit | `AX` | `DX` |

```
MOV DX,1
MOV AX,86A1H
MOV BX,100
DIV BX
```


### `dd`伪指令

`dd`伪指令：double word双字数据


问题8.1：用div计算data段中第一个数据除以第二个数据后的结果，商存在第三个数据的存储单元中。
```ASM
ASSUME CS:CODESG,DS:DATASG

DATASG SEGMENT
	DD 100001
	DW 100
	DW 0
DATASG ENDS

CODESG SEGMENT
START:
	MOV AX,DATASG
	MOV DS,AX
	MOV	BX,0

	MOV	AX,DS:[BX]
	MOV	DX,DS,[BX+2]
	DIV	WORD PTR [BX+4]   ; 需要标识除法位数
	MOV	[BX+6],AX

	MOV	AX,4C00H
	INT 21H

CODESG ENDS

END START
```

### `dup`指令

`dup`指令：操作，按照内容存储多个重复数据
- `db 3 dup (0)`=`db 0,0,0`
- `db 4 dup (0,1,2)`=`db 0,1,2,0,1,2,0,1,2,0,1,2`


实验7 寻址方式在结构化数据访问中的应用

Power idea公司从1975年成立一直到1995年的基本情况如下。

| 年份   | 收入(千美元) | 雇员(人) | 人均收入(千美元) |
| ---- | ------- | ----- | --------- |
| 1975 | 16      | 3     | ?         |
| 1976 | 22      | 7     | ?         |
| 1977 | 382     | 9     | ?         |
| 1978 | 1356    | 13    | ?         |
| 1979 | 2390    | 28    | ?         |
| 1980 | 8000    | 38    | ?         |
| 1995 | 5937000 | 17800 | ?         |

用汇编指令计算人均收入，并保存为一个二维表`('YEAR SUMM NE ?? ')`

```
ASSUME CS:CODESG

DATA SEGMENT
	;+0
	DB	'1975','1976','1977','1978','1979','1980','1981','1982','1983'
	DB	'1984','1985','1986','1987','1988','1989','1990','1991','1992'
	DB	'1993','1994','1995'
	;以上是表示21年的21个字符串
	;+84
	DD 	16,22,382,1356,2390,8000,16000,24486,50065,97479,140417,197514
	DD 	345980,590827,803530,1183000,1843000,2759000,3753000,4649000,5937000
	; 以上是表示21年公司总收入的21个DWORD型数据
	;+168
	DW 	3,7,9,13,28,38,130,220,476,778,1001,1442,2258,2793,4037,5635,8226  
	DW 	11542,14430,15257,17800
	; 以上是表示21年公司雇员人数的21个WORD型数据
DATA ENDS

TABLE SEGMENT
	DB 21 DUP('YEAR SUMM NE ?? ')
TABLE ENDS

CODESG SEGMENT
START:
	MOV AX,DATA
	MOV	DS,AX
	MOV	BX,0

	MOV AX,TABLE
	MOV ES,AX
	MOV DI,0

	MOV CX,21
	MOV	SI,0
	MOV	DI,0

S:	MOV	AX,DS:[SI]
	MOV DX,DS:[SI+2]
	MOV	ES:[BX],AX
	MOV	ES:[BX+2],DX

	MOV	AX,DS:[84+SI]
	MOV DX,DS:[84+SI+2]
	MOV	ES:[BX+5],AX
	MOV	ES:[BX+7],DX

	DIV	WORD PTR DS:[168+DI]
	MOV	ES:[BX+13],AX

	MOV	AX,DS:[168+DI]
	MOV	ES:[BX+0],AX

	ADD	SI,4
	ADD	DI,2
	ADD	BX,10H
	LOOP S


	MOV AX,4C00H
	INT 21H

CODESG ENDS

END START
```
