
## 内存中字存储

任何两个地址连续的内存单元， n号单元和n+1号单元，可以将它们看成两个内存单元，也可以看成一个地址为n的字单元中的高位字节单元和低位字节单元。
- 字：低址低位

### DS和\[address\]

DS寄存器：存放要访问的数据的段地址
- 通过使用`mov 寄存器,[数据段偏移地址]`
- 不能将数据直接送入段寄存器

```asm
mov bx,1000H
mov ds,bx
mov al,[0]
```

- 将 `al`中的数据送入内存单元10000H
```asm
mov bx,1000H
mov ds,bx
mov [0],al
```

### 字的传输

在8086CPU中，可以一次传输1个字

```asm
mov bx,1000H
mov ds,bx
mov ax,[0]      ; 传输了一个字
mov [0],cx
```


- 写出下面指令执行后寄存器ax，bx，cx中的值

```asm
; 1000:0 23 11 22 66
mov ax,1000H
mov ds,ax
mov ax,[0]   ;ax=1123H
mov bx,[2]   ;bx=6622H
mov cx,[1]   ;cx=2211H
add bx,[1]   ;bx+=2211H=8833H
add cx,[2]   ;cx+=6622H=8833H
```


```asm
; 1000:0 23 11 22 11
mov ax,1000H 
mov ds,ax
mov ax,11316 ; ax=2C34H
mov [0],ax   ; 1000:0 34 2C 22 11
mov bx,[0]   ; bx= 2C34H
sub bx,[2]   ; bx-=1122H=1B12H
mov [2],bx   ; 1000:0 2C 34 12 1B
```

## 转移加减指令
### mov

- `mov 寄存器,数据`
- `mov 寄存器,寄存器`
- `mov 寄存器,[内存单元]`
- `mov [内存单元],寄存器`
- `mov 段寄存器,寄存器`
- `mov 寄存器,段寄存器`


```
mov ax,8
mov ax,bx
mov ax,[0]
mov [0],ax
mov ds,ax
mov ax,ds
```

### add

- `add 寄存器,数据`
- `add 寄存器,寄存器`
- `add 寄存器,[内存单元]`
- `add [内存单元],寄存器`

```
add ax,8
add ax,bx
add ax,[0]
add [0],ax
```
### sub

- `sub 寄存器,数据`
- `sub 寄存器,寄存器`
- `sub 寄存器,[内存单元]`
- `sub [内存单元],寄存器`

```
sub ax,8
sub ax,bx
sub ax,[0]
sub [0],ax
```

## 栈

栈：后进先出的数据结构

### CPU提供的栈机制

- 入栈(push)：将数据压入栈顶，
- 出栈(pop)：将栈顶数据送出
- 每次操作后都会影响`SS:SP`对应的栈顶寄存器
	- push操作会先使`SP-=2`，再写入数据
	- pop操作会先写出数据，再使`SP+=2`

### 栈顶越界的问题

栈顶越界：当栈满时，进行入栈操作，会覆盖非法数据；当栈空时，进行出栈操作，会获取非法数据
- 危险性：会获取或覆盖其他代码段、数据段的数据

### push、pop指令

- `push 寄存器`
- `push [内存单元]`
- `pop 寄存器`
- `pop [内存单元]`
