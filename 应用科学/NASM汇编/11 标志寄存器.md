# 标志寄存器
标志寄存器：16位寄存器，存储的信息是状态字（PSW）
- 受影响指令：add、sub、mul、div、inc、or、and等


注意：只有在运算时候会发生变化，当只有转移指令时，标志显示之前的状态，不会发生改变

| 位数  | 状态  | 位数  | 状态  | 位数  | 状态  | 位数  | 状态  |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 15  |     | 11  | OF  | 7   | SF  | 3   |     |
| 14  |     | 10  | DF  | 6   | ZF  | 2   | PF  |
| 13  |     | 9   | IF  | 5   |     | 1   |     |
| 12  |     | 8   | TF  | 4   | AF  | 0   | CF  |

## 标志
### ZF标志

ZF标志：第6位，零标志位。结果为0则为1；结果不为0则为0

```
mov ax,1
sub ax,1 ; ZF=1

mov ax,2
sub ax,1 ; ZF=0
```

### PF标志

PF标志：第2位，奇偶标志位。判断结果的所有bit位中1的个数是否为偶数。如果1的个数为偶数，pf=1,如果为奇数，那么pf=0

```
mov al,1
add al,10 ; 结果为00001011B,其中有3(奇数)个1,则pf=0;

mov al,1
or al,2 ;结果为00000011B,其中有2(偶数)个1,则pf=1;

sub al,al
执行后，结果为00000000B,其中有0(偶数)个1,则pf=1。
```

### SF标志

SF标志：第7位，符号标志位。判断结果是否为负。如果结果为负，sf=1；如果非负，sf=0。

```
mov al,10000001B
add al,1；结果为10000010B,sf=1

mov al,10000001B
add al,01111111B ;结果为0,sf=0负。
```

检测点11.1：写出下面每条指令执行后，ZF、PF、SF等标志位的值。

```

sub al,al	; al=0   	ZF=1 PF=1 SF=0
mov al,1 	; al=1		ZF=1 PF=1 SF=0
push ax 	; al=1		ZF=1 PF=1 SF=0
pop bx 		; al=1		ZF=1 PF=1 SF=0
add al,bl ; al=2   	ZF=0 PF=0 SF=0
add al,10 ; al=12  	ZF=0 PF=1 SF=0
mul al 		; al=144 	ZF=0 PF=1 SF=1
```

### CF标志

CF：第0位，进位标志位。一般情况下，在进行无符号数运算时，记录运算结果的最高有效位，向更高位的进位值，或从更高位的借位值。
- 两个8位数据：98H+98H,将产生进位


```
mov al,98H
add al,al  ;执行后：(al)=30H,CF=1,CF记录了从最高有效位向更高位的进位值
add al,al  ;执行后：(al)=60H,CF=0,CF记录了从最高有效位向更高位的进位值

mov al,97H
sub al,98H ;执行后：(al)=FFH,CF=1,CF记录了向更高位的借位值
sub al,al  ;执行后：(al)=0,CF=0,CF记录了向更高位的借位值
```

### OF标志

溢出：有符号数运算的时候，结果超过了机器所能表示的范围

OF：第11位，溢出标志位。记录有符号数运算的结果是否发生了溢出。如果发生溢出，OF=1;如果没有，OF=0。

```
mov al,98
add al,99  ; CF=0,OF=1  正+正=负

mov al,0FOH
add al,88H ; CF=1,OF=1  负+负=正

mov al,0FOH
add al,78H ; CF=1,OF=0  正+负=不确定
```

|             | al        | OF OV/NV | SF NG/PL | ZF ZR/NZ | PF PE/PO | CF CY/NC |
| ----------- | --------- | -------- | -------- | -------- | -------- | -------- |
| sub al,al   | 0000 0000 | 0        | 0        | 1        | 1        | 0        |
| mov al,10H  | 0001 0000 | 0        | 0        | 1        | 1        | 0        |
| add al,90H  | 1010 0000 | 0        | 1        | 0        | 1        | 0        |
| mov al,80H  | 1000 0000 | 0        | 1        | 0        | 1        | 0        |
| add al,80H  | 0000 0000 | 1 OV     | 0 PL     | 1 ZR     | 1 PE     | 1 CY     |
| mov al,0FCH | 1111 1100 | 1        | 0        | 1        | 1        | 1        |
| add al,05H  | 0000 0001 | 0 NV     | 0 PL     | 0 NZ     | 0 PO     | 1 CY     |
| mov al,7DH  | 0111 1101 | 0        | 0        | 0        | 0        | 1        |
| add al,0BH  | 1000 1000 | 1 OV     | 1 NG     | 0 NZ     | 1 PE     | 0 NC     |

## 指令

### `adc`指令

`adc`指令：带进位加法指令，它利用了CF位上记录的进位值，用于计算超大位数的加法
- (ax)=(ax)+(bx)+CF

```
mov ax,2
mov bx,1
sub bx,ax
adc ax,1; (ax)=(ax)+1+CF=2+1+1=4

mov ax,1
add ax,ax
adc ax,3;(ax)=(ax)+3+CF=2+3+0=5

mov al,98H
add al,al
adc al,3; (al)=(al)+3+CF=30H+3+1=34H

add ax,bx ; =add al,b1 adc ah,bh
```

编程，计算1EF000H+201000H,结果放在ax(高16位)和bx(低16位)中

```
ASSUME CS:CODESG

CODESG SEGMENT
  START: MOV AX,001EH
         MOV BX,0F000H
         ADD BX,1000H
         ADC AX,0020H
         MOV AX,4C00H
         INT 21H
CODESG ENDS
END START
```

编程，计算1EF0001000H+2010001EFOH,结果放在ax(最高16位),bx(次高16位),cx(低16位)中。

```
ASSUME CS:CODESG

CODESG SEGMENT
  START: MOV AX,1EH
         MOV BX,0F000H
         MOV CX,1000H
         ADD CX,1EF0H
         ADC BX,1000H
         ADC AX,20H

         MOV AX,4C00H
         INT 21H
CODESG ENDS
END START
```


名称：add128
功能：两个128位数据进行相加。
参数：ds:si指向存储第一个数的内存空间，因数据为128位，所以需要8个字单元，
由低地址单元到高地址单元依次存放128位数据由低到高的各个字。运算结果存储在第一
个数的存储空间中。
ds:di指向存储第二个数的内存空间。

```
ASSUME CS:CODESG,DS:DATASG

DATASG SEGMENT
         DW 8 DUP(8888H)
         DW 8 DUP(1111H)
DATASG ENDS



CODESG SEGMENT
  START: MOV  AX,DATASG
         MOV  DS,AX
         MOV  SI,0
         MOV  DI,16

         CALL ADD128
         MOV  AX,4C00H
         INT  21H

  ADD128:PUSH AX
         PUSH CX
         PUSH SI
         PUSH DI

         SUB  AX,AX
         MOV  CX,8
  EXCL:  MOV  AX,DS:[SI]
         ADC  AX,DS:[DI]
         MOV  DS:[SI],AX
         INC  SI
         INC  SI
         INC  DI
         INC  DI
         LOOP EXCL

         POP  DI
         POP  SI
         POP  CX
         POP  AX
         RET
CODESG ENDS
END START
```

注意：inc指令不能替换为add指令，因为会改变CF标志

### `sbb`指令

sbb：带借位减法指令，利用了CF位上记录的借位值。用于计算超大位数的减法
- (ax)=(ax)-(bx)-CF

计算003E1000H-00202000H,结果放在ax,bx中

```
mov bx,1000H
mov ax,003EH
sub bx,2000H
sbb ax,0020H
```

### `cmp`指令

cmp：比较指令，相当于减法指令，不保存结果，只影响标志寄存器，用于判断两个操作数的关系。可直接访问内存


```
cmp ax,ax ; zf=1,pf=1,sf=0,cf=0,of=0。

mov ax,8
mov bx,3
cmp ax,bx; (ax)=8,zf=0,pf=1,sf=0,cf=0,of=0
```


- 无符号数检测zf、cf

| 关系        | 结果                   | 标志          |
| --------- | -------------------- | ----------- |
| (ax)=(bx) | (ax)-(bx)=0          | zf=1        |
| (ax)≠(bx) | (ax)-(bx)≠0          | zf=0        |
| (ax)<(bx) | (ax)-(bx)借位          | cf=1        |
| (ax)≥(bx) | (ax)-(bx)不借位         | cf=0        |
| (ax)>(bx) | (ax)-(bx)不借位，结果不为0   | cf=0 且 zf=0 |
| (ax)≤(bx) | (ax)-(bx)可能借位，结果可能为0 | cf=1 或 zf=1 |

- 有符号数检测 zf、sf、of

| 关系        | 结果          | 标志          |
| --------- | ----------- | ----------- |
| (ax)=(bx) | (ax)-(bx)=0 | zf=1        |
| (ax)≠(bx) | (ax)-(bx)≠0 | zf=0        |
| (ax)<(bx) | 不溢出，结果为负    | sf=1 且 of=0 |
| (ax)≥(bx) | 不溢出，结果为正    | sf=0 且 of=0 |
| (ax)>(bx) | 溢出，结果为负     | sf=1 且 of=1 |
| (ax)≤(bx) | 溢出，结果为正     | sf=0 且 of=1 |



### 检测比较结果的条件转移指令

- 无符号数的比较结果进行转移的条件转移指令
	- n：not
	- e：equal
	- b：below
	- a：above

| 指令  | 转移条件 | 检测标志        |     |
| --- | ---- | ----------- | --- |
| je  | 等于   | zf=1        |     |
| jne | 不等于  | zf=0        |     |
| jb  | 低于   | cf=1        |     |
| jnb | 不低于  | cf=0        |     |
| ja  | 高于   | cf=0 且 zf=0 |     |
| jna | 不高于  | cf=1 或 zf=1 |     |

编程：如果(ah)=(bh)则(ah)=(ah)+(ah),否则(ah)=(ah)+(bh)

```
	  CMP AH,BH
	  JNE S
	  ADD AH,AH
	  JMP SHORT OK
S:  ADD AH,BH
OK: RET
```

### DF标志和串传送指令

DF：第10位，方向标志位。在串处理指令中，控制每次操作后si、di的增减
- df=0 每次操作后si、di自动递增
- df=1 每次操作后si、di自动递减

`movsb`/`movsw`：传送字节/字
- `((es)<<4+(di))=((ds)<<16+(si))`
- 如果df=0则
	- `(si)=(si)+移动大小`
	- `(di)=(di)+移动大小`
- 如果df=1则
	- `(si)=(si)-移动大小`
	- `(di)=(di)-移动大小`


`rep 指令`：根据cx的值进行重复操作
- 通常配合`movsb`/`movsw`
- `rep movsb`/`rep movsw`

```
s:movsb
loop s
```


`cld`指令：将标志寄存器的df位，置0
`std`指令：将标志寄存器的df位，置1

编程，用串传送指令，将data段中的第一个字符串复制到它后面的空间中。
```ASM
ASSUME CS:CODE,DS:DATA,SS:STACK

DATA SEGMENT
       DB 'WELCOME TO MASM!'
       DB 16 DUP(0)
DATA ENDS

CODE SEGMENT
  START:MOV AX,DATA
        MOV DS,AX
        MOV SI,0
        MOV DI,16
        MOV CX,16

				CLD
        REP MOVSB
        
        MOV AX,4C00H
        INT 21H
CODE ENDS
END START
```

编程，用串传送指令，将F000H段中的最后16个字符复制到data段中。
```
ASSUME CS:CODE,DS:DATA
DATA SEGMENT
            DB 16 DUP(0)
DATA ENDS

END START
```

### `pushf`和`popf`

`pushf`和`popf`：为直接访问标志寄存器提供了一种方法
- pushf：将标志寄存器的值压栈
- popf：从栈中弹出数据，送入标志寄存器中

检测点11.4：下面的程序执行后：(ax)=0045H

```
mov ax,0
push ax
popf ; 清空标志寄存器
mov ax,0fff0h 
add ax,0010h  ; 进位 0 无溢出 偶数位
pushf
pop ax
and al,1100 0101B 
and ah,0000 1000B
;0 2 6 7 11  cf pf zf sf of
```
