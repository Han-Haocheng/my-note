### whereis
```bash
用法：  
whereis [选项] [-BMS <dir>... -f] <名称>  
  
定位某个命令的二进制文件、源码和帮助页文件。  
  
选项：  
-b         只搜索二进制文件  
-B <目录>  定义二进制文件查找路径  
-m         只搜索手册和信息  
-M <目录>  定义 man 和 info 查找路径  
-s         只搜索源代码  
-S <目录>  定义源代码查找路径  
-f         终止 <目录> 参数列表  
-u         搜索不常见记录  
-g         interpret name as glob (pathnames pattern)  
-l         输出有效查找路径  
  
-h, --help     显示此帮助  
-V, --version  显示版本
```
