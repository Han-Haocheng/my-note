# 文件和目录管理命令


## 目录操作

pwd：打印当前工作目录的名字
- 默认情况下，"pwd" 的行为和给定了 "-L" 时一致
- 退出状态：返回 0，除非使用了无效的选项，或者无法读取当前目录

```bash
pwd: pwd [-LP]  

选项：  
-L, --logical          使用环境变量中的 PWD，即使其中包含符号链接  
-P, --physical         避免所有符号链接

```
