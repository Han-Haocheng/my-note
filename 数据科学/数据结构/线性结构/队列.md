# 队列

![[数据科学/数据结构/attachments/2026-01-16_185646.svg]]

**队列(queue)**：一种先进先出的限定存取位置的线性表，只允许一段插入，另一端删除
- **先进先出(Frist In Frist Out，FIFO)**：率先插入，率先使用
- 队尾rear：允许插入的一端
- 队头front：允许删除的一端
- 入队：从队尾插入元素的操作
- 出队：从队头删除元素的操作
## 1 假溢


假溢：当队列不断运行时，队尾元素会指向空间末尾，此时有空闲空间，但不可再继续插入新的队尾元素，否则会因数组越界而遭到程序代码破坏
- 解决方式：将顺序内存视为头尾相连的圆环，形成循环队列
- 解决方式：使用链队列列
## 2 队列存储类型
### 2.1 循环队列

循环队列：允许队列从数组中下标最大的位置延续到下标最小的位置，可通过取模实现
### 2.2 链队列
链队列：链式存储结构的队列

头尾指针相同状态问题：当头尾指针相同时，既可以表示队空，也可以表示队满
- 解决方式1:少用一个元素空间，约定入队前测试尾指针在循环意义下加一后是否等于头指针

```c
q->front==q->rear; // 队空条件
(q->rear+1)%MaxSize == q->front; // 队满条件
```

### 2.3 双端队列
双端队列：两端都可以进行进队和出队操作



## 3 队列的抽象数据类型
### 3.1 数据对象
队列的数据对象是具有相同特性的数据元素的集合。
$$ D = \{ a_i \mid a_i \in ElemSet, i=1, 2, \dots, n, n \ge 0 \} $$
其中，$ElemSet$ 是元素集合，$n$ 为队列中元素的个数。
### 3.2 数据关系
队列是线性表，数据元素之间存在着“一对一”的逻辑关系。
$$ R = \{ \langle a_{i-1}, a_i \rangle \mid a_{i-1}, a_i \in D, i=2, \dots, n \} $$
*   约定：$a_n$ 端为**队尾**，$a_1$ 端为**队头**。
*   **特征**：
    *   队头元素是第一个被插入的元素。
    *   队尾元素是最后一个被插入的元素。
    *   允许在**队尾**进行插入操作。
    *   允许在**队头**进行删除操作。
### 3.3 基本操作
假设队列定义为 $Q$，元素类型为 $ElemType$，常用的基本操作如下：
1.  **InitQueue(&Q)**
    *   **操作结果**：构造一个空队列 $Q$。
2.  **DestroyQueue(&Q)**
    *   **初始条件**：队列 $Q$ 已存在。
    *   **操作结果**：队列 $Q$ 被销毁，不再存在。
3.  **ClearQueue(&Q)**
    *   **初始条件**：队列 $Q$ 已存在。
    *   **操作结果**：将 $Q$ 清为空队列。
4.  **QueueEmpty(Q)**
    *   **初始条件**：队列 $Q$ 已存在。
    *   **操作结果**：若 $Q$ 为空队列，则返回 `TRUE`，否则返回 `FALSE`。
5.  **QueueLength(Q)**
    *   **初始条件**：队列 $Q$ 已存在。
    *   **操作结果**：返回 $Q$ 的元素个数，即队列的长度。
6.  **GetHead(Q, &e)**
    *   **初始条件**：队列 $Q$ 已存在且非空。
    *   **操作结果**：用 $e$ 返回 $Q$ 的**队头**元素。（注意：不修改队列状态）
7.  **EnQueue(&Q, e)**
    *   **初始条件**：队列 $Q$ 已存在。
    *   **操作结果**：插入元素 $e$ 为 $Q$ 的新的**队尾**元素。
8.  **DeQueue(&Q, &e)**
    *   **初始条件**：队列 $Q$ 已存在且非空。
    *   **操作结果**：删除 $Q$ 的**队头**元素，并用 $e$ 返回其值。

### 5.2 队列的销毁

```c

```
链队列
```c
void DestroyQueue(SqQueue*&q){
	// 初始化临时指针
	DataNode*pre=q->front,*p;
	
	// 过滤空队列情况
	if(pre==NULL){
		free(q);
		return;
	}
	
	// 初始化指向下一个节点
	p=pre->next;
	
	// 循环删除
	while(p!=NULL){
		free(pre);
		pre=p;
		p=p->next;
	}
	free(pre);
	free(q);
}
```

### 5.4 队列的入队

```c
bool EnQueue(SqQueue*&q,ElemType e){
	if(q->rear==MaxSize)return false;
	q->data[q->rear]=e;
	q->rear++;
	return true;
}

```

顺序循环队列
```c
bool EnQueue(SqQueue*&q,ElemType e){
	if((q->rear+1)%MaxSize==q->front)return false;
	q->data[q->rear]=e;
	q->rear=(q->rear+1)%MaxSize;
	return true;
}
```

链队列

```c
bool EnQueue(SqQueue*&q,ElemType e)
{
	//初始化临时数据
	DataNode*p;
	p=(DataNode*)malloc(sizeof(DataNode));
	p->data=e;
	p->next=NULL;
	
	// 过滤空队列情况
	if(rear==NULL){
		q->front=q->rear=p;
		return;
	}
	
	// 插入元素到队尾
	q->rear->next=p;
	q->rear=p;
}
```

### 5.5 队列的出队

```c
bool DeQueue(SqQueue*&q,ElemType e){
	if(q->rear==MaxSize)return false;
	e=q->data[q->front];
	q->front++;
	return true;
}
```

顺序循环队列

```c
bool DeQueue(SqQueue*&q,ElemType e){
	if(q->rear==q->front) return false;
	
	e=q->data[q->front];
	q->front=(q->front+1)%MaxSize; 
	return true;
}
```


链队列

```c
bool DeQueue(SqQueue*&q,ElemType e)
{
	if(q->rear==NULL)
		return false;
	DataNode*t=q->front;
	
	// 选择
	if(q->front==q->rear)
		q->front=q->rear=NULL;
	else 
		q->front=q->front->next;
	
	// 返回和释放
	e=t->data;
	free(t);
	return true;
}
```
