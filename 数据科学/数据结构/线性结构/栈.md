# 栈

- 栈和队列时限定插入和删除，只能在端点进行的*线性表*

![[数据科学/数据结构/attachments/20250421161800.svg]]

**栈(stack)**：是一个特殊的线性表，是限定在一端(通常是表尾)进行插入和删除操作的线性表，是*后进先出*的线性表
- **后进先出(Last In First Out，LIFO)**：率先使用最后插入的数据
- **栈顶(Top)**：表尾($a_{n}$端)
- **栈底(Base)**：表头($a_{1}$端)，
- **入栈(Push)**：将元素插入的到*栈顶*的操作
- **出栈(Pop)**：从*栈顶*删除最后一个元素的操作

```c
void conversion()
```

## 2 括号匹配

括号匹配：左括号压栈，每个右括号要求与栈顶元素匹配

### 2.1 表达式求值
前缀表达式（波兰表达式）：与算法在两个操作数的前面，完全不依赖括号和运算符优先级来确定顺序，从右往左，先遇到先运算

1. 初始化一个栈stack；
2. 从右往左遍历表达式；
3. 遇到数字，压入栈；
4. 遇到运算符，弹出栈顶的两个数字，执行运算（顺序为栈顶元素作为左操作数，次顶元素作为右操作数）后将结果压入栈；
5. 重复上述过程，直到遍历完成；
6. 栈顶的数字即为最终结果。


后缀表达式（逆波兰表达式）：运算符在两个操作数后面，不需要括号，运算顺序唯一明确；从左往右，先遇到先运算
 1. 初始化一个栈stack；
 2. 从左至右遍历表达式；
 3. 遇到数字，压入栈；
 4. 遇到运算符，从栈顶弹出两个数字，执行运算（顺序为栈顶元素作为右操作数，次顶元素作为左操作数）后将结果压入栈；
 5. 重复上述操作，直到遍历结束；
 6. 栈顶元素即为最终计算结果。


中缀表达式转为后缀表达式，核心目标：不改变运算顺序的前提下，消除括号并明确运算符优先级，转换规则如下：
1. ==从左到右依次扫描==，遇到操作数：直接输出到结果。
2. 遇到运算符：比较当前运算符与栈顶运算符的优先级。若==栈顶优先级≥当前运算符，弹出栈顶并输出，重复此过程直到栈顶优先级＜当前运算符，再将当前运算符压栈==。
3. 遇到左括号直接入栈，遇到右括号时弹出栈内所有运算符直到遇到左括号，并丢弃左括号
4. 表达式结束：弹出栈中所有剩余运算符并输出。

中缀表达式转为前缀表达式，转换规则如下：
1. ==从右往左扫描表达式==，遇到操作数时直接输出到结果。
2. 遇到运算符：比较当前运算符与栈顶运算符的优先级。==若栈顶优先级＞当前运算符，弹出栈顶并输出，直到栈顶优先级≤当前运算符，再将当前运算符压栈==。
3. 遇到右括号）：直接压栈（因为从右向左扫描，右括号视为“待匹配”符号），遇到左括号（：弹出栈顶并输出，直到遇到右括号），并丢弃右括号。
4. 表达式结束：弹出栈中所有剩余运算符并输出，==最终反转整个结果==。

## 3 栈的抽象数据类型定义

数据对象：$D=\{a_i | a_i \in ElemSet, i\in Z^+\}$

数据关系：$R1=\{<a_i-1,a_i>|a_i-1,a_i\in D,i=2,\dots,n\}$
- 约定$a_{n}$端为栈顶，$a_{1}$端为栈底。

基本操作
- 初始化操作：`InitStack(&S)`
	- 操作结果：构造一个空栈$S$
- 销毁栈操作：`DestroyStack(&S)`
	- 初始条件：栈$S$已存在
	- 操作结果：栈$S$被销毁
- 判断$S$是否为空栈：`StackEmpty(S)`
	- 初始条件：栈$S$已存在
	- 操作结果：若栈$S$为空栈，返回`TRUE`，否则 `FALSE`
- 求栈的长度`StackLength(S)`
	- 初始条件：栈$S$已存在
	- 操作结果：返回$S$的元素个数，即栈的长度
- 获取栈顶元素：`GetTop(S,&e)`
	- 初始条件：栈$S$已经存在，且非空
	- 操作结果：用$e$返回$S$的栈顶元素
- 栈置空操作：`ClearStack(&S)`
	- 初始条件：栈$S$已存在
	- 操作结果：将$S$清为空栈
- 入栈操作：`Push(&S,e)`
	- 初始条件：栈$S$已经存在
	- 操作结果：插入元素$e$为新的栈顶元素
- 出栈操作：`Pop(&S,&e)`
	- 初始条件：栈$S$已存在且非空
	- 操作结果：删除$S$的栈顶元素$a_{n}$，并用$e$返回其值


## 4 数据结构

顺序栈：用连续的内存空间存储栈
```c
#define MAXSIZE 100
typedef struct
{ 
	SElemType *base;
	SElemType *top;
	int stacksize;
}SqStack;
```

共享栈：用一个数组来实现两个栈，一个栈位于起始，另一个位于末尾
```c
typedef struct
{
	SElemType data[MAXSIZE];
	int top1,top2;
}DStack;
```

链栈：用链式存储结构的栈，通常不带头节点的单链表表示
```c
typedef struct StackNode{
	SElemType data;
	struct StackNode *next;
}StackNode,*LlinkStack;
```

## 5 基本操作
### 5.1 栈初始化

```C++
Status InitStack(SqStack&S)
{
	S.base = (SElemType*)malloc(MAXSIZE*sizeof(SElemType));
	if(!S.base) exit(OVERFLOW); // 内存分配失败
	S.top = S.base;
	S.stacksize = MAXSIZE;
	return OK;
};
```

链栈

```c
Status InitStack(LinkStack&S)
{
	S=NULL;
	return OK;
}
```

### 5.2 栈判空

```c++
Status StackEmpty(SqStack S){
	return S.top == S.base;
}

Status StackEmpty(LinkStack &S){
	return S == NULL;
}
```

### 5.3 栈获取长度

```c++
int StackLength(SqStack S){
	return S.top - S.base;
}
```


### 5.4 栈清空

```c++
Status ClearStack(SqStack S){
	if(S.base) S.top = S.base;
	return OK;
}
```

### 5.5 栈销毁

```c++
Status DestroyStack(SqStack &S){
	if(!S.base)return OK;
	delete S.base;
	S.stacksize = 0;
	S.base = S.top = NULL;
	return OK;
}
```

链栈
```c
Status Push(LinkStack &s, SElemType e){
	struct StackNode*p;
	p = (struct StackNode*)malloc(sizeof(struct StackNode));
	p->data=e;
	p->next=s;
	S = p;
	return OK
}
```

### 5.7 栈出栈

顺序栈出栈
```C++
Status Pop(SqStack &S, SElemType &e){
	if(S.top == S.base)return ERROR;
	e = *(--S.top);
	return OK;
}
```

链栈出栈

```c
status Pop(LinkStack &s,SElemType &e){
	struct StackNode*p;
	if(s==NULL)return ERROR;
	e=s->data;
	p=s;
	s= s->next;
	free(p);
	return OK;
}
```

### 5.8 栈获取栈顶元素

顺序栈
```c
Status GetTop(SqStack S, ElemType&e){
	if(S.top == S.base) return ERROR;
	e=*(S.top-1);
	return OK
}
```

链栈
```c++
SElemType GetTop(LinkStack S){
	if(S==NULL) exit();
	return S->data;
}
```
