# Socket编程

网络程序设计接口

- 直接网卡编程
- PacketDriver：屏蔽网卡细节，适用于所有网卡
- NDIS：Windows网络
- LibPcap/WinPcap：抓包
- Libnet：构造发送
- Libnids：入侵检测
- Libicmp：ICMP协议处理
- NetBIOS：Windows提供的会话层API
- Socket：应用层和传输层之间
- Web/PRC/中间件：应用层编程

应用编程接口API（Application Programming Interface）：应用进程的控制器和操作系统的控制权进行转换的系统调用接口

套接字接口socket：BerkeleyUNIX操作系统定义的API
- WINSOCK（Windows Socket Interface）
- TLI（Transport Layer Interface）

## SocketAPI概述

SocketAPI：事实上的工业标准，兼容性强，Internet网络典型的API，应用进程间通信的抽象机制

- 标识通信端点：对外，IP地址+端口号
- 套接字描述符：对内，操作系统/进程标识套接字Socket descriptor，小整数

1. 应用进程创建套接字
2. 操作系统分配该进程套接字相关信息的数据结构存储
3. 返回套接字描述符（类似于Windows中的句柄）

```
socket数据结构{
	family: PF INET
	service: sock stream
	local IP
	remote IP
	local port
	remote port
}
```

使用TCP/IP协议族的网络应用进程，使用sockaddr_in声明端点地址变量

```c
struct sockaddr_in{
	u_char sin_len;          // 地址长度
	u_char sin_family;       // 地址族（TCP/IP: AF_INET）
	u_short sin_port;        // 端口号
	struct in_addr sin_addr; // IP地址
	char sin_zero[8];        // 未用置0
}
```

## SocketAPI函数

### 初始化资源

`WSAStartup`：初始化Windows Sockets API，使用Socket之前必须首先调用

```c
int WSAStartup(WORD wVersionRequested,LPWSADATA lpWSAData);
```
- `wVersionRequested`：指明应用程序使用的WinSock版本
	- 高位字节：指明副版本
	- 低位字节：指明主版本
- `lpWSAData`：返回实际的WinSock的版本，指向`WSADATA`结构的指针
- 返回：错误代码


```c
WORD wVersionRequested;
WSADATA wsaData;

wVersionRequested = MAKEWORD(2,1);
err = WSAStartup(wVersionRequested,&wsaData)
```


### 释放资源

`WSACleanup`：释放所使用的Windows Sockets DLL，最后要调用，解除与Socket库的绑定，释放系统资源

```c
int WSACleanup(void)
```

- 返回：错误代码

### 创建套接字
`socket`：创建套接字

```c
SOCKET socket(protofamily,type,proto)
```

- 协议族`protofamily`
	- `PF_INET`：使用TCP/IP协议族
- 套接字类型`type`
	- `SOCK_STREAM`：流式套接字，TCP，可靠、面向连接、字节流传输、点对点
	- `SOCK_DGRAM`：数据报套接字，UDP，不可靠，无连接，数据报传输
	- `SOCK_RAW`：原始套接字，网络层
- 协议号proto：默认为0
- 返回：套接字描述符

```c
struct protoent *p;
p = getprotobyname("tcp")
SOCKET sd = socket(PF_INET,SOCK_STREAM,p->p_proto);
```

### 关闭套接字

`closesocket：关闭一个描述符为sd的套接字。多进程共享套接字，则计数减1，至0关闭。多线程套接字不计数

```c
int closesocket(SOCKET sd);
```

- 套接字描述符`sd`
- 返回：错误代码
	- 0：成功
	- SOCKET_ERROR：失败

### 设置端点地址

`bind`：绑定套接字的本地端点地址。客户端一般不必调用bind函数

```c
int bind(sd,sockaddr_in localaddr,addrlen)
```

- 套接字描述符sd
- 端点地址localaddr：设置localaddr.sin_addr=INADDR_ANY从而设置IP地址
- `addrlen`
