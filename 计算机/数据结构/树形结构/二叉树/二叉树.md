# 二叉树

二叉树：是$n(n \geq 0)$个结点的有限集合，当$n=0$时，是空二叉树；当$n>0$时，由一个根结点和左右子树组成，左右子树严格区分
- 左子树：位于节点左侧的二叉树
- 右子树：位于节点右侧的二叉树
- 二叉树的递归定义：$T = \begin{cases} \Phi, & n = 0 \\ \{root, T_L, T_R\}, & n > 0 \end{cases}$
- 二叉树是[[计算机/数据结构/树形结构/树|树]]的延伸，是特殊的树形结构

二叉树基本形态
- 空二叉树
- 只含根的二叉树
- 右子树为空
- 左子树为空
- 左右子树非空


根据[[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]和[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]定义可知：
- [[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]是[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]的特例
  [[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]一定是一棵[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]
  而[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]不一定是一棵[[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]
- [[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]的叶子结点全都在最底层
  而[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]的叶子结点可以分布在最下面两层。
- 空二叉树和只含根的二叉树，既是[[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]，也是[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]。

特殊二叉树：[[计算机/数据结构/树形结构/二叉树/满二叉树|满二叉树]]、[[计算机/数据结构/树形结构/二叉树/完全二叉树|完全二叉树]]、[[计算机/数据结构/树形结构/二叉树/二叉搜索树|二叉搜索树]]、[[计算机/数据结构/树形结构/二叉树/平衡二叉树|平衡二叉树]]、[[计算机/数据结构/树形结构/二叉树/线索二叉树|线索二叉树]]

## 1 二叉树的性质

#计算机/数据结构/性质 二叉树的第$i$层上至多有$2^{i-1}$个结点（$i \geq 1$）

> [!cite]- 证明：使用归纳法
> - 当$i=1$时
> 	- 非空二叉树在第1层只有一个根结点，$2^{1-1}=2^{0}=1$，结论成立。
> - 现假定对于所有的$j,1 \leq j < i$结论成立
> 	- 即第$j$层上至多有$2^{j-1}$个结点
> 	- 那么可以证明$j=i$时命题也成立
> - 由归纳假设：在第$i-1$层上至多有$2^{i-2}$个结点
> 	- 由于二叉树每个结点的度至多为$2$
> 	- 因而第$i$层上的最大结点数为第$i-1$层上最大结点数的2倍
> 		- 即$2 \times 2^{i-2}=2^{i-1}$
> 	- 故性质1成立

#计算机/数据结构/性质  深度为k（$k\geq1$）的二叉树至多有$2^k-1$个结点（$k\geq1$）

> [!cite]- 证明
> - $k\geq1$时，二叉树非空，且具有k层，对任一层i(1≤i≤k),根据性质1可知第i层上至多有$2^{i-1}$个结点。
> - 在具有相同深度的二叉树中，仅当每一层都具有最大的结点数时，二叉树中的结点数才最多，则整个二叉树中所具有的最大结点数为：$$\sum_{i=1}^{k}(\text{第}i\text{层上的最大结点数}) = \sum_{i=1}^{k}2^{i-1} = 2^k - 1$$
> - 故性质2成立。

#计算机/数据结构/性质  对任何一棵非空二叉树，如果其叶子结点数为$n_0$，度为2的结点数为$n_2$，则$n_0=n_2+1$

> [!cite]- 证明：设二叉树中度为1的结点数为$n_1$。
> - 因为：二叉树中所有结点的度数均不超过2
> - 所以：树中的结点总数为$n=n_0+n_1+n_2$
> - 又：二叉树中除根结点外，其余每一结点都有且仅有一个双亲结点，进入它们的分支数均为1，故二叉树中总的分支数为：$e=n-1=n_0+n_1+n_2-1$ ①
> - 又：分支均由度为1或2结点发出，故总分分支数为：$e=n_1+2n_2$ ②
> - 由①和②得：$n_0+n_1+n_2-1=n_1+2n_2$，即$n_0=n_2+1$
> - 故性质3成立
## 2 二叉树的抽象数据类型
### 2.1 数据对象
二叉树的数据对象是一个集合 $D$。
*   $D$ 是具有相同特性的数据元素的集合。
*   如果 $D$ 为空集，则称为空二叉树。
*   如果 $D$ 非空，则其中存在一个唯一的称为**根** 的数据元素，该元素没有前驱；其余元素可分为两个互不相交的子集：左子树 $L$ 和右子树 $R$。$L$ 和 $R$ 本身也是二叉树。
### 2.2 数据关系
二叉树中的数据元素之间存在**层次关系**（一对多的非线性关系）。
*   若 $D$ 为空，则为空树。
*   若 $D$ 不为空，则存在唯一的根节点 $root$。
*   根节点 $root$ 最多有两个后继，分别称为**左孩子** 和 **右孩子**。
*   根节点的左子树 $L$ 中的所有节点都是 $root$ 的后代，且在 $root$ 的左侧；右子树 $R$ 中的所有节点都是 $root$ 的后代，且在 $root$ 的右侧。
*   **有序性**：二叉树的子树有左右之分，其次序不能任意颠倒。即使某节点只有一个子树，也必须明确指出该子树是左子树还是右子树

### 2.3 基本操作
假设二叉树采用 `BiTree` 类型定义，二叉树中的节点元素类型为 `TElemType`。
1.  **InitBiTree(&T)**
    *   **操作结果**：构造一棵空二叉树 $T$。
2.  **DestroyBiTree(&T)**
    *   **初始条件**：二叉树 $T$ 存在。
    *   **操作结果**：销毁二叉树 $T$。
3.  **CreateBiTree(&T, definition)**
    *   **初始条件**：definition 给出二叉树 $T$ 的定义。
    *   **操作结果**：按 definition 给出的定义构造二叉树 $T$。
4.  **ClearBiTree(&T)**
    *   **初始条件**：二叉树 $T$ 存在。
    *   **操作结果**：将二叉树 $T$ 清为空树。
5.  **BiTreeEmpty(T)**
    *   **初始条件**：二叉树 $T$ 存在。
    *   **操作结果**：若 $T$ 为空二叉树，返回 `TRUE`，否则返回 `FALSE`。
6.  **BiTreeDepth(T)**
    *   **初始条件**：二叉树 $T$ 存在。
    *   **操作结果**：返回二叉树 $T$ 的深度（或高度）。
7.  **Root(T)**
    *   **初始条件**：二叉树 $T$ 存在。
    *   **操作结果**：返回 $T$ 的根节点的值；若 $T$ 为空，返回“空”或特定标识。
8.  **Value(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：返回 $e$ 的值。
9.  **Assign(T, &e, value)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：将节点 $e$ 的赋值为 `value`。
10. **Parent(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：若 $e$ 是 $T$ 的非根节点，则返回它的双亲；否则返回“空”。
11. **LeftChild(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：返回 $e$ 的左孩子。若 $e$ 无左孩子，则返回“空”。
12. **RightChild(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：返回 $e$ 的右孩子。若 $e$ 无右孩子，则返回“空”。
13. **LeftSibling(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：返回 $e$ 的左兄弟。若 $e$ 是 $T$ 的左孩子或无左兄弟，则返回“空”。
14. **RightSibling(T, e)**
    *   **初始条件**：二叉树 $T$ 存在，$e$ 是 $T$ 中某个节点。
    *   **操作结果**：返回 $e$ 的右兄弟。若 $e$ 是 $T$ 的右孩子或无右兄弟，则返回“空”。
15. **InsertChild(&T, p, LR, c)**
    *   **初始条件**：二叉树 $T$ 存在，$p$ 指向 $T$ 中某个节点，$LR$ 为 0 或 1，非空二叉树 $c$ 与 $T$ 不相交且右子树为空
    *   **操作结果**：根据 $LR$ 为 0 或 1，插入 $c$ 为 $T$ 中 $p$ 所指节点的左或右子树。$p$ 所指节点的原有左或右子树则成为 $c$ 的右子树。
16. **DeleteChild(&T, p, LR)**
    *   **初始条件**：二叉树 $T$ 存在，$p$ 指向 $T$ 中某个节点，$LR$ 为 0 或 1
    *   **操作结果**：根据 $LR$ 为 0 或 1，删除 $T$ 中 $p$ 所指节点的左或右子树
17. **PreOrderTraverse(T, Visit())**
    *   **初始条件**：二叉树 $T$ 存在，`Visit` 是对节点操作的应用函数。
    *   **操作结果**：先序遍历 $T$，对每个节点调用函数 `Visit` 一次且仅一次。一旦 `Visit()` 失败，则操作失败
18. **InOrderTraverse(T, Visit())**
    *   **初始条件**：二叉树 $T$ 存在，`Visit` 是对节点操作的应用函数。
    *   **操作结果**：中序遍历 $T$，对每个节点调用函数 `Visit` 一次且仅一次
19. **PostOrderTraverse(T, Visit())**
    *   **初始条件**：二叉树 $T$ 存在，`Visit` 是对节点操作的应用函数。
    *   **操作结果**：后序遍历 $T$，对每个节点调用函数 `Visit` 一次且仅一次
20. **LevelOrderTraverse(T, Visit())**
    *   **初始条件**：二叉树 $T$ 存在，`Visit` 是对节点操作的应用函数。
    *   **操作结果**：层序遍历 $T$，对每个节点调用函数 `Visit` 一次且仅一次



### 2.4 二叉树遍历

遍历二叉树：遵从某种顺序，顺着某一条搜索路径访问二叉树中的各个结点，使得每个结点均被访问一次，且仅被访问一次（又称周游）。
- 由二叉树的递归定义，二叉树的三个基本组成单元是：根结点、左子树和右子树
- 根据先序和中序、中序和后序可以得到唯一二叉树
- 不能根据先序和后序得到唯一二叉树

遍历方式
- 设访问根结点记作D
- 遍历根的左子树记为L
- 遍历根的右子树记为R
- 则可能遍历的次序有：DLR、DRL、LDR、LRD、RDL、RLD
- 通常先遍历左子树，后遍历右子树
- 则遍历方式为：DLR、LDR、LRD

#### 2.4.1 先序遍历

先序遍历DLR
1. 如果二叉树为空则结束遍历
2. 访问根节点
3. 先序遍历左子树
4. 先序遍历右子树


遍历结果：ABDGCEF

```c
void PreOrder(BTNode*T){
	if(T==NULL)
		return;
	visit(T->data);
	PreOrder(T->Lchild);
	PreOrder(T->Rchild);
}
```

非递归实现：先序遍历的非递归算法实现：定义一个栈并初始化，根结点入栈
1. 栈顶元素出栈
2. 访问栈顶元素指向的结点，如果有右孩子或左孩子则入栈（右孩子先于左孩子）
3. 如此循环继续，直至栈空循环退出，遍历结束。

```c
void PreOrder(BTNode*T){
	if(T==NULL)
		return;
	BTNode*Stack[maxSize];
	int top = -1;
	BTNode*p;
	Stack[++top]=T;// 根节点入栈
	while(top!=-1){
		p=Stack[top--];
		visit(p->data);
		if(p->rchild!=NULL)
			Stack[++top]=p->rchild
		if(p->lchild!=NULL)
			Stack[++top]=p->lchild
	}
}
```
#### 2.4.2 中序遍历

中序遍历LDR
1. 如果二叉树为空则结束遍历
2. 中序遍历左子树
3. 访问根节点
4. 中序遍历右子树


遍历结果：DGBAECF
```c
void InOrder(BTNode*T){
	if(T==NULL)
		return;
	InOrder(T->Lchild);
	visit(T->data);
	InOrder(T->Rchild);
}
```

#### 2.4.3 后序遍历

后序遍历LRD
1. 如果二叉树为空则结束遍历
2. 后序遍历左子树
3. 后序遍历右子树
4. 访问根节点


遍历结果：GDBEFCA
```c
void PostOrder(BTNode*T){
	if(T==NULL)
		return;
	PostOrder(T->Lchild);
	PostOrder(T->Rchild);
	visit(T->data);
}
```
#### 2.4.4 层序遍历

层序遍历：如果二叉树为空，则遍历结束；否则，根结点入队：
1. 队首元素出队
2. 访问队首元素所指向的结点
3. 将队首元素的左、右子结点依次入队，直到队空为止

```c
#define MAX_NODE 50

#define MAX_NODE 50
void levelorder (BTNode *T) {
	BTNode *Queue[MAX_NODE], *p=T ;
	int front=0, rear=0;
	if (p == NULL) 
		return
	
	Queue[++rear]=p;                // 根结点入队
	while (front<rear) {
		p=Queue[++front];
		visit (p->data);
		if (p->Lchild != NULL)
		Queue[++rear]=p->Lchild; // 左结点入队
		if (p->Rchild != NULL)
		Queue[++rear]=p->Rchild; // 右结点入队
	}
}
```

## 3 存储结构
二叉树的存储结构应能反映结点的双亲和孩子关系，即体现二叉树的逻辑关系
- 二叉树存储结构：顺序存储结构、链式存储结构。
### 3.1 二叉树的顺序存储结构

二叉树的顺序存储：所有结点按照层层次从上到下、每层从左到右依次编号，其编号从小到大的顺序就是结点存放在连续存储单元中的先后次序
- 各结点之间的逻辑关系即通过编号（存储位置）来确定，若把二叉树存储到一维数组中，那么该编号就与下标值加1一一对应。

## 4 树与二叉树的转换

树与二叉树的对应关系：树和二叉树可用二叉链表存储，给定一颗
- 给定一棵树可以找到唯一的一棵二叉树与之对应
- 物理结构：树和二叉树的二叉链表相同，解释不同
- 任何一颗和树对应的二叉树，其右子树为空

![[计算机/数据结构/attachments/2026-01-27_185053.svg]]


### 4.1 将一棵树转换成二叉树的其他方法

1. **加线**：将树中所有相邻兄弟之间加一条连线。
2. **抹线**：对树中的每个结点，只保留它与第一个孩子结点之间的连线，删除它与其他孩子结点之间的连线。
3. **旋转**：以树的根结点为轴心，将整棵树顺时针旋转45°，使之成为一棵层次分明的二叉树
