# 乘法运算

## 定点乘法运算

分析笔算乘法

- $A=-0.1101$ ，$B=0.1011$
- $A\times B=-0.1000 1111$

结果特点
- 符号位单独处理
- 乘数的某一位决定是否加被乘数
- 4 个位积一起相加
- 乘积的位数扩大一倍

笔算乘法改进

$$\begin{eqnarray}
A\cdot B& = & A\cdot 0.1011\\
&=&0.1A+0.00A+0.001A+0.0001A\\
&=&0.1A+0.00A+0.001(A+0.1A)\\
&=&0.1A+0.01[0\cdot A+0.1(A+0.1A)]\\
&=&0.1\{A+0.1[0\cdot A+0.1(A+0.1A)]\}\\
&=&2^{-1}\{1\cdot A+2^{-1}[0\cdot A+2^{-1}(1\cdot A+2^{-1}(1\cdot A+0))]\}
\end{eqnarray}$$
其中 $2^{-1}$ 在机器中表示右移 1位
1. 被乘数 A+0
2. 逻辑右移一位，生成的新的部分积
3. 部分积+被乘数
4. ...

| 部分积     | 乘数        | 说明          |
| ------- | --------- | ----------- |
| 0.0000  | 101**1**  | 初态部分积=0     |
| +0.1101 |           | 乘数为 1，加被乘数  |
|         |           |             |
| 0.1101  |           |             |
| 0.0110  | 1 10**1** | ->1，形成新的部分积 |
| +0.1101 |           | 乘数为 1，加被乘数  |
|         |           |             |
| 1.0011  | 1         |             |
| 0.1001  | 11 1**0** | ->1，形成新的部分积 |
| +0.0000 |           | 乘数为 0，加 0   |
|         |           |             |
| 0.1001  | 11        |             |
| 0.0100  | 111 **1** | ->1，形成新的部分积 |
| +0.1101 |           | 乘数为 1，加被乘数  |
|         |           |             |
| 1.0001  | 111       |             |
| 0.1000  | 1111      | ->1，得结果     |



**乘法**运算：用**加和逻辑位移实现** n=4,加 4 次，移 4 次，
- 由乘数的末位决定被乘数是否与原部分积相加，然后右移 1 位形成新的部分积，同时乘数右移 1 位（末位移丢），空出高位存放部分积的低位
- 被乘数只与部分积的高位相加

**乘法**运算硬件：3 个寄存器（其中 2 个具有位移功能），1 个全加器
- 乘商寄存器 MQ：存储乘数
- 通用寄存器 X：存储被乘数
- 累加寄存器ACC：存储高位


## 原码乘法

以小数为例
- 设 $[x]_原=x_0.x_1x_2\dots x_n$
- $[y]_原=y_0.y_1y_2\dots y_n$
- $[x\cdot y]_原=(x_0\oplus y_0).(0.x_1x_2\dots x_n)(0.y_1y_2...y_n)=(x_0\oplus y_0).x^*y^*$
- $x^*=0.x_1x_2\dots x_n$
- $y^*=0.y_1y_2\dots y_n$
- 乘积的符号单独处理 $x_0\oplus y_0$
- 数值部分为绝对值相乘 $x^*y^*$


原码一位乘递推公式

$$
\begin{eqnarray}x^*\cdot y^* 
& = & x^*(0.y_1y_2\dots y_n)\\ 
& = & x^*(y_12^{-1}+y_22^{-2}\dots y_n2^{-n})\\ 
& = & \underbrace{2^{-1}(y_1x^*+2^{-1}(y_2x^*+\dots\underbrace{2^{-1}(y_nx^*+\underset{\mathcal{z_0}}{0})\dots)}_\mathcal{z_1})}_\mathcal{z_n}  
\end{eqnarray}
$$

特点
- 绝对值运算
- 用移位的次数判断乘法是否结束
- 逻辑移位
