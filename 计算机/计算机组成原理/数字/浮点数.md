
# 浮点数

浮点数：小数点的位置通过，通常采用定点小数形式

浮点数的一般形式：$N=S \times r^j$
- 尾数$S$：小数，可正可负
- 阶码 $j$：整数，可正可负
- 尾数的基值 $r$：通常取2、4、8、16等



> [!example] 举例
> 当r=2、N=11.0101时，$N=S \times r^j=0.110101\times2^{10}$，

## 尾数规格化

尾数规格化：尾数用定点小数表示，尾数数值部分最高位必须是有效值（带权的值）
- 原码规格化：最高位为 1
- 补码规格化：最高位和符号位相反

## IEEE 754 浮点表示法


![[2026-03-06_164418.svg]]
IEEE 754 浮点表示法：阶码用移码表示，尾数用原码顶点小数表示
- 数值= $(-1)^{A}\times 1.S \times 2^{j-偏置值}$
- 移码表示阶码：移码方便比较大小

| 类型     | 符号/位 S | 阶码/位 | 尾数/位 | 总位数 | 阶码偏置值  |
| ------ | ------ | ---- | ---- | --- | ------ |
| 单精度短实数 | 1      | 8    | 23   | 32  | $7 F$  |
| 双精度长实数 | 1      | 11   | 52   | 64  | $3FFH$ |
| 临时实数   | 1      | 15   | 64   | 80  |        |

- 移码无符号值=阶码真值+偏移值
- 尾数为规格化表示：尾数非 0 的有效位最高位为 隐含的 1

| 阶码             | 伪码  | 含义          |
| -------------- | --- | ----------- |
| $1\sim2^{m}-2$ | 任意  | 规范值         |
| 0              | 0   | 正负 0        |
| 0              | 非 0 | 非规范值        |
| $2^m-1$        | 0   | $\pm\infty$ |
| $2^m-1$        | 非 0 | $NaN$       |

- 最小负数：$-2^{2^m-1}\times(1-2^{-n})$
- 最大负数：$-2^{-(2^m-1)}\times2^{-n}$
- 最小正数：$2^{-(2^m-1)}\times2^{-n}$
- 最大正数：$2^{2^m-1}\times(1-2^{-n})$
