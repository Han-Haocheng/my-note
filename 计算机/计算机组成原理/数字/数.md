# 数


进制：进位计数制中表示每个数码所处位置对应单位值的数学概念，其值为基数的整数次幂

位权展开法：进制转换的核心，通过将数字按位权展开，可以将任意进制数转换为十进制数值



# 数制

进位计数制（数制）：用进位的原则进行计数的一类符号系统
- 数码：一组用来表示某种数制的符号
- 基数（基，R）：数制所使用的数码个数称
- 位权：指数码在不同位置上的权值，处于不同数位的数码代表的数值不同

## 数制转换

数制转换：同一个数在不同进制之间表示出来，不同进制只是“书写方式”不同，数值可以相等。

常见的进制有：
- 十进制：0～9，逢十进一，日常使用。
- 二进制：0、1，逢二进一，计算机内部使用。
- 八进制：0～7，逢八进一，常与二进制配合使用。
- 十六进制：0～9、A～F，逢十六进一，计算机中常用（A=10，B=11，…，F=15）。



- **各种进制 → 十进制**：按权展开求和。
- **十进制 → 其他进制**：整数除基取余（倒排），小数乘基取整（顺序）。
- **二进制 ↔ 八进制 / 十六进制**：以小数点为界，3 位或 4 位分组对应。
- **任意两种进制**：可以先转成十进制，或先转成二进制再转换。

### 按权展开与 “除基取余”
1. **任意进制 → 十进制：按权展开求和**  
   把每一位的数字乘以该位的“权”（基数的幂），再相加。  
   例如二进制数：
   $(1011)_2 = 1\times 2^3 + 0\times 2^2 + 1\times 2^1 + 1\times 2^0 = 8 + 0 + 2 + 1 = (11)_{10}$
   八进制、十六进制同理，只是基数换成 8 或 16。
2. **十进制 → 其他进制：整数部分“除基取余”，小数部分“乘基取整”**  
   - 整数部分：不断除以目标基数，余数倒序排列。  
     例如十进制 13 转二进制：  
     13 ÷ 2 = 6 余 1  
     6 ÷ 2 = 3 余 0  
     3 ÷ 2 = 1 余 1  
     1 ÷ 2 = 0 余 1  
     余数倒排：1101，所以 (13)₁₀ = (1101)₂。
   - 小数部分：不断乘以基数，取整数部分顺序排列，直到小数为 0 或达到精度要求。  
     如 0.625 × 2 = 1.25 取 1，0.25 × 2 = 0.5 取 0，0.5 × 2 = 1.0 取 1，顺序排列得 0.101，即 (0.625)₁₀ ≈ (0.101)₂。

### 二、八、十六进制之间的转换
由于 8 = 2³、16 = 2⁴，所以它们与二进制之间可以直接“分组对应”：
1. **二进制 ↔ 八进制**  
   - 二进制 → 八进制：从小数点开始向左、向右每 3 位一组，不足补 0，每组转成一位八进制。  
     例：  
     (10110.0011)₂ → 分组：010 110 . 001 100 → 2 6 . 1 4 → (26.14)₈
   - 八进制 → 二进制：每一位八进制写成 3 位二进制即可。  
     例：(37.416)₈ → 3→011, 7→111, 4→100, 1→001, 6→110 → (11111.10000111)₂
2. **二进制 ↔ 十六进制**  
   - 二进制 → 十六进制：每 4 位一组，方法同上。  
     例：(11101101)₂ → 1110 → E, 1101 → D → (ED)₁₆
   - 十六进制 → 二进制：每一位十六进制写成 4 位二进制。
3. **八进制 ↔ 十六进制**  
   一般先转成二进制，再转成目标进制，或通过十进制中转。
