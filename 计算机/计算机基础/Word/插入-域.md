# 域

#计算机/软件应用/MSOffice/Word操作 **域**：是一种占位符，是一种插入到文档中的代码，表现的内容可以自动变化
- 举例：自动更新的日期、页码、目录

| 操作    | 含义                                |
| ----- | --------------------------------- |
| 插入域   | 将文本转换成域                           |
| 显示选中域 | 对所选的域切换域代码和它的显示内容                 |
| 显示域   | 对所有域切换域代码和它的显示内容                  |
| 解除域   | 解除域的链接，转为普通文本(文字将不带底纹，并失去自动更新的功能) |
| 锁定域   | 关闭域的自动更新                          |
| 解锁域   | 解除域的锁定                            |

- 插入域：`Ctrl`+`F9` #计算机/软件应用/快捷键/MSOffice 
- 显示选中域：`Shift`+`F9` #计算机/软件应用/快捷键/MSOffice 
- 显示域：`Alt`+`F9` #计算机/软件应用/快捷键/MSOffice 
- 解除域：`Ctrl`+`Shift`+`F9` #计算机/软件应用/快捷键/MSOffice 
- 锁定域：`Ctrl`+`F11` #计算机/软件应用/快捷键/MSOffice 
- 解锁域：`Ctrl`+`Shift`+`F11` #计算机/软件应用/快捷键/MSOffice
