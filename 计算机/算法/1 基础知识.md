## 函数的渐进的界



- 设 $f$ 和 $g$是定义域为自然数集$\mathbb{N}$上的函数
- 若存在正数 $c$ 和 $n_{0}$
	- 使得对一切 $n ≥ n_{0}$有$0 \leq f(n) \leq c g(n)$成立
- 则称 $f(n)$ 的渐近的上界是 $g(n)$
- 记作$f (n) = O(g(n))$


- 设 $f$ 和 $g$是定义域为自然数集$\mathbb{N}$上的函数
- 若存在正数 $c$ 和 $n_{0}$
	- 使得对一切 $n ≥ n_{0}$有 $0 \leq cg(n) \leq f(n)$ 成立
- 则称 f(n) 的渐近的下界是 g(n)
- 记作$f (n)$ =$\Omega(g(n))$.

- 设 f 和 g是定义域为自然数集 N上的函数
- 若对于任意正数 $c$ 都存在 $n_{0}$
	- 使得对一切 $n \geq n_{0}$有 $0 \leq f(n) < c g(n)$ 成立
- 则记作$f (n) = o(g(n))$.

- 设 $f$和 $g$是定义域为自然数集 $\mathbb{N}$上的函数
- 若对于任意正数 $c$ 都存在 $n_{0}$
	- 使得对一切 $n \geq n_{0}$有$0 \leq cg(n) \leq f(n)$成立
- 则记作$f (n) = ω (g(n))$


- 若 $f (n) = O (g(n))$
	- 且 $f (n) = Ω (g(n))$
- 则记作$f (n) = \Theta (g(n))$

### 函数的渐进的界定理

- 如果 $\lim_{ n \to \infty } \frac{f(n)}{g(n)}=c$,且$c>0$,那么$f(n)=\Theta(g(n))$
- 如果 $\lim_{ n \to \infty } \frac{f(n)}{g(n)}=0$,那么$f(n)=o(g(n))$
- 如果 $\lim_{ n \to \infty } \frac{f(n)}{g(n)}=+\infty$,那么$f(n)=\omega(g(n))$

- 如果$f=O(g)$,且$g=O(h)$，那么$f=O(h)$
- 如果$f=\Omega(g)$,且$g=\Omega(h)$，那么$f=\Omega(h)$
- 如果$f=\Theta(g)$,且$g=\Theta(h)$，那么$f=\Theta(h)$

- 多项式函数的渐进界=多项式最高次项的渐进界
- O(logn)<O(n!)
- $n! =\sqrt[]{2\pi n  }\left( \frac{n}{e} \right)^{n}\left( 1+\Theta\left( \frac{1}{n} \right) \right)$
- $n!=o(n^{n})$
- $\log(n!)=\Theta(n\log n)$


## 求和方法

$$\sum_{k=1}^{n} \frac{1}{k}=\Theta(\log n)$$

$$\sum_{k=1}^{n}n^2=\frac{{n(n+1)(2n+1)}}{6}$$

- 等差数列$a_{n}$
$$\sum_{k=1}^{n}a_{n}=n(a_{1}+a_{n})$$

- 等比数列$a_{n}$

$$\sum_{k=1}^{n}a_{n}=\frac{a_{1}(1-r^{n})}{1-r}=\frac{a_{1}}{1-r}$$




## 递推方程求解方法

主定理

- 设$a\geq 1$，$b>1$为常数，$f(n)$为函数$T(n)$为非负整数
	- 且$T(n)=aT\left( \frac{n}{b} \right)+f(n)$

- 若$f(n)=O(n^{\log a -\epsilon})$
	- 且$\epsilon>0$ 
- 那么$T(n)=\Theta(n^{\log_{b}a})$

- 若$f(n)=\Theta(n^{\log_{b}a})$
- 那么$T(n)=\Theta(n^{\log _{b}a}\log n)$

- 若$f(n)=\Omega(n^{\log_{b}a+\epsilon})$
	- 且$\epsilon>0$
	- 且存在常数$c<1$和所有充分大的$n$
		- 有$af\left( \frac{n}{b} \right)\leq cf(n)$
- 那么$T(n)=\Theta(f(n))$
